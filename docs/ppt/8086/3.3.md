# 8086 算术、逻辑、移位指令总结

## 1. 汇编语句基本格式

8086 汇编语句大体分两类：

### 1）执行性语句

也就是 CPU 真正执行的机器指令。

```asm
标号: 指令助记符  目的操作数, 源操作数  ; 注释
```

例：

```asm
START: MOV AX, DATA
       ADD AL, BL
```

### 2）说明性语句

也就是伪指令，不直接变成 CPU 指令，主要告诉汇编器如何组织程序、定义变量、设置段等。

```asm
名字  伪指令助记符  参数, 参数, ...  ; 注释
```

例：

```asm
DATA SEGMENT
NUM  DB 12H
DATA ENDS
```

---

## 2. 操作数基本规则

常见格式：

```asm
指令  dest, src
指令  reg/mem
```

注意：

```asm
ADD AX, BX      ; 合法，寄存器对寄存器
ADD AX, 1234H   ; 合法，立即数作源操作数
ADD AX, [BX]    ; 合法，内存作源操作数
ADD [BX], AX    ; 合法，内存作目的操作数

ADD [BX], [SI]  ; 错，两个操作数不能同时是内存
MOV 1234H, AX   ; 错，立即数不能作目的操作数
```

如果内存操作数大小不明确，要写明 BYTE PTR 或 WORD PTR：

```asm
INC BYTE PTR [BX]
INC WORD PTR [BX]
NEG BYTE PTR [SI]
```

---

# 一、算术运算指令

8086 算术运算主要处理：

1. 无符号二进制数
2. 带符号二进制数，使用补码
3. 压缩 BCD 码
4. 非压缩 BCD 码 / ASCII 数字

---

## 1. 加法类指令

### ADD：普通加法

```asm
ADD dest, src
```

功能：

```asm
dest = dest + src
```

例：

```asm
ADD AL, 18H
ADD AX, BX
ADD AL, [BX]
```

影响标志位：

```text
CF OF AF ZF SF PF
```

### ADC：带进位加法

```asm
ADC dest, src
```

功能：

```asm
dest = dest + src + CF
```

主要用于多字节 / 多字长加法。

例：32 位加法，低 16 位用 ADD，高 16 位用 ADC：

```asm
ADD AX, CX
ADC DX, BX
```

### INC：自增 1

```asm
INC reg/mem
```

功能：

```asm
reg/mem = reg/mem + 1
```

例：

```asm
INC BL
INC CX
INC BYTE PTR [BX]
```

注意：

```text
INC 不影响 CF
```

---

## 2. 减法类指令

### SUB：普通减法

```asm
SUB dest, src
```

功能：

```asm
dest = dest - src
```

例：

```asm
SUB AX, BX
SUB DX, 1850H
SUB BL, [BX]
```

### SBB：带借位减法

```asm
SBB dest, src
```

功能：

```asm
dest = dest - src - CF
```

主要用于多字节 / 多字长减法。

例：

```asm
SUB AX, CX
SBB DX, BX
```

### DEC：自减 1

```asm
DEC reg/mem
```

功能：

```asm
reg/mem = reg/mem - 1
```

例：

```asm
DEC BX
DEC WORD PTR [BP]
```

注意：

```text
DEC 不影响 CF
```

### NEG：取负

```asm
NEG reg/mem
```

功能：

```asm
reg/mem = 0 - reg/mem
```

也可以理解成：

```text
按位取反 + 1
```

例：

```asm
NEG AX
NEG BYTE PTR [BX]
```

标志位特点：

```text
如果操作数原来是 0，则 CF = 0
如果操作数原来不是 0，则 CF = 1
```

特殊溢出：

```text
8 位数：对 -128 取负会溢出
16 位数：对 -32768 取负会溢出
```

---

## 3. CMP 比较指令

```asm
CMP dest, src
```

功能：

```asm
dest - src
```

但是结果不保存，只影响标志位。

例：

```asm
CMP AL, 80H
CMP BX, DATA1
```

CMP 本质上类似 SUB，但是不改变原操作数。

### 无符号数比较

```text
ZF = 1：两数相等
CF = 1：dest < src
CF = 0 且 ZF = 0：dest > src
```

### 带符号数比较

带符号数不能只看 CF，要看 SF 和 OF。

```text
SF XOR OF = 1：dest < src
SF XOR OF = 0：dest >= src
```

简单记：

```text
CMP 后：
无符号看 CF
有符号看 SF 和 OF
是否相等看 ZF
```

---

# 二、乘法和除法指令

## 1. MUL：无符号乘法

```asm
MUL src
```

注意：MUL 只有一个显式操作数，另一个操作数隐含在 AL 或 AX 中。

### 8 位乘法

```asm
MUL r/m8
```

功能：

```asm
AX = AL * r/m8
```

例：

```asm
MOV AL, 55H
MOV BL, 14H
MUL BL
```

结果放在 AX。

### 16 位乘法

```asm
MUL r/m16
```

功能：

```asm
DX:AX = AX * r/m16
```

例：

```asm
MUL CX
MUL WORD PTR [BX]
```

标志位：

```text
如果结果高半部分不为 0，则 CF = OF = 1
否则 CF = OF = 0
```

---

## 2. IMUL：带符号乘法

```asm
IMUL src
```

功能类似 MUL，但是把操作数看成带符号补码数。

### 8 位

```asm
AX = AL * r/m8
```

### 16 位

```asm
DX:AX = AX * r/m16
```

标志位：

```text
如果高半部分不是低半部分的符号扩展，则 CF = OF = 1
否则 CF = OF = 0
```

---

## 3. DIV：无符号除法

```asm
DIV src
```

### 除数是 8 位

```asm
DIV r/m8
```

要求：

```asm
被除数在 AX
```

结果：

```asm
AL = 商
AH = 余数
```

例：

```asm
MOV AX, 0037H
MOV BL, 05H
DIV BL
```

### 除数是 16 位

```asm
DIV r/m16
```

要求：

```asm
被除数在 DX:AX
```

结果：

```asm
AX = 商
DX = 余数
```

注意：

```text
如果商放不进 AL 或 AX，会产生除法错误中断。
```

---

## 4. IDIV：带符号除法

```asm
IDIV src
```

功能类似 DIV，但是操作数看成带符号数。

余数符号规则：

```text
余数符号和被除数相同
```

使用 IDIV 前，通常要做符号扩展。

---

## 5. CBW 和 CWD：符号扩展

### CBW：字节扩展成字

```asm
CBW
```

功能：

```asm
把 AL 的符号位扩展到 AH
```

也就是：

```text
如果 AL 最高位是 0，则 AH = 00H
如果 AL 最高位是 1，则 AH = FFH
```

用于 8 位带符号数除法前。

### CWD：字扩展成双字

```asm
CWD
```

功能：

```asm
把 AX 的符号位扩展到 DX
```

也就是：

```text
如果 AX 最高位是 0，则 DX = 0000H
如果 AX 最高位是 1，则 DX = FFFFH
```

用于 16 位带符号数除法前。

---

# 三、BCD / ASCII 调整指令

BCD 运算容易出问题，因为 CPU 默认按二进制运算，而 BCD 是十进制规则。

## 1. BCD 类型

### 压缩 BCD

一个字节放两个十进制数字。

例：

```text
59 = 0101 1001B = 59H
```

### 非压缩 BCD

一个字节只放一个十进制数字，低 4 位有效，高 4 位通常为 0。

例：

```text
9 = 0000 1001B = 09H
```

### ASCII 数字

数字字符要加 30H。

例：

```text
'9' = 39H
'5' = 35H
```

---

## 2. DAA：压缩 BCD 加法调整

```asm
DAA
```

使用场景：

```asm
ADD AL, src
DAA
```

功能：

```text
把 AL 中的二进制加法结果调整成压缩 BCD 结果
```

例：

```asm
MOV AL, 27H
ADD AL, 18H
DAA
; AL = 45H
```

---

## 3. DAS：压缩 BCD 减法调整

```asm
DAS
```

使用场景：

```asm
SUB AL, src
DAS
```

功能：

```text
把 AL 中的二进制减法结果调整成压缩 BCD 结果
```

例：

```asm
MOV AL, 27H
SUB AL, 18H
DAS
; AL = 09H
```

---

## 4. AAA：非压缩 BCD / ASCII 加法调整

```asm
AAA
```

使用场景：

```asm
ADD AL, src
AAA
```

功能：

```text
调整 AL，结果放在 AX 中
AH 存十位进位
AL 存个位
```

例：

```asm
MOV AH, 0
MOV AL, 09H
MOV BL, 05H
ADD AL, BL
AAA
; AX = 0104H，表示 14
```

如果原来是 ASCII 数字，最后可转回 ASCII：

```asm
OR AX, 3030H
```

---

## 5. AAS：非压缩 BCD / ASCII 减法调整

```asm
AAS
```

使用场景：

```asm
SUB AL, src
AAS
```

功能：

```text
对非压缩 BCD 或 ASCII 数字减法结果进行调整
```

---

## 6. AAM：非压缩 BCD 乘法调整

```asm
AAM
```

使用场景：

```asm
MUL src
AAM
```

功能：

```asm
AH = AL / 10
AL = AL % 10
```

例：

```asm
MOV AL, 09H
MOV BL, 06H
MUL BL
AAM
; AH = 05H, AL = 04H，表示 54
```

如果是 ASCII 数字相乘，先屏蔽高 4 位：

```asm
AND AL, 0FH
AND BL, 0FH
MUL BL
AAM
OR AX, 3030H
```

---

## 7. AAD：非压缩 BCD 除法调整

```asm
AAD
```

注意：AAD 是除法前使用，不是除法后使用。

使用场景：

```asm
AAD
DIV src
```

功能：

```asm
AL = AH * 10 + AL
AH = 0
```

例：

```asm
; AX = 0307H，表示十进制 37
; BL = 05H

AAD
DIV BL

; AL = 7，商
; AH = 2，余数
```

---

# 四、逻辑运算指令

逻辑运算是按位操作。

## 1. NOT：按位取反

```asm
NOT reg/mem
```

功能：

```text
0 变 1，1 变 0
```

例：

```asm
NOT AX
NOT BL
NOT BYTE PTR [BX]
```

注意：

```text
NOT 不影响标志位
```

---

## 2. AND：按位与

```asm
AND dest, src
```

功能：

```text
dest = dest & src
```

常用作用：清零某些位，保留某些位。

规律：

```text
和 1 相与：保持不变
和 0 相与：清 0
```

例：屏蔽高 4 位，只保留低 4 位：

```asm
AND AL, 0FH
```

例：清除 AL 的第 0、1 位：

```asm
AND AL, 0FCH
```

---

## 3. OR：按位或

```asm
OR dest, src
```

功能：

```text
dest = dest | src
```

常用作用：把某些位置 1。

规律：

```text
和 0 相或：保持不变
和 1 相或：置 1
```

例：把 AL 的第 5 位置 1：

```asm
OR AL, 20H
```

例：把 BCD 转 ASCII：

```asm
OR AX, 3030H
```

---

## 4. XOR：按位异或

```asm
XOR dest, src
```

功能：

```text
dest = dest XOR src
```

常用作用：翻转某些位，或者清零寄存器。

规律：

```text
和 0 异或：保持不变
和 1 异或：取反
```

例：翻转 AL 的第 0、1 位：

```asm
XOR AL, 03H
```

例：清零 AX：

```asm
XOR AX, AX
```

---

## 5. TEST：测试位

```asm
TEST dest, src
```

功能：

```text
dest & src
```

但是结果不保存，只影响标志位。

TEST 类似 AND，但不改变原操作数。

例：测试 AL 的 D7 是否为 1：

```asm
TEST AL, 80H
JNZ T_ALARM
```

如果结果非 0，说明对应位中至少有一位是 1。

---

## 6. 逻辑指令对标志位影响

对 AND、OR、XOR、TEST：

```text
CF = 0
OF = 0
根据结果设置 SF、ZF、PF
AF 不确定
```

NOT：

```text
不影响标志位
```

---

# 五、移位指令

移位指令格式：

```asm
移位指令  操作数, 计数值
```

计数值只能是：

```asm
1
CL
```

也就是说：

```asm
SHL AX, 1
MOV CL, 3
SHL AX, CL
```

---

## 1. SHL / SAL：左移

```asm
SHL dest, count
SAL dest, count
```

SHL 和 SAL 功能相同。

功能：

```text
所有位左移
最低位补 0
最高位移入 CF
```

效果：

```text
左移 1 位，相当于无符号数乘以 2
```

例：

```asm
MOV AH, 00000110B
SHL AH, 1
; AH = 00001100B
```

---

## 2. SHR：逻辑右移

```asm
SHR dest, count
```

功能：

```text
所有位右移
最高位补 0
最低位移入 CF
```

适合无符号数。

效果：

```text
右移 1 位，相当于无符号数除以 2
```

例：

```asm
MOV AL, 10000101B
MOV CL, 3
SHR AL, CL
```

---

## 3. SAR：算术右移

```asm
SAR dest, count
```

功能：

```text
所有位右移
最高位保持原符号位
最低位移入 CF
```

适合带符号数。

效果：

```text
相当于带符号数除以 2
```

例：

```asm
MOV AL, 10000000B
MOV CL, 3
SAR AL, CL
; AL = 11110000B，即 -16
```

---

# 六、循环移位指令

普通移位会丢掉移出去的位。

循环移位不会直接丢失，而是把移出去的位绕回另一端，或者经过 CF 参与循环。

---

## 1. ROL：循环左移

```asm
ROL dest, count
```

功能：

```text
最高位移到最低位，同时进入 CF
```

CF 不作为循环的一部分。

---

## 2. ROR：循环右移

```asm
ROR dest, count
```

功能：

```text
最低位移到最高位，同时进入 CF
```

CF 不作为循环的一部分。

---

## 3. RCL：带进位循环左移

```asm
RCL dest, count
```

功能：

```text
把 CF 也看成循环链条的一部分
最高位进入 CF
原 CF 进入最低位
```

---

## 4. RCR：带进位循环右移

```asm
RCR dest, count
```

功能：

```text
把 CF 也看成循环链条的一部分
最低位进入 CF
原 CF 进入最高位
```

---

## 5. ROL/ROR 和 RCL/RCR 的区别

```text
ROL/ROR：小循环，不带 CF 参与循环
RCL/RCR：大循环，把 CF 也算进循环
```

例：

```asm
ROL BX, CL
ROR WORD PTR [SI], 1
RCL AL, 1
RCR AL, 1
```

---

# 七、标志位总结

常见标志位：

```text
CF：进位 / 借位标志
OF：溢出标志
AF：辅助进位，低 4 位向高 4 位进位
ZF：零标志，结果为 0 则置 1
SF：符号标志，结果最高位为 1 则置 1
PF：奇偶标志，低 8 位中 1 的个数为偶数则置 1
```

---

## 1. ADD / ADC

影响：

```text
CF OF AF ZF SF PF
```

含义：

```text
CF：无符号加法是否进位
OF：带符号加法是否溢出
ZF：结果是否为 0
SF：结果最高位
PF：结果低 8 位中 1 的个数是否为偶数
AF：低 4 位是否向高 4 位进位
```

---

## 2. INC

影响：

```text
OF AF ZF SF PF
```

不影响：

```text
CF
```

---

## 3. SUB / SBB / CMP

影响：

```text
CF OF AF ZF SF PF
```

含义：

```text
CF = 1：无符号减法发生借位，即 dest < src
OF = 1：带符号减法溢出
ZF = 1：两数相等
```

---

## 4. DEC

影响：

```text
OF AF ZF SF PF
```

不影响：

```text
CF
```

---

## 5. NEG

等价于：

```asm
0 - 操作数
```

特点：

```text
操作数为 0：CF = 0
操作数非 0：CF = 1
```

---

## 6. MUL / IMUL

主要看：

```text
CF 和 OF
```

MUL：

```text
高半部分为 0：CF = OF = 0
高半部分不为 0：CF = OF = 1
```

IMUL：

```text
高半部分是符号扩展：CF = OF = 0
否则：CF = OF = 1
```

---

## 7. DIV / IDIV

```text
除法后所有条件标志位无意义
```

---

## 8. 逻辑指令

AND、OR、XOR、TEST：

```text
CF = 0
OF = 0
根据结果设置 SF、ZF、PF
AF 不确定
```

NOT：

```text
不影响标志位
```

---

# 八、最容易考 / 最容易错的点

## 1. ADD 和 ADC 搭配做多字节加法

```asm
ADD AX, low
ADC DX, high
```

低位加法产生的 CF，会被高位 ADC 用上。

---

## 2. SUB 和 SBB 搭配做多字节减法

```asm
SUB AX, low
SBB DX, high
```

低位减法产生的借位 CF，会被高位 SBB 用上。

---

## 3. INC / DEC 不影响 CF

```asm
INC AX
DEC AX
```

它们会影响 ZF、SF、OF 等，但不影响 CF。

---

## 4. CMP 不保存结果

```asm
CMP AX, BX
```

只改标志位，AX 和 BX 原值不变。

---

## 5. TEST 不保存结果

```asm
TEST AL, 80H
```

只改标志位，AL 原值不变。

---

## 6. MUL / DIV 有隐含寄存器

MUL：

```asm
MUL BL      ; AX = AL * BL
MUL BX      ; DX:AX = AX * BX
```

DIV：

```asm
DIV BL      ; AX / BL，商 AL，余数 AH
DIV BX      ; DX:AX / BX，商 AX，余数 DX
```

---

## 7. IDIV 前要符号扩展

8 位被除数：

```asm
CBW
IDIV r/m8
```

16 位被除数：

```asm
CWD
IDIV r/m16
```

---

## 8. BCD 调整指令要记使用位置

```text
DAA：压缩 BCD，加法后
DAS：压缩 BCD，减法后

AAA：非压缩 BCD / ASCII，加法后
AAS：非压缩 BCD / ASCII，减法后
AAM：非压缩 BCD，乘法后
AAD：非压缩 BCD，除法前
```

口诀：

```text
加减乘：调整在后
除法：AAD 在前
```

---

## 9. SHL/SAL 一样

```asm
SHL AX, 1
SAL AX, 1
```

二者在 8086 中功能相同，都是左移补 0。

---

## 10. SHR 和 SAR 区别

```text
SHR：逻辑右移，最高位补 0，适合无符号数
SAR：算术右移，最高位保持符号位，适合带符号数
```

---

## 11. ROL/ROR 和 RCL/RCR 区别

```text
ROL/ROR：普通循环，不带 CF 参与
RCL/RCR：带进位循环，CF 也参与循环
```

---

# 九、常用模板

## 1. 32 位加法

```asm
; DX:AX = DX:AX + BX:CX

ADD AX, CX
ADC DX, BX
```

---

## 2. 32 位减法

```asm
; DX:AX = DX:AX - BX:CX

SUB AX, CX
SBB DX, BX
```

---

## 3. 无符号数乘 10

```asm
XOR AH, AH
SHL AX, 1      ; 2 * AL
MOV BX, AX
SHL AX, 1      ; 4 * AL
SHL AX, 1      ; 8 * AL
ADD AX, BX     ; 10 * AL
```

---

## 4. 测试某一位是否为 1

```asm
TEST AL, 80H
JNZ  LABEL
```

---

## 5. 清某些位

```asm
AND AL, mask
```

例如清第 0、1 位：

```asm
AND AL, 0FCH
```

---

## 6. 置某些位

```asm
OR AL, mask
```

例如置第 5 位：

```asm
OR AL, 20H
```

---

## 7. 翻转某些位

```asm
XOR AL, mask
```

例如翻转第 0、1 位：

```asm
XOR AL, 03H
```

# 十、一句话总览

这一章核心就是：

```text
算术指令负责加减乘除；
调整指令负责让 BCD / ASCII 数字结果正确；
逻辑指令负责按位清零、置位、取反、测试；
移位指令负责乘除 2 或处理二进制位；
循环移位负责让移出的位绕回来，RCL/RCR 还会把 CF 一起算进去。
```
