# 8253 定时计数器摘要

## 课件定位

- 对应课件：`微机原理-第7章8253.pptx`
- 主题：8253/8254 结构、控制字、六种工作方式、定时/分频应用

## 本讲主线

课件把 8253 作为典型可编程定时器来讲。需要掌握的不是所有波形细节，而是控制字含义、六种模式特点和初值计算。

## 页序摘要

### 1. 芯片和引脚

- 8253 有 3 个独立计数通道。
- 每个通道有 `CLK`、`GATE`、`OUT`。
- 另有控制字寄存器统一配置工作模式。

### 2. 控制字

- 指定通道号。
- 指定读写格式。
- 指定工作方式。
- 指定二进制或 BCD 计数。

### 3. 六种工作方式

- 方式 0：计数结束输出。
- 方式 1：可编程单稳。
- 方式 2：分频器。
- 方式 3：方波发生器。
- 方式 4：软件触发选通。
- 方式 5：硬件触发选通。

### 4. 编程与读写

- 先向控制口写控制字。
- 再按规定格式写入计数初值。
- 必要时读取当前计数值或等待 `OUT` 变化。

### 5. 应用举例

- 周期定时。
- 方波产生。
- 事件计数。
- 多路并行高斯或分频题的接口实现。

## 应掌握的结论

- 做题先选模式，再算初值。
- 方式 2 和方式 3 都能分频，但输出波形不同。
- 读写顺序必须与控制字中设定的格式一致。
