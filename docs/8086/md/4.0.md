# 汇编程序设计摘要

## 课件定位

- 对应课件：`微机原理-第4章汇编程序设计.pptx`
- 主题：源程序格式、伪指令、DOS/BIOS 调用、顺序/分支/循环/子程序设计

## 本讲主线

第 4 章把前面学过的指令组织成“可运行的汇编程序”。课件结构很长，但可以压缩成三个部分：程序框架、系统调用、设计方法。

## 页序摘要

### 1. 课程结构和重点

- 课件开头给出三块内容：程序格式与伪指令、DOS/BIOS 调用、程序设计方法。
- 教学重点落在源程序框架、变量/标号、基本程序结构和子程序设计。

### 2. 汇编源程序格式

- 区分硬指令和伪指令。
- 说明名字、标号、操作码、操作数、注释的标准写法。
- 用 `END`、`SEGMENT`、`ENDS`、`ASSUME`、`PROC`、`ENDP` 等伪指令组织程序。

### 3. 常量、变量和操作符

- 介绍 `DB/DW/DD`、字符/字符串定义、重复定义。
- 介绍 `OFFSET`、`SEG`、`PTR`、`TYPE`、`LENGTH` 等操作符。
- 强调变量类型和存储空间大小要和指令匹配。

### 4. 完整程序框架

- 展示含代码段、数据段、堆栈段、附加段的完整模板。
- 说明何时必须设置 `ES`，尤其是字符串处理程序。
- 强调 `DS/SS/ES` 的初始化。

### 5. DOS / BIOS 系统调用

- 用 `INT 21H` 完成字符串输出、键盘输入、程序返回等任务。
- 用 `INT 10H`、`INT 16H` 完成显示和键盘底层服务。
- 课件通过缓冲区输入输出例子说明系统功能号和寄存器约定。

### 6. 顺序、分支、循环程序设计

- 用基本结构组织算法。
- 强调先画流程图，再决定寄存器分工。
- 例题覆盖分类统计、范围判断、查表和数制转换。

### 7. 子程序设计

- 说明参数如何通过寄存器或内存传递。
- 说明返回值如何通过寄存器返回。
- 强调现场保护与恢复。
- 给出十进制转二进制、字符显示等过程例子。

## 应掌握的结论

- 能写出完整汇编程序骨架。
- 能区分硬指令、伪指令、操作符。
- 能把题目拆成顺序、分支、循环、子程序四种结构。
- 需要与操作系统交互时优先想到 `INT 21H`。
