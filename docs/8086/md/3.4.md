# 8086 字符串与控制转移指令摘要

## 课件定位

- 对应课件：`微机原理-第3章4指令之字符串、处理控制转移.pptx`
- 主题：条件转移、调用返回、中断、字符串处理指令与重复前缀

## 本讲主线

这一讲把“程序流程如何改变”和“连续数据如何批量处理”放在一起。前半段是跳转/调用/中断，后半段是字符串指令族。

## 页序摘要

### 1. 控制转移基础

- 介绍近转移、段内/段间转移以及目的地址的不同寻址形式。
- 强调控制转移会改变 `IP`，远转移还会改变 `CS`。

### 2. 条件转移

- 解释 `JZ/JE`、`JNZ/JNE`、`JS`、`JC/JB`、`JA/JAE/JBE` 等含义。
- 说明这些跳转本质上依赖前一条运算或比较指令设置的标志位。
- 区分有符号比较和无符号比较。

### 3. CALL / RET / INT / IRET

- `CALL` 保存返回地址后进入子程序。
- `RET` 从栈中取回返回地址。
- `INT i8` 进入中断服务程序。
- `IRET` 用于中断返回。

### 4. 字符串指令的隐含约定

- 源串通常在 `DS:SI`。
- 目的串通常在 `ES:DI`。
- `CX` 常作为长度计数器。
- `DF` 决定地址自增还是自减。

### 5. 重复前缀

- `REP`：重复执行直到 `CX=0`。
- `REPE/REPZ`：边重复边要求 `ZF=1`。
- `REPNE/REPNZ`：边重复边要求 `ZF=0`。

### 6. 字符串传送与比较

- `MOVSB/MOVSW`：串传送。
- `CMPSB/CMPSW`：串比较。
- 课件用 `REP MOVS`、`REPE CMPSB` 展示批量复制和密码串比较。

### 7. 扫描、装入、存入

- `SCASB/SCASW`：在串中扫描目标值。
- `LODSB/LODSW`：把源串元素装入 `AL/AX`。
- `STOSB/STOSW`：把 `AL/AX` 写入目标串。

## 应掌握的结论

- 条件转移前一定要先看是谁设置了标志位。
- 字符串题先初始化 `SI/DI/CX/DF` 再写主体。
- `REP/REPE/REPNE` 的退出条件要和 `ZF` 一起判断。
