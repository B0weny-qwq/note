# 存储器摘要

## 课件定位

- 对应课件：`微机原理-第5章1存储器.pptx`
- 主题：存储器分类、RAM/ROM/Flash、性能指标、芯片结构、扩展与片选

## 本讲主线

这份课件前半讲器件知识，后半讲接口设计。真正与 8086 实验和习题最紧的是：容量、位宽、地址线、片选和扩展。

## 页序摘要

### 1. 存储器分类

- 从内部存储器、外部存储器、Cache 切入。
- 区分 RAM、ROM、半导体存储器、磁存储器、光存储器。
- 再按易失性、非易失性、随机/顺序/直接存取方式细分。

### 2. RAM

- `SRAM`：速度高、无需刷新、容量较小。
- `DRAM`：容量大、成本低、需要刷新。
- 介绍常见刷新方式和内存条演进。

### 3. ROM 家族

- `PROM`、`EPROM`、`EEPROM`、`Flash` 的擦写方式和用途不同。
- 课件用 `2716/2764/2817A/2864A` 等芯片说明典型引脚和应用。
- 强调掉电保存参数、程序固化等使用场景。

### 4. 性能指标

- 容量。
- 存取时间。
- 存储周期。
- 带宽和成本。

### 5. 芯片结构与典型引脚

- 以 SRAM、EPROM 为例说明地址线、数据线和控制信号。
- 强调片内地址线数量由芯片容量决定。

### 6. 存储器扩展与片选

- 用位扩展解决位宽不足。
- 用字扩展适配 8086 的 16 位数据总线。
- 用容量扩展拼接更大的地址空间。
- 讲解线选法、全译码法、部分译码法三类片选思路。

## 应掌握的结论

- `SRAM` 和 `DRAM` 的区别一定要能快速说清。
- 芯片扩展题先算地址线、再看位宽、最后写片选。
- 8086 连接存储器时必须考虑高低字节体和 `A0/BHE#`。
