# 8086 寻址方式和机器码摘要

## 课件定位

- 对应课件：`微机原理-第3章1寻址方式和机器码.pptx`
- 主题：8086 指令格式、操作数类型、寻址方式、机器码基础

## 本讲主线

这份课件先从“什么是汇编语句”入手，再过渡到“CPU 如何找到操作数”。真正的核心是把各种寻址方式和默认段寄存器理清楚。

## 页序摘要

### 1. 课程导入

- 介绍第 3 章主题：8086 的寻址方式和指令系统。
- 用 `MOV` 例子说明一条指令由操作码和操作数构成。

### 2. 汇编语句与指令格式

- 区分执行性语句和说明性语句。
- 说明操作码、目的操作数、源操作数、注释的书写格式。
- 引出 `i8/i16/r8/r16/m8/m16` 等操作数记号。

### 3. 立即数与寄存器寻址

- 立即数直接写在指令中，只能作源操作数。
- 寄存器寻址速度快，操作数直接来自寄存器。
- 强调 8 位和 16 位寄存器宽度要匹配。

### 4. 存储器寻址方式

- 直接寻址：偏移地址在指令中给出。
- 寄存器间接寻址：偏移地址放在 `BX/BP/SI/DI` 中。
- 相对寻址：寄存器加位移量。
- 基址变址寻址：基址寄存器与变址寄存器组合。
- 相对基址变址寻址：再叠加位移量。

### 5. 默认段寄存器和物理地址

- 默认情况下 `BX/SI/DI` 走 `DS`。
- 只要用了 `BP`，默认段一般变成 `SS`。
- 物理地址按 `段地址 × 10H + 偏移地址` 计算。
- 课件通过多组已知寄存器值练习物理地址求法。

### 6. 机器码基础

- 说明机器指令在代码段中的组织方式。
- 介绍操作码和寻址方式如何体现在机器码中。
- 强调汇编写法和机器内部编码是一一对应关系。

## 应掌握的结论

- 看到内存操作数先判断寻址方式，再判断默认段寄存器。
- 看到 `BP` 默认想到 `SS`。
- 看到 `OFFSET` 取地址，普通变量名多半取内容。
- 立即数不能作双操作数指令的目的操作数。

## 复习重点

- 五类常见寻址方式的写法。
- 每类寻址方式的物理地址计算。
- 执行性语句与说明性语句的区别。
