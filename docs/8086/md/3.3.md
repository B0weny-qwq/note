# 8086 算术、逻辑与移位指令摘要

## 课件定位

- 对应课件：`微机原理-第3章3指令之算数、逻辑、移位指令.pptx`
- 主题：算术运算、乘除法、BCD 调整、逻辑运算、移位与循环移位

## 本讲主线

这一讲把算数类和按位类指令集中梳理，重点不只是公式，而是“哪些标志位会变”和“什么场景该用哪类指令”。

## 页序摘要

### 1. 基本格式回顾

- 再次区分执行性语句与说明性语句。
- 回顾双操作数和单操作数指令的通用规则。

### 2. 加减类指令

- `ADD`、`ADC`、`INC`。
- `SUB`、`SBB`、`DEC`、`NEG`。
- 说明 `ADC/SBB` 在多字节运算中的作用。
- 强调 `INC/DEC` 不影响 `CF`。

### 3. 比较与标志位

- `CMP` 只改标志位，不保存结果。
- 无符号比较主要看 `CF` 和 `ZF`。
- 有符号比较要结合 `SF` 与 `OF`。

### 4. 乘除法

- `MUL/IMUL` 的隐含寄存器规则。
- `DIV/IDIV` 的被除数位置和商余数位置。
- `CBW/CWD` 用于带符号扩展。
- 强调商溢出会触发除法错误。

### 5. BCD / ASCII 调整

- `DAA/DAS` 用于压缩 BCD 加减法后调整。
- `AAA/AAS` 用于非压缩 BCD 或 ASCII 加减法后调整。
- `AAM/AAD` 分别服务乘法后、除法前处理。

### 6. 逻辑运算

- `NOT`、`AND`、`OR`、`XOR`、`TEST`。
- 说明清零、置位、翻转、掩码测试等典型用途。
- 强调 `TEST` 和 `CMP` 类似，只改标志位。

### 7. 移位和循环移位

- `SHL/SAL`、`SHR`、`SAR`。
- `ROL/ROR`、`RCL/RCR`。
- 说明逻辑右移和算术右移的区别。
- 说明 `RCL/RCR` 会把 `CF` 一起纳入循环链。

## 应掌握的结论

- 多精度加减一定想到 `ADC/SBB`。
- 带符号除法前常要 `CBW/CWD`。
- `SHR` 适合无符号数，`SAR` 适合有符号数。
- 逻辑题大多都能转成掩码与位操作。
