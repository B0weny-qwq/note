# 8086 数据传送指令摘要

## 课件定位

- 对应课件：`微机原理-第3章2指令之数据传送.pptx`
- 主题：数据传送类指令、寻址应用、堆栈、I/O 端口、地址传送

## 本讲主线

这部分围绕“数据从哪里来到哪里去”展开。课件先复习寻址方式，然后系统整理 `MOV`、`PUSH/POP`、`XCHG`、`IN/OUT`、`LEA/LDS/LES` 等指令。

## 页序摘要

### 1. 寻址方式复习

- 通过 `MOV AX,[SI+06H]`、`MOV AX,[BX+SI]` 等例子复习源操作数寻址。
- 配合已知 `DS/ES/SS/BX/BP/SI` 数值计算物理地址。

### 2. 数据传送类指令总览

- 常见来源和去向包括寄存器、存储器、立即数、段寄存器、端口、标志寄存器。
- 大多数数据传送类指令不影响标志位。

### 3. MOV

- 给出合法格式：寄存器、内存、立即数之间的传送。
- 强调位宽匹配、`BYTE PTR/WORD PTR` 的必要性。
- 强调不能内存到内存。
- 强调段寄存器不能直接接立即数，`CS` 不能作为 `MOV` 目的操作数。

### 4. 数据定义和地址

- 结合 `DB`、`DW`、`DUP` 说明变量布局。
- 用 `OFFSET` 区分“取内容”和“取偏移地址”。
- 引出小端存储规则。

### 5. PUSH / POP

- 说明栈在 `SS` 段中，`SP` 指向栈顶。
- 栈按字操作，方向为高地址到低地址。
- 说明入栈、出栈过程中 `SP` 的变化。
- 举例说明后进先出。

### 6. XCHG 和 XLAT

- `XCHG` 用于寄存器和寄存器/内存交换。
- `XLAT` 用于查表，隐含使用 `BX` 和 `AL`。

### 7. IN / OUT

- 区分直接端口地址和 `DX` 间接端口地址。
- 强调数据寄存器固定为 `AL/AX`。
- 强调端口号超过 `FFH` 必须走 `DX`。

### 8. LEA / LDS / LES

- `LEA` 取有效地址，不取内容。
- `LDS`、`LES` 从内存装入远指针。
- 对比 `LEA` 和 `MOV` 的差别。

### 9. 标志寄存器传送

- `LAHF/SAHF` 处理标志低字节。
- `PUSHF/POPF` 保存和恢复整个标志寄存器。

## 应掌握的结论

- `MOV` 最容易错在宽度、段寄存器、内存到内存。
- `PUSH/POP` 只操作字。
- `IN/OUT` 只允许 `AL/AX` 传数据。
- `LEA` 取地址，`MOV` 取内容。
