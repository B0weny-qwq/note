# 信息论与编码 第 4 章摘要

## 课件定位

- 对应课件：`第4章 信息率失真函数.pdf`
- 主题：失真函数、平均失真、离散率失真函数、连续率失真函数

## 本讲主线

这章讨论有损压缩的理论下界：允许一定失真后，最少需要多少码率。

## 页序摘要

### 1. 失真度量

- 定义失真函数。
- 用失真矩阵表示离散信源的失真。
- 计算平均失真。

### 2. 离散信源率失真函数

- 由互信息最小化定义 `R(D)`。
- 分析 `R(D)` 的曲线和 `Dmax`。
- 讨论性质：单调不增、下凸。

### 3. 一般计算方法

- 课件给出参量表达式法。
- 通过给定概率分布与失真矩阵求 `R(D)`。

### 4. 连续信源率失真函数

- 讨论平方误差失真。
- 给出高斯信源的经典结果。

## 应掌握的结论

- `R(D)` 是最小互信息，不是最大互信息。
- `D=0` 回到无失真编码边界。
- `D` 增大时，理论最低码率下降。
