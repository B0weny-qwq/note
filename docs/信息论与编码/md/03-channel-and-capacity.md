# 信息论与编码 第 3 章摘要

## 课件定位

- 对应课件：`第3章 信道与信道容量.pdf`
- 主题：离散信道、BSC、强对称/准对称信道、DMC、连续信道、香农公式

## 本讲主线

第 3 章的中心任务是定义容量，并学会在不同信道模型下求容量。

## 页序摘要

### 1. 信道模型与容量概念

- 给出编码信道和狭义信道模型。
- 容量定义为互信息最大值。

### 2. 二进制对称信道 BSC

- 通过信道矩阵描述转移概率。
- 推出容量与误码率的关系。

### 3. 单符号离散信道容量

- 讨论强对称信道和准对称信道。
- 说明何时输入等概率可达到容量。

### 4. 离散无记忆序列信道

- 从单符号容量推广到序列容量。
- 利用无记忆性分解分析。

### 5. 连续信道及其容量

- 分析高斯信道容量。
- 引入带宽、功率、信噪比对容量的影响。
- 给出香农公式。

## 应掌握的结论

- `C=max I(X;Y)` 是全章核心。
- BSC、强对称信道、高斯信道是最常用的三种模板。
