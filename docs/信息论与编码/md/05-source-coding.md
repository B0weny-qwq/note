# 信息论与编码 第 5 章摘要

## 课件定位

- 对应课件：`第5章 信源编码.pdf`
- 主题：唯一可译码、即时码、平均码长、香农/费诺/哈夫曼/算术编码

## 本讲主线

这一章讨论如何构造尽量短、同时可正确译码的码。

## 页序摘要

### 1. 编码分类与基本要求

- 定长码与变长码。
- 唯一可译码、即时码和非即时码的关系。

### 2. 码树和判定条件

- 用码树表示前缀关系。
- 判断码字集合是否满足即时码要求。
- 计算平均码长和编码效率。

### 3. 香农编码、费诺编码、哈夫曼编码

- 香农码构造规则清晰。
- 费诺码依概率递归分组。
- 哈夫曼码给出最优前缀码。

### 4. 算术编码

- 把整个符号序列映射到一个概率区间。
- 压缩性能更接近理论极限。

### 5. 后续扩展

- 课件后部还涉及量化和常见编码方法简介。

## 应掌握的结论

- 构造最优变长前缀码优先想到哈夫曼编码。
- 即时码一定唯一可译，反过来不成立。
- 编码效率靠平均码长与熵的接近程度衡量。
