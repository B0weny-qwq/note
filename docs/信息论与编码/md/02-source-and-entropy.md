# 信息论与编码 第 2 章摘要

## 课件定位

- 对应课件：`第2章 信源及信源熵.pdf`
- 主题：信源建模、自信息、熵、互信息、熵率、冗余度

## 本讲主线

这一章把“消息”抽象成随机变量或随机过程，再用概率定义平均信息量。

## 页序摘要

### 1. 信源的描述和分类

- 区分离散/连续、单符号/序列、无记忆/有记忆信源。
- 强调用概率论语言描述信源。

### 2. 单符号离散信源

- 定义自信息 `I(x)=-log p(x)`。
- 定义熵 `H(X)` 作为平均不确定度。
- 说明等概率分布时熵最大。

### 3. 条件熵与互信息

- 已知一个变量后剩余不确定度用条件熵表示。
- 两个变量共享的信息量用互信息表示。

### 4. 离散序列信源的熵

- 引出熵率。
- 强调相关性会降低单位符号平均信息量。

### 5. 连续信源的熵和互信息

- 用概率密度和微分熵描述连续情形。
- 与离散熵做性质区分。

### 6. 冗余度

- 用来衡量信源中可压缩的冗余部分。

## 应掌握的结论

- 熵是无失真压缩极限的基础。
- 互信息既在本章出现，也会在信道容量和率失真中继续出现。
