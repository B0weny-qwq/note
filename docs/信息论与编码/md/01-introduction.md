# 信息论与编码 第 1 章摘要

## 课件定位

- 对应课件：`第1章 绪论.pdf`
- 主题：课程目标、研究对象、信息论与编码的基本问题

## 本讲主线

绪论主要搭建全课程视角：信息论负责建立数学极限，编码理论负责构造实现方法。

## 页序摘要

### 1. 课程说明

- 介绍教材、参考书、考核要求和资料渠道。
- 说明本课程是电子通信类的重要专业基础课。

### 2. 信息论与编码的研究范围

- 狭义信息论：以香农理论为核心，研究信息量度和高效、可靠、安全传输。
- 广义信息论：扩展到识别、翻译、控制等更广的领域。

### 3. 通信系统模型

- 信源、编码器、信道、解码器、信宿。
- 课程的后续各章都围绕这个模型展开。

### 4. 本课程三条主线

- 信息量如何度量。
- 信源如何压缩。
- 信道中如何可靠传输。

## 应掌握的结论

- 熵、互信息、容量是全课最核心的三个量。
- 信源编码和信道编码分别对应压缩与抗差错两个方向。
