# 信息论与编码 第 6 章摘要

## 课件定位

- 对应课件：`第6章 信道编码.pdf`
- 主题：检错纠错思想、线性分组码、汉明码、循环码、卷积码

## 本讲主线

第 6 章讨论如何通过增加监督位来提升传输可靠性。

## 页序摘要

### 1. 差错控制方式

- 反馈重发。
- 前向纠错。
- 混合纠错。

### 2. 线性分组码

- 用生成矩阵和校验矩阵描述码。
- 从向量空间角度构造许用码组。

### 3. 码距与纠错能力

- 最小码距决定检错能力和纠错能力。
- 给出典型定理和例题。

### 4. 汉明码

- 说明监督位个数关系。
- 通过校验矩阵和伴随式定位错误。

### 5. 循环码

- 介绍循环移位封闭性质。
- 用生成多项式构造系统循环码。
- 分析编码、校验和译码流程。

### 6. 卷积码与后续方案

- 介绍卷积码、状态图、网格图。
- 课件后部还引到 Turbo、TCM 等现代方案。

## 应掌握的结论

- `G/H`、最小码距、伴随式是分组码题的三件套。
- 汉明码是单错纠正的标准模型。
- 循环码常通过多项式方法处理。
