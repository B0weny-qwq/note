# 怎么写 8255 控制字判断题

## 这类题在考什么

这类题通常考：

```text
根据题目要求判断 8255 的工作方式
再写出或判断控制字
```

这是 8255 章节最核心的考试题型。

## 先记公式

做题时最重要的不是死背某一个数，  
而是记住控制字分析顺序：

```text
先判断是模式设置字还是 BSR
再看 A 组模式和方向
再看 B 组模式和方向
最后再看 PC 各位用途
```

## 做题步骤

### 第一步：先看题目要求的是“整体模式配置”还是“单个位操作”

- 配整组端口模式，想到模式设置字
- 只改 `PC` 某一位，想到 `BSR`

### 第二步：看 A 组和 B 组各做输入还是输出

这一步通常直接来自题意。

### 第三步：确定工作模式

最常见入门题先区分：

- 模式 0：简单输入输出
- 模式 1：带握手
- 模式 2：A 组双向

### 第四步：最后再把控制字写出来或反推

考试里经常先让你“判断对不对”，  
所以前面分析过程比死记数字更重要。

## 最简单例子

如果题目要求：

```text
PA 输出，PB 输入，工作在模式 0
```

那你应先判断：

```text
这是模式设置字，不是 BSR
```

再根据：

- A 组模式 0
- PA 输出
- B 组模式 0
- PB 输入

去写控制字。

## 易错点

- 把 BSR 和模式字混掉。
- 模式 2 误认为 B 组也能用。
- 只看 PA/PB，忘了 PC 某些位可能参与握手。

## 注意点

- 小白做题时先把题目翻译成“哪个口输入、哪个口输出、要不要握手”。
- 先分析结构，再写控制字，错误会少很多。
