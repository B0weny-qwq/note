# 怎么做存储器扩展地址范围题

## 这类题在考什么

这类题通常考：

```text
给一片或几片存储器
问地址范围怎么分配
```

这是微机原理里很典型的硬件计算题。

## 先记公式

先记最基础关系：

```text
容量 = 地址单元数 × 每单元位宽
```

做地址范围时常抓：

```text
起始地址 + 容量 - 1 = 结束地址
```

## 做题步骤

### 第一步：先看芯片容量

明确它一片能覆盖多少地址单元。

### 第二步：算出片内需要多少地址线

这决定低位地址线哪些用来片内寻址。

### 第三步：再确定起始地址

题目常会给某片从哪里开始映射。

### 第四步：用“起始地址 + 容量 - 1”求结束地址

这是最直接的做题入口。

## 最简单例子

如果某片存储器容量覆盖：

```text
4KB
```

并且起始地址是：

```text
8000H
```

则结束地址为：

```text
8000H + 1000H - 1 = 8FFFH
```

所以地址范围是：

```text
8000H ~ 8FFFH
```

## 易错点

- 结束地址忘了减 1。
- 容量单位 `KB` 和十六进制范围换算错。
- 起止地址写成长度。

## 注意点

- 小白先把“起始 + 容量 - 1”这条固定住。
- 地址范围题很多分就卡在最后这一小步。
