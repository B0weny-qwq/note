# 怎么区分 8255 模式0、模式1、模式2

## 这类题在考什么

这类题通常考：

```text
某个应用场景该用 8255 的哪种模式
```

常见是选择题或判断题。

## 先记公式

最稳的三句话：

```text
模式0：简单输入输出，无握手
模式1：选通输入输出，有握手
模式2：双向总线方式，仅 A 组支持
```

## 做题步骤

### 第一步：先看是否需要握手

如果不需要严格同步、只是简单读写，  
优先想到模式 0。

### 第二步：如果需要状态线配合、要收发同步

这时想到模式 1。

### 第三步：如果题目强调双向总线

并且主要围绕 A 组，  
那要想到模式 2。

### 第四步：顺手记住模式 2 的限制

它不是所有组都能用，  
通常只和 A 组有关。

## 最简单例子

如果题目说：

```text
控制 LED、读取简单开关，不需要握手
```

那最合适的是：

```text
模式 0
```

如果题目说：

```text
与打印机之类设备做选通传输，需要握手
```

更应想到：

```text
模式 1
```

## 易错点

- 只要看见输入输出就默认模式 0。
- 忘了模式 1 的关键是握手。
- 忘了模式 2 只对 A 组有特殊意义。

## 注意点

- 小白先抓应用关键词：`简单 I/O`、`握手`、`双向总线`。
- 这类题靠场景判断比背表格更稳。
