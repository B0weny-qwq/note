# 怎么分析 8086 奇偶存储体

## 为什么 8086 有奇偶存储体

8086 的数据总线是 16 位：

```text
D15~D0
```

但内存按字节编址。地址 `00000H` 是一个字节，`00001H` 也是一个字节。

如果用 $8$ 位存储芯片组成 16 位存储器，就必须分成两组：

```text
偶地址存储体 -> D7~D0
奇地址存储体 -> D15~D8
```

## A0 和 BHE# 怎么分工

`A0` 判断低 8 位是否参与。

`BHE#` 表示高 8 位数据允许，低有效。

| 访问 | `A0` | `BHE#` | 选中情况 |
| --- | --- | --- | --- |
| 偶地址字节 | 0 | 1 | 只选偶体 |
| 奇地址字节 | 1 | 0 | 只选奇体 |
| 偶地址字 | 0 | 0 | 奇偶体同时选中 |
| 奇地址字 | 1 | 0 | 先访问奇体，再访问下一偶地址 |

做题记法：

```text
A0 选偶体，BHE# 选奇体。
```

更准确地说，偶体片选常由 `A0=0` 参与形成，奇体片选常由 `BHE#=0` 参与形成。

## 为什么芯片地址线要右移一位

对 8086 来说，一个字地址包含两个字节地址。

如果 $8$ 位芯片做偶体或奇体，芯片内部相邻单元对应 CPU 地址每次加 2。

所以芯片地址线从 CPU 的 `A1` 开始接，CPU 的 `A0` 留给奇偶体选择。

例：2732 有 12 根地址线 `A11~A0`。

接到 8086 时：

```text
芯片 A11~A0 <- CPU A12~A1
```

6264 有 13 根地址线 `A12~A0`。

接到 8086 时：

```text
芯片 A12~A0 <- CPU A13~A1
```

考点：`A0` 不进芯片地址脚，它用来区分偶体。

## PPT 8086 连接例题

题目使用：

- 4 片 2732 EPROM，容量 $4\mathrm{K}\times8$。
- 2 片 6264 SRAM，容量 $8\mathrm{K}\times8$。
- 8086 数据总线 16 位。
- ROM 从 `00000H` 开始，ROM 和 RAM 地址连续。

目标：

```text
ROM：8K 字
RAM：8K 字
```

8086 中 1 个字是 2 字节，所以：

```math
8\mathrm{K\ words}=16\mathrm{KB}
```

因此：

```text
ROM 地址范围：00000H~03FFFH
RAM 地址范围：04000H~07FFFH
```

## ROM 怎么分组

2732 是 $4\mathrm{K}\times8$。

两片并联成一组：

```text
一片做偶体，接 D7~D0
一片做奇体，接 D15~D8
```

一组容量为：

```math
4\mathrm{K\ words}=8\mathrm{KB}
```

4 片 2732 分成 2 组：

| 组 | 字节地址范围 | 作用 |
| --- | --- | --- |
| ROM1 | `00000H~01FFFH` | 前 4K 字 |
| ROM2 | `02000H~03FFFH` | 后 4K 字 |

组内芯片地址线：

```text
2732 A11~A0 <- CPU A12~A1
```

组间选择：

```text
A13=0 -> ROM1
A13=1 -> ROM2
```

## RAM 怎么分组

6264 是 $8\mathrm{K}\times8$。

两片并联：

```text
一片偶体，接 D7~D0
一片奇体，接 D15~D8
```

形成：

```math
8\mathrm{K\ words}=16\mathrm{KB}
```

地址范围：

```text
04000H~07FFFH
```

芯片地址线：

```text
6264 A12~A0 <- CPU A13~A1
```

RAM 位于 `04000H~07FFFH`，其关键高位为：

```text
A14=1，A19~A15=00000
```

## 片选逻辑怎么读

先看大范围：

| 地址范围 | 大范围判断 |
| --- | --- |
| `00000H~03FFFH` | ROM 区 |
| `04000H~07FFFH` | RAM 区 |

再看组内：

| 细分条件 | 选中对象 |
| --- | --- |
| ROM 区且 `A13=0` | ROM1 |
| ROM 区且 `A13=1` | ROM2 |
| RAM 区 | 6264 RAM 组 |

最后看奇偶体：

| 条件 | 选中存储体 |
| --- | --- |
| `A0=0` | 偶体可能选中 |
| `BHE#=0` | 奇体可能选中 |

只有“大范围 + 组选择 + 奇偶体选择”同时满足，具体芯片才被选中。

## 做题模板

遇到 8086 存储器连接题，按 5 步写：

1. 写 CPU 数据总线宽度：8086 是 16 位。
2. 看芯片位宽：若芯片是 8 位，必须分奇偶体。
3. 算每组容量：两片 $n\mathrm{K}\times8$ 组成 $n\mathrm{K}$ 字，也就是 $2n\mathrm{KB}$。
4. 写地址范围：用字节地址写闭区间。
5. 写片选条件：大范围译码、组选择、`A0/BHE#` 奇偶选择。

## 易错点

- 把 `8K 字` 当成 `8KB`。8086 中 `8K 字 = 16KB`。
- 把 CPU `A0` 接到芯片地址脚。
- 忘记偶地址字访问会同时选中奇体和偶体。
- 只写 ROM/RAM 大范围，没有写 ROM1、ROM2 或奇偶体。
- ROM 和 RAM 控制线混接。ROM 只读，RAM 要接读写控制。
