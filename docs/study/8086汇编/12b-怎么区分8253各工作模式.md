# 怎么区分 8253 各工作模式

## 这类题在考什么

这类题通常考：

```text
8253 六种模式里
某个应用场景该选哪一种
```

通常是概念判断题。

## 先记公式

入门复习最先抓这几种：

```text
模式0：终端计数
模式2：分频脉冲
模式3：方波发生器
```

另外：

```text
模式1：单稳态
模式4：软件触发选通脉冲
模式5：硬件触发选通脉冲
```

## 做题步骤

### 第一步：先看题目要的是“定时到点”还是“周期输出”

如果只是到某个时刻给一次结果，  
常想到模式 0。

### 第二步：如果要稳定周期脉冲或分频

优先想到模式 2。

### 第三步：如果题目强调方波输出

优先想到模式 3。

### 第四步：如果题目强调软触发还是硬触发

- 软件触发想到模式 4
- 硬件触发想到模式 5

## 最简单例子

如果题目说：

```text
用 8253 产生规则时钟节拍
```

通常优先想到：

```text
模式 2
```

如果题目说：

```text
要产生近似方波驱动蜂鸣器
```

通常优先想到：

```text
模式 3
```

## 易错点

- 模式 2 和模式 3 混掉。
- 只背模式号，不知道应用场景。
- 看到“定时”两个字就一律选模式 0。

## 注意点

- 小白先别一次全背细节，先把 `0/2/3` 三个最常考的抓住。
- 这类题最值钱的是“场景 -> 模式”的快速映射。
