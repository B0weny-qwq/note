# 怎么准备字符串指令现场

## 这类题在考什么

这类题通常考：

```text
为什么字符串指令执行前总要先配好几个寄存器
```

像 `MOVSB`、`CMPSB`、`SCASB` 都常这样考。

## 先记公式

字符串指令常见依赖：

```text
SI -> 源地址
DI -> 目的地址或扫描地址
CX -> 次数
DF -> 方向
```

并且：

```text
CLD -> 正向
STD -> 反向
```

## 做题步骤

### 第一步：先明确源和目的

源通常放在 `DS:SI`，  
目的常在 `ES:DI`。

### 第二步：再给 `CX` 装次数

很多重复前缀都要用它。

### 第三步：显式设置方向标志

最常见是：

```asm
CLD
```

表示正向处理。

### 第四步：再执行字符串指令

顺序别反了。

## 最简单例子

如果要复制 `len` 个字节：

```asm
CLD
MOV SI, OFFSET src
MOV DI, OFFSET dst
MOV CX, len
REP MOVSB
```

这里：

- `SI` 给源地址
- `DI` 给目的地址
- `CX` 给次数
- `CLD` 保证正向

## 易错点

- 忘了设置 `DF`。
- `MOVSB` 前没准备 `ES:DI`。
- `MOVSW` 时 `CX` 还按字节数装。

## 注意点

- 小白先记住字符串题四要素：`SI DI CX DF`。
- 这四个没配好，后面指令背再熟也会错。
