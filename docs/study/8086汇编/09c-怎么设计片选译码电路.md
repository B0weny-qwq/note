# 怎么设计片选译码电路

## 片选题的核心

CPU 访问存储器时，地址线要完成两件事：

```text
低位地址线选片内单元
高位地址线产生片选信号
```

片选译码题只问一件事：某个地址来时，哪一片芯片被选中。

## 74LS138 要先会读

74LS138 的输出低有效。只有使能条件满足时，`CBA` 的编码才决定哪个 `Y#` 输出为 0。

使能条件：

```text
G1 = 1
G2A# = 0
G2B# = 0
```

输出：

| `CBA` | 有效输出 |
| --- | --- |
| `000` | `Y0#` |
| `001` | `Y1#` |
| `010` | `Y2#` |
| `011` | `Y3#` |
| `100` | `Y4#` |
| `101` | `Y5#` |
| `110` | `Y6#` |
| `111` | `Y7#` |

做题时要确认 `A`、`B`、`C` 哪根接哪根地址线。PPT 按 `CBA` 写编码。

## 线选法

线选法直接用高位地址线做片选。

PPT 例 5.5 用 2 个 2764 EPROM。2764 容量为：

```math
8\mathrm{K}\times8=2000\mathrm{H}
```

片内地址线为：

```text
A12~A0
```

若某片在一个窗口中从 `04000H` 开始，则结束地址：

```math
04000\mathrm{H}+2000\mathrm{H}-1=05FFF\mathrm{H}
```

线选法没有让全部高位参加译码，`A19~A15` 可以变化。PPT 举出的镜像地址包括：

```text
84000H~85FFFH
C4000H~C5FFFH
```

考点：线选法要写出“电路简单，但有地址重叠”。如果题目问优缺点，这句话就是得分点。

## 全译码法

全译码法让所有固定高位都参加片选。

PPT 例 5.6：把 27128 EPROM 映射到：

```text
1C000H~1FFFFH
```

27128 容量：

```math
16\mathrm{KB}=4000\mathrm{H}=2^{14}
```

所以低 14 位 `A13~A0` 接芯片地址线。

地址范围的固定高位：

```text
A19~A14 = 000111
```

译码思路：

- `A17 A16 A15 = 011` 接 74LS138 的 `CBA`，选 `Y3#`。
- `A19=0`、`A18=0`、`A14=1` 放入使能逻辑。
- `Y3#` 接 27128 的片选端。

这样只有 `1C000H~1FFFFH` 会选中该芯片。

考点：全译码题要说明“地址唯一，无镜像”。

## 部分译码法

部分译码法只让一部分高位地址参加译码。

PPT 例 5.7：用 4 片 2732 EPROM 组成 $16\mathrm{K}\times8$，从 `10000H` 开始。

2732 容量：

```math
4\mathrm{KB}=1000\mathrm{H}
```

4 片连续地址：

```text
10000H~13FFFH
```

片内地址线：

```text
A11~A0
```

选片地址线：

```text
A13 A12
```

PPT 方案选 `A16~A12` 参与译码，不译 `A19~A17`。这会产生镜像地址。

镜像规律：

```text
A19~A17 可以取 000~111
```

所以除 `10000H~13FFFH` 外，还会有：

```text
30000H~33FFFH
50000H~53FFFH
70000H~73FFFH
90000H~93FFFH
B0000H~B3FFFH
D0000H~D3FFFH
F0000H~F3FFFH
```

考点：部分译码可以使用，但必须写出镜像地址。

## 设计片选电路的步骤

### 第一步：算单片大小

确定每片芯片覆盖多少地址。

例如 2732：

```math
4\mathrm{KB}=1000\mathrm{H}
```

### 第二步：确定片内地址线

2732 用 `A11~A0`，27128 用 `A13~A0`，6264 用 `A12~A0`。

### 第三步：确定高位固定条件

根据起始地址和容量写出固定高位。

### 第四步：选择译码方式

- 要地址唯一，用全译码。
- 电路简单且允许镜像，用部分译码。
- 小规模临时设计可用线选法，但要防止多片同时选中。

### 第五步：写地址范围

每片都写闭区间。

```text
起始地址~结束地址
```

结束地址用：

```math
\text{起始地址}+\text{容量}-1
```

## 易错点

- 忘记 74LS138 输出低有效。
- 只看 `CBA`，不看使能端。
- 全译码题漏掉某一位固定高位，导致实际变成部分译码。
- 部分译码题不写镜像地址。
- 地址范围写成半开区间，少算或多算 1 个单元。
