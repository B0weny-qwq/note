# 怎么判断默认段寄存器

## 这类题在考什么

这类题通常考：

```text
一条内存访问指令默认用 DS 还是 SS
什么时候会用 ES
```

这是物理地址题的前半步。

## 先记公式

最重要的规则直接记成两句：

```text
没有 BP，默认多半是 DS
有 BP，默认多半是 SS
```

如果显式写了段超越前缀：

```text
以显式写出的段寄存器为准
```

## 做题步骤

### 第一步：先看寻址表达式里有没有 `BP`

比如：

- `[BX]`
- `[SI]`
- `[DI]`

这些通常默认 `DS`。

而：

- `[BP]`
- `[BP+SI]`
- `[BP+DI]`

这些通常默认 `SS`。

### 第二步：再看有没有段超越前缀

如果题目写：

```asm
MOV AL, ES:[BX]
```

那就直接用 `ES`，  
不再按 `DS` 默认规则。

### 第三步：字符串题额外注意 `ES:DI`

像 `STOS`、`SCAS` 这类字符串指令，  
常和 `ES:DI` 有关。

## 最简单例子

如果题目给：

```asm
MOV AL, [BX]
```

那默认段寄存器是：

```text
DS
```

如果题目给：

```asm
MOV AL, [BP+4]
```

那默认段寄存器是：

```text
SS
```

## 易错点

- 看到内存操作数就一律当 `DS`。
- 忘了 `BP` 的特殊性。
- 显式写了 `ES:` 还继续按默认段判断。

## 注意点

- 默认段判断错了，后面的物理地址通常整题都错。
- 小白做题时先盯 `BP`，这是最省事的判断入口。
