# 怎么根据 CMP 选择条件跳转

## 这类题在考什么

这类题通常考：

```text
CMP 之后到底该接 JA/JB 还是 JG/JL
```

这是 8086 分支题最容易丢分的点。

## 先记公式

无符号比较常用：

```text
JA  >    JAE >=
JB  <    JBE <=
```

有符号比较常用：

```text
JG  >    JGE >=
JL  <    JLE <=
```

相等判断：

```text
JE/JZ
JNE/JNZ
```

## 做题步骤

### 第一步：先看题目中的数是“有符号”还是“无符号”

这是第一判断点。  
如果这一步错了，跳转基本就全错。

### 第二步：如果是无符号，优先想到 `JA/JB`

比如地址、长度、纯正整数计数，  
很多都按无符号看。

### 第三步：如果是有符号，优先想到 `JG/JL`

比如正负比较、温度偏差这类可正可负量，  
更像有符号比较。

### 第四步：`CMP` 后尽量马上跟跳转

因为中间插入别的改标志位指令，  
会把比较结果冲掉。

## 最简单例子

如果题目说：

```text
比较两个无符号数 AX 和 BX，若 AX > BX 则跳转
```

应写：

```asm
CMP AX, BX
JA  greater_u
```

如果题目说：

```text
比较两个有符号数 AX 和 BX，若 AX > BX 则跳转
```

应写：

```asm
CMP AX, BX
JG  greater_s
```

## 易错点

- 无符号比较错用 `JG/JL`。
- 有符号比较错用 `JA/JB`。
- `CMP` 后又写了会改标志位的指令。

## 注意点

- 小白先别背太多，先牢牢记住：
  `JA/JB` 给无符号，`JG/JL` 给有符号。
- 这是考试里特别爱出的选择题和改错题。
