# 怎么求 8253 初值

## 这类题在考什么

这类题通常考：

```text
已知输入时钟和目标时间/频率
求计数初值 N
```

这是 8253 最高频的计算题。

## 先记公式

常用关系：

```math
T=\frac{N}{f_{clk}}
```

```math
f_{out}=\frac{f_{clk}}{N}
```

参数解释：

- $T$：定时时间
- $N$：计数初值
- $f_{clk}$：输入时钟频率
- $f_{out}$：输出频率

## 做题步骤

### 第一步：先看题目给的是“时间”还是“频率”

- 给时间，用 `N = T \times fclk`
- 给输出频率，用 `N = fclk / fout`

### 第二步：统一单位

这是最容易错的地方。  
要把：

- MHz
- kHz
- ms
- us

统一好再算。

### 第三步：算出 `N`

再看它是否在 16 位计数范围内。

### 第四步：把 `N` 拆成低字节和高字节

因为很多题最后还会问你怎么写入。

## 最简单例子

如果输入时钟：

```text
1 MHz
```

要求定时：

```text
1 ms
```

则：

```math
N = T \times f_{clk} = 1ms \times 1MHz = 1000
```

所以初值应为：

```text
1000 = 03E8H
```

## 易错点

- 单位没统一，直接错一千倍。
- 只会算十进制，不会转成十六进制写初值。
- 算出 `N` 后忘了检查范围。

## 注意点

- 小白先记：`给时间乘，给频率除`。
- 8253 初值题本质上就是单位换算题 + 一个公式。
