# 怎么求 8086 物理地址

## 这类题在考什么

这类题通常考：

```text
已知段寄存器和偏移地址
求物理地址
```

这是 8086 最高频的基础计算题之一。

## 先记公式

```math
\text{物理地址}=\text{段地址}\times 10H+\text{偏移地址}
```

参数解释：

- 段地址：通常来自 `CS/DS/SS/ES`
- 偏移地址：通常来自 `IP/BX/SI/DI/BP` 或立即位移

## 做题步骤

### 第一步：先判断该用哪个段寄存器

不是所有题都默认 `DS`。  
必须先看寻址方式。

### 第二步：再找偏移地址

偏移地址可能是：

- 直接给定常数
- 寄存器内容
- 寄存器相加后再加位移

### 第三步：代公式

把段地址左移一位十六进制位，  
也就是乘 `10H`，  
再加偏移地址。

### 第四步：结果写成十六进制

8086 题目里通常默认十六进制表达。

## 最简单例子

如果：

```text
DS = 2000H
SI = 00ADH
```

指令是：

```asm
MOV AL, [SI]
```

则物理地址为：

```math
2000H \times 10H + 00ADH = 200ADH
```

## 易错点

- 还没判断默认段，就直接拿 `DS` 去算。
- `10H` 是十六进制的 16，不是十进制 10。
- 加法时十六进制进位算错。

## 注意点

- 小白先固定流程：`先找段 -> 再找偏移 -> 再套公式`。
- 只要这个顺序不乱，物理地址题就很稳。
