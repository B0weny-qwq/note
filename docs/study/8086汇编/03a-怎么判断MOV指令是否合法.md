# 怎么判断 MOV 指令是否合法

## 这类题在考什么

这类题通常考：

```text
给出几条 MOV 指令
判断哪些能写，哪些不能写
```

这是 8086 最常见的选择题之一。

## 先记公式

先记最关键几条规则：

```text
两个操作数不能同时是内存
位宽必须匹配
立即数不能作目的操作数
段寄存器传送有限制
```

## 做题步骤

### 第一步：先看是不是“内存到内存”

比如：

```asm
MOV [BX], [SI]
```

这是非法的。

### 第二步：再看位宽是否一致

比如：

```asm
MOV AL, BX
```

8 位和 16 位不匹配，非法。

### 第三步：再看目的操作数是不是立即数

比如：

```asm
MOV 1234H, AX
```

这是非法的。

### 第四步：最后检查段寄存器规则

像：

```asm
MOV DS, AX
```

是合法的。

但：

```asm
MOV DS, 1000H
```

通常不合法。

## 最简单例子

合法例子：

```asm
MOV AX, BX
MOV AL, [SI]
MOV [DI], AL
MOV AX, 1234H
```

非法例子：

```asm
MOV [BX], [SI]
MOV AL, BX
MOV 1234H, AX
```

## 易错点

- 看到 `MOV` 就觉得都能搬。
- 忘了“内存到内存”不行。
- 内存位宽不明确时不写 `BYTE PTR/WORD PTR`。

## 注意点

- 小白做题时就按四步排查：内存、位宽、立即数、段寄存器。
- `MOV` 合法性题不是靠感觉，要靠规则逐条筛。
