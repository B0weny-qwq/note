# 怎么写 INT 21H 单字符输入输出题

## 这类题在考什么

这类题通常考：

```text
从键盘读一个字符
再把它显示出来
```

这是 DOS 调用入门题。

## 先记公式

输入单字符常用：

```asm
MOV AH, 01H
INT 21H
```

执行后字符通常在：

```text
AL
```

输出单字符常用：

```asm
MOV AH, 02H
MOV DL, 字符
INT 21H
```

## 做题步骤

### 第一步：输入时先装功能号 `01H`

调用后把读到的字符从 `AL` 取出来。

### 第二步：如果要输出，转到 `DL`

因为 `02H` 输出单字符时常用 `DL`。

### 第三步：装 `AH = 02H`

再执行中断。

### 第四步：最后按程序正常退出

## 最简单例子

读一个字符再原样输出：

```asm
MOV AH, 01H
INT 21H

MOV DL, AL
MOV AH, 02H
INT 21H
```

## 易错点

- 输入完忘了字符在 `AL`。
- 输出时没把字符送到 `DL`。
- 把 `01H` 和 `02H` 功能号记反。

## 注意点

- 小白直接记：
  `01H` 读到 `AL`，`02H` 从 `DL` 打出去。
- 这是后面很多交互程序的小积木。
