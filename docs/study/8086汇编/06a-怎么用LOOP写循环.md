# 怎么用 LOOP 写循环

## 这类题在考什么

这类题通常考：

```text
用 CX 和 LOOP 写重复执行
```

也常考你会不会分析循环次数。

## 先记公式

`LOOP label` 的核心行为是：

```text
CX = CX - 1
如果 CX != 0，就跳到 label
```

## 做题步骤

### 第一步：先把 `CX` 初始化成循环次数

没有这一步，`LOOP` 就没法用。

### 第二步：写循环体

把每轮真正要做的操作放进去。

### 第三步：用 `LOOP` 回跳

它会自动减 `CX`，  
所以通常不需要自己再 `DEC CX`。

### 第四步：空循环或零次循环时可先用 `JCXZ`

这是更稳的写法。

## 最简单例子

如果要执行 5 次某操作：

```asm
MOV CX, 5
L1:
    ; 循环体
    LOOP L1
```

这表示：

```text
循环体共执行 5 次
```

## 易错点

- `LOOP` 已经会减 `CX`，又手动 `DEC CX`。
- 忘了先初始化 `CX`。
- 把 `LOOP` 理解成“先判断后减”。

## 注意点

- 小白做题时就把 `LOOP` 当成“自带 `DEC CX` 的跳转”。
- 看循环次数题时，一定按执行顺序一轮轮想。
