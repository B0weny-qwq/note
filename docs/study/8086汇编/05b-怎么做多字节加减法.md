# 怎么做多字节加减法

## 这类题在考什么

这类题通常考：

```text
两个 32 位数怎么在 8086 上相加或相减
```

因为 8086 一次主要处理 16 位，  
所以多字节运算是基础题。

## 先记公式

多字加法核心：

```asm
ADD 低位, 低位
ADC 高位, 高位
```

多字减法核心：

```asm
SUB 低位, 低位
SBB 高位, 高位
```

## 做题步骤

### 第一步：先把数拆成高位和低位

例如 32 位数可拆成：

- 高 16 位
- 低 16 位

### 第二步：先算低位

低位直接用：

```asm
ADD
```

或

```asm
SUB
```

### 第三步：高位必须带上进位或借位

所以加法用：

```asm
ADC
```

减法用：

```asm
SBB
```

### 第四步：不要把 `INC/DEC` 当成进位链工具

它们不处理 `CF`。

## 最简单例子

如果有：

```text
dst_hi:dst_lo += src_hi:src_lo
```

则应写：

```asm
ADD dst_lo, src_lo
ADC dst_hi, src_hi
```

如果是减法：

```asm
SUB dst_lo, src_lo
SBB dst_hi, src_hi
```

## 易错点

- 高位继续用 `ADD`，把低位进位丢了。
- 高位继续用 `SUB`，把低位借位丢了。
- 误以为 `INC/DEC` 能代替进位链。

## 注意点

- 小白直接记一句：
  `低位普通算，高位带进借位算`。
- 这是考试里很标准的模板题。
