# 磁介质电感与磁能

## 这篇在解决什么问题

磁介质问题和电介质问题很像：材料会被外场极化或磁化，于是需要区分“自由源”和“束缚源”。

电介质里引入：

```math
\vec D=\varepsilon_0\vec E+\vec P
```

磁介质里引入：

```math
\vec H=\frac{\vec B}{\mu_0}-\vec M
```

其中 $\vec M$ 是磁化强度。

## 磁化强度

磁介质可以看成由很多微小磁偶极子组成。

磁化强度定义为单位体积内的磁偶极矩：

```math
\vec M
=
\lim_{\Delta V\to 0}
\frac{\sum \vec m_i}{\Delta V}
```

磁化会产生等效磁化电流：

```math
\vec J_m=\nabla\times\vec M
```

表面磁化电流密度为：

```math
\vec J_{sm}=\vec M\times\vec n
```

## 磁场强度

把磁化电流从总电流中分离后，引入磁场强度：

```math
\vec H=\frac{\vec B}{\mu_0}-\vec M
```

于是安培环路定律可以写成只和自由电流有关：

```math
\nabla\times\vec H=\vec J_f
```

积分形式为：

```math
\oint_L \vec H\cdot d\vec l=I_f
```

线性各向同性磁介质中：

```math
\vec B=\mu\vec H
```

## 磁介质边界条件

磁场始终满足磁通连续：

```math
\vec n\cdot(\vec B_1-\vec B_2)=0
```

也就是：

```math
B_{1n}=B_{2n}
```

切向 $\vec H$ 的跳变由自由面电流决定：

```math
\vec n\times(\vec H_1-\vec H_2)=\vec J_s
```

如果分界面没有自由面电流：

```math
H_{1t}=H_{2t}
```

可以这样记：

```text
磁介质边界：法向看 B，切向看 H。
```

## 磁通和磁通链

磁通定义为：

```math
\Phi=\iint_S \vec B\cdot d\vec S
```

若线圈有 $N$ 匝，每匝磁通相同，则磁通链为：

```math
\Psi=N\Phi
```

磁通链描述线圈整体“链接”了多少磁通。

## 自感

如果线圈自身电流产生的磁通链与电流成正比：

```math
\Psi=LI
```

比例系数 $L$ 称为自感。

自感的物理意义是：电流变化时，线圈自己会通过磁通变化产生感应电动势，阻碍电流变化。

## 互感

两个线圈之间，一个线圈的电流会在另一个线圈中产生磁通链。

可写成：

```math
\Psi_{21}=M_{21}I_1
```

其中 $M_{21}$ 是互感。

在互易条件下：

```math
M_{12}=M_{21}=M
```

## 磁场能量

磁场能量密度为：

```math
w_m=\frac12\vec B\cdot\vec H
```

线性介质中：

```math
w_m=\frac12\mu H^2
```

也可写成：

```math
w_m=\frac{B^2}{2\mu}
```

单个电感储存的能量为：

```math
W_m=\frac12LI^2
```

多个耦合线圈时，能量中会出现互感项。

## 由能量求力

载流系统中的力常可以通过磁场能量求出。

如果电流保持不变，常用形式为：

```math
F=\left.\frac{\partial W_m}{\partial x}\right|_{I=\text{常量}}
```

具体符号要结合选取的位移方向和能量函数判断。

## 易错点

- $\vec M$ 描述磁化强度，$\vec H$ 是把自由电流单独拿出来后得到的辅助场。
- 法向 $B$ 连续不代表法向 $H$ 连续，因为两侧 $\mu$ 可能不同。
- 自感是线圈自身的磁通链与自身电流之比，互感是两个线圈之间的耦合。
- 磁场能量不是“存在电流里”，而是储存在磁场分布中。
