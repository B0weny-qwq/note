# 恒定磁场基础

## 这篇在解决什么问题

恒定磁场研究恒定电流产生的磁场。

它和静电场最不一样的地方是：

```text
静电场有电荷源，恒定磁场没有磁荷源；磁场主要体现为绕电流的环流。
```

## 磁场对运动电荷的力

运动电荷在磁场中受洛伦兹力：

```math
\vec F=q\vec v\times\vec B
```

如果一段导线中有电流 $I$，长度元为 $d\vec l$，受到的磁场力为：

```math
d\vec F=I\,d\vec l\times\vec B
```

方向由右手定则判断。

## 毕奥-萨伐尔定律

电流元产生的磁感应强度为：

```math
d\vec B
=
\frac{\mu_0}{4\pi}
\frac{I\,d\vec l\times\vec e_R}{R^2}
```

对整条电流路径积分：

```math
\vec B
=
\frac{\mu_0}{4\pi}
\int_L
\frac{I\,d\vec l\times\vec e_R}{R^2}
```

它适合直接积分求一些简单电流分布的磁场。

## 磁场无散

磁场满足：

```math
\nabla\cdot\vec B=0
```

积分形式为：

```math
\oiint_S \vec B\cdot d\vec S=0
```

物理意义是：任意闭合面穿出的总磁通为 0。

所以磁感线总是闭合的，不会像静电场线那样从正电荷出发、到负电荷终止。

## 安培环路定律

真空中恒定磁场满足：

```math
\oint_L \vec B\cdot d\vec l=\mu_0 I
```

微分形式为：

```math
\nabla\times\vec B=\mu_0\vec J
```

在介质中更常写成：

```math
\oint_L \vec H\cdot d\vec l=I_f
```

```math
\nabla\times\vec H=\vec J_f
```

其中 $I_f$ 和 $\vec J_f$ 表示自由电流。

## 什么时候用安培环路定律

安培环路定律最适合高对称电流分布，例如：

- 无限长直导线。
- 无限大电流片。
- 长螺线管。
- 环形螺线管。

关键是选出一条安培环路，使磁场在环路上方向和大小容易处理。

## 矢量磁位

因为：

```math
\nabla\cdot\vec B=0
```

所以可以引入矢量磁位 $\vec A$：

```math
\vec B=\nabla\times\vec A
```

这和静电场中的：

```math
\vec E=-\nabla\varphi
```

形成对照。

静电场适合用标量势描述，因为它无旋；磁场适合用矢量势描述，因为它无散。

## 和静电场的对照

| 静电场 | 恒定磁场 |
| --- | --- |
| 源是电荷 $\rho$ | 源是电流 $\vec J$ |
| $\nabla\cdot\vec D=\rho$ | $\nabla\times\vec H=\vec J$ |
| $\nabla\times\vec E=0$ | $\nabla\cdot\vec B=0$ |
| 可用标量势 $\varphi$ | 可用矢量磁位 $\vec A$ |
| 场线可起止于电荷 | 磁感线闭合 |

## 易错点

- 磁场无源不是说磁场没有来源，而是说不存在孤立磁荷，磁场来源体现在旋度方程里。
- 磁场力方向一般既不沿速度方向，也不沿磁场方向，而是由叉乘决定。
- 安培环路定律总成立，但只有对称性足够好时才方便直接求磁场。
- $\vec B$ 和 $\vec H$ 不要混用；介质问题里通常要分清自由电流和磁化电流。
