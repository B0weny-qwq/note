# 均匀平面波在无界媒质中的传播

## 这章考什么

第 5 章把 Maxwell 方程落到一种最常用的波：均匀平面波。

做题时先判断媒质：

```text
理想介质：无耗、无衰减，E 和 H 同相位
导电媒质：有损耗，E 和 H 不同相位
良导体：衰减很快，要用趋肤深度
```

再判断题目问什么：

- 求相速、波长、波阻抗、场强幅值。
- 根据相位项判断传播方向和波矢量。
- 根据 $E_x$、$E_y$ 的振幅和相位差判断极化。
- 在导电媒质中求 $\alpha$、$\beta$、$\eta_c$、$v$、$\lambda$、$\delta$。
- 在色散媒质中求群速。

## 均匀平面波是什么

`波阵面`
: 空间中相位相同的点组成的曲面。

`平面波`
: 波阵面是平面的电磁波。

`均匀平面波`
: 在同一波阵面上，电场和磁场的方向、振幅都不变。

沿 $+z$ 方向传播时，理想介质中的均匀平面波可写成：

```math
\vec E(z)=\vec e_x E_m e^{-jkz}
```

瞬时形式为：

```math
\vec E(z,t)=\vec e_x E_m\cos(\omega t-kz+\phi)
```

相位项中：

- $\omega t-kz$ 表示沿 $+z$ 方向传播。
- $\omega t+kz$ 表示沿 $-z$ 方向传播。

## TEM 横电磁波

无源、线性、各向同性、均匀理想介质中，均匀平面波沿 $z$ 轴传播时：

```math
\frac{\partial}{\partial x}=0,\quad
\frac{\partial}{\partial y}=0
```

由 $\nabla\cdot \vec E=0$ 可得：

```math
E_z=0
```

同理：

```math
H_z=0
```

结论：

```text
均匀平面波的 E、H 都垂直于传播方向，是 TEM 波。
```

## 理想介质中的 E 和 H

沿 $+z$ 方向传播时：

```math
\vec H=\frac{1}{\eta}\vec e_z\times \vec E
```

沿 $-z$ 方向传播时：

```math
\vec H=\frac{1}{\eta}(-\vec e_z)\times \vec E
```

本征阻抗：

```math
\eta=\sqrt{\frac{\mu}{\varepsilon}}
```

真空中：

```math
\eta_0=\sqrt{\frac{\mu_0}{\varepsilon_0}}=120\pi\ \Omega\approx377\ \Omega
```

理想介质中 $\eta$ 为实数，所以 $E$ 和 $H$ 同相位。

## 理想介质传播参数

角频率：

```math
\omega=2\pi f
```

相位常数：

```math
k=\omega\sqrt{\mu\varepsilon}
```

波长：

```math
\lambda=\frac{2\pi}{k}
```

相速：

```math
v=\frac{\omega}{k}=\frac{1}{\sqrt{\mu\varepsilon}}
```

若 $\mu_r=1$，无耗介质中常用：

```math
v=\frac{c}{\sqrt{\varepsilon_r}}
```

```math
\eta=\frac{\eta_0}{\sqrt{\varepsilon_r}}
```

## 能量密度和坡印廷矢量

理想介质中：

```math
w_e=\frac{1}{2}\varepsilon E^2
```

```math
w_m=\frac{1}{2}\mu H^2
```

均匀平面波满足：

```math
w_e=w_m
```

平均坡印廷矢量：

```math
\vec S_{\mathrm{av}}
=
\frac{1}{2}\operatorname{Re}(\vec E\times \vec H^*)
```

若 $E_m$ 是电场幅值，理想介质中：

```math
S_{\mathrm{av}}=\frac{E_m^2}{2\eta}
```

做平均功率题时，再乘面积：

```math
P_{\mathrm{av}}=S_{\mathrm{av}}A
```

圆面积为：

```math
A=\pi R^2
```

## 例题 5.1.1：聚乙烯中求传播参数

题目给：

```text
f = 9.4 GHz
epsilon_r = 2.26
H_m = 7 mA/m
```

无耗介质，且默认 $\mu_r=1$。

做题顺序：

1. 用 $\varepsilon_r$ 求相速。
2. 用 $v/f$ 求波长。
3. 用 $\eta_0/\sqrt{\varepsilon_r}$ 求波阻抗。
4. 用 $E_m=H_m\eta$ 求电场幅值。

计算：

```math
v=\frac{3\times10^8}{\sqrt{2.26}}
=1.996\times10^8\ \mathrm{m/s}
```

```math
\lambda=\frac{v}{f}
=\frac{1.996\times10^8}{9.4\times10^9}
=2.12\times10^{-2}\ \mathrm{m}
```

```math
\eta=\frac{377}{\sqrt{2.26}}=251\ \Omega
```

```math
E_m=H_m\eta=7\times10^{-3}\times251=1.757\ \mathrm{V/m}
```

考点：PPT 里波长写成 `2.12 m` 容易误读；按数量级应为 $2.12\times10^{-2}\ \mathrm{m}$。

## 例题 5.1.2：由 H 写出 E 和频率

题目给：

```text
H_m = 1/(3π) A/m
beta = 30 rad/m
空气中沿 -z 方向传播
t=0,z=0 时 H 指向 -y
```

沿 $-z$ 方向传播，相位写：

```math
\omega t+\beta z
```

磁场：

```math
\vec H(z,t)=-\vec e_y\frac{1}{3\pi}
\cos(\omega t+30z)\ \mathrm{A/m}
```

空气中：

```math
\eta_0=120\pi\ \Omega
```

电场幅值：

```math
E_m=\eta_0 H_m=120\pi\cdot\frac{1}{3\pi}=40\ \mathrm{V/m}
```

方向用右手关系：

```text
E × H = 传播方向
```

传播方向为 $-\vec e_z$，$H$ 为 $-\vec e_y$，所以 $E$ 为 $+\vec e_x$。

```math
\vec E(z,t)=\vec e_x40\cos(\omega t+30z)\ \mathrm{V/m}
```

波长：

```math
\lambda=\frac{2\pi}{\beta}=\frac{\pi}{15}\ \mathrm{m}
```

频率：

```math
f=\frac{c}{\lambda}
=\frac{3\times10^8}{\pi/15}
=1.43\times10^9\ \mathrm{Hz}
```

考点：相位里写 $+\beta z$，方向就是 $-z$。

## 例题 5.1.3：由某点达到幅值求初相

题目给：

```text
f = 100 MHz
epsilon_r = 4
mu_r = 1
沿 +z 传播
t=0,z=1/8 m 时 E 达到幅值 10^-4 V/m
```

设：

```math
\vec E(z,t)=\vec e_x10^{-4}\cos(\omega t-kz+\phi)
```

角频率：

```math
\omega=2\pi\times10^8\ \mathrm{rad/s}
```

相位常数：

```math
k=\frac{\omega}{c}\sqrt{\varepsilon_r\mu_r}
=\frac{2\pi\times10^8}{3\times10^8}\cdot2
=\frac{4\pi}{3}\ \mathrm{rad/m}
```

达到幅值时余弦相角取 0。代入 $t=0,z=1/8$：

```math
-k\frac{1}{8}+\phi=0
```

```math
\phi=\frac{k}{8}=\frac{\pi}{6}
```

所以：

```math
\vec E(z,t)=
\vec e_x10^{-4}\cos\left(2\pi\times10^8t-\frac{4\pi}{3}z+\frac{\pi}{6}\right)
\ \mathrm{V/m}
```

介质阻抗：

```math
\eta=\frac{\eta_0}{\sqrt{\varepsilon_r}}=60\pi\ \Omega
```

磁场：

```math
\vec H(z,t)=
\vec e_y\frac{10^{-4}}{60\pi}
\cos\left(2\pi\times10^8t-\frac{4\pi}{3}z+\frac{\pi}{6}\right)
\ \mathrm{A/m}
```

考点：先用“达到幅值”求初相，再用 $H=E/\eta$。

## 例题 5.1.4：平均功率

题目给自由空间中：

```math
\vec E=\vec e_x50\cos(\omega t-kz)\ \mathrm{V/m}
```

求 $z=z_0$ 处垂直穿过半径 $R=2.5\ \mathrm{m}$ 圆面的平均功率。

自由空间：

```math
\eta_0=120\pi\ \Omega
```

平均功率密度：

```math
S_{\mathrm{av}}=\frac{E_m^2}{2\eta_0}
=\frac{50^2}{2\cdot120\pi}
=\frac{125}{12\pi}\ \mathrm{W/m^2}
```

圆面积：

```math
A=\pi R^2=\pi(2.5)^2
```

平均功率：

```math
P_{\mathrm{av}}
=S_{\mathrm{av}}A
=\frac{125}{12\pi}\pi(2.5)^2
=65.1\ \mathrm{W}
```

考点：幅值形式下平均坡印廷矢量有 $\frac{1}{2}$。

## 任意方向传播的平面波

沿单位方向 $\vec e_n$ 传播：

```math
\vec E(\vec r)=\vec E_m e^{-jk\vec e_n\cdot\vec r}
```

波矢量：

```math
\vec k=k\vec e_n
```

相位写成：

```math
\vec k\cdot\vec r=k_xx+k_yy+k_zz
```

场的横向条件：

```math
\vec e_n\cdot \vec E_m=0
```

相伴磁场：

```math
\vec H(\vec r)=\frac{1}{\eta}\vec e_n\times\vec E(\vec r)
```

若题目先给 $\vec H$，则：

```math
\vec E(\vec r)=\eta \vec H(\vec r)\times \vec e_n
```

## 例题 5.1.5：由相位项求波矢量

题目中的相位项形如：

```math
e^{-j\pi(4x+3z)}
```

直接读：

```math
\vec k=4\pi\vec e_x+3\pi\vec e_z
```

大小：

```math
k=\sqrt{(4\pi)^2+(3\pi)^2}=5\pi
```

传播方向：

```math
\vec e_n=\frac{\vec k}{k}
=\frac{4}{5}\vec e_x+\frac{3}{5}\vec e_z
```

波长：

```math
\lambda=\frac{2\pi}{k}=\frac{2}{5}\ \mathrm{m}
```

空气中频率：

```math
f=\frac{c}{\lambda}=7.5\times10^8\ \mathrm{Hz}
```

若磁场幅值向量含未知数：

```math
\vec H_m=-A\vec e_x+2\vec e_y+4\vec e_z
```

用横向条件：

```math
\vec k\cdot\vec H_m=0
```

```math
4\pi(-A)+3\pi(4)=0
```

```math
A=3
```

考点：看到 $e^{-j(k_xx+k_yy+k_zz)}$，先读 $\vec k$；再用 $\vec k\cdot\vec H_m=0$ 求未知量。

## 极化怎么判断

沿 $+z$ 方向传播时：

```math
\vec E=\vec e_xE_x+\vec e_yE_y
```

```math
E_x=E_{xm}\cos(\omega t-kz+\phi_x)
```

```math
E_y=E_{ym}\cos(\omega t-kz+\phi_y)
```

令：

```math
\Delta\phi=\phi_y-\phi_x
```

判断表：

| 条件 | 极化 |
| --- | --- |
| $\Delta\phi=0,\pm\pi$ | 线极化 |
| $E_{xm}=E_{ym}$ 且 $\Delta\phi=\frac{\pi}{2}$ | 左旋圆极化 |
| $E_{xm}=E_{ym}$ 且 $\Delta\phi=-\frac{\pi}{2}$ | 右旋圆极化 |
| 其它情况 | 椭圆极化 |

若 $0<\Delta\phi<\pi$，通常判左旋椭圆极化。

若 $-\pi<\Delta\phi<0$，通常判右旋椭圆极化。

## 例题 5.2.1：判断极化方式

做极化题按三步：

1. 把 $E_x$、$E_y$ 都化成同一种三角函数，最好用 $\cos(\omega t-kz+\phi)$。
2. 读出 $E_{xm}$、$E_{ym}$、$\phi_x$、$\phi_y$。
3. 算 $\Delta\phi=\phi_y-\phi_x$。

PPT 四个小题对应：

| 小题 | 判断 |
| --- | --- |
| (1) $E_x=E_m\sin(\omega t-kz)$，$E_y=E_m\cos(\omega t-kz)$ | 左旋圆极化 |
| (2) 复数式中 $E_y=-jE_m e^{-jkz}$ | 右旋圆极化 |
| (3) 两个分量相位相同 | 线极化 |
| (4) 振幅不等，相位差为 $\frac{\pi}{2}$ | 左旋椭圆极化 |

考点：$\sin\theta=\cos(\theta-\frac{\pi}{2})$，别直接拿 $\sin$ 和 $\cos$ 比相位。

## 导电媒质中的均匀平面波

导电媒质有：

```math
\sigma\ne0
```

存在传导电流：

```math
\vec J=\sigma\vec E
```

沿 $+z$ 方向传播时：

```math
\vec E(z)=\vec e_xE_{xm}e^{-\gamma z}
```

传播常数：

```math
\gamma=\alpha+j\beta
```

所以：

```math
\vec E(z)=\vec e_xE_{xm}e^{-\alpha z}e^{-j\beta z}
```

瞬时形式：

```math
\vec E(z,t)=\vec e_xE_{xm}e^{-\alpha z}
\cos(\omega t-\beta z)
```

其中：

- $\alpha$ 是衰减常数，单位 $\mathrm{Np/m}$。
- $\beta$ 是相位常数，单位 $\mathrm{rad/m}$。

导电媒质本征阻抗为复数：

```math
\eta_c=|\eta_c|e^{j\phi}
```

磁场比电场滞后 $\phi$：

```math
\vec H(z,t)=
\vec e_y\frac{E_{xm}}{|\eta_c|}e^{-\alpha z}
\cos(\omega t-\beta z-\phi)
```

## 一般导电媒质公式

常用参数：

```math
p=\frac{\sigma}{\omega\varepsilon}
```

衰减常数：

```math
\alpha
=\omega\sqrt{\frac{\mu\varepsilon}{2}}
\left[
\sqrt{1+p^2}-1
\right]^{1/2}
```

相位常数：

```math
\beta
=\omega\sqrt{\frac{\mu\varepsilon}{2}}
\left[
\sqrt{1+p^2}+1
\right]^{1/2}
```

平均坡印廷矢量：

```math
\vec S_{\mathrm{av}}
=
\vec e_z
\frac{E_{xm}^2}{2|\eta_c|}
e^{-2\alpha z}\cos\phi
```

做题时先算 $p$：

- $p\ll1$：弱导电媒质。
- $p\gg1$：良导体。
- 不能近似时，用一般公式。

## 弱导电媒质近似

条件：

```math
\frac{\sigma}{\omega\varepsilon}\ll1
```

近似：

```math
\alpha\approx\frac{\sigma}{2}\sqrt{\frac{\mu}{\varepsilon}}
```

```math
\beta\approx\omega\sqrt{\mu\varepsilon}
```

```math
\eta_c\approx
\sqrt{\frac{\mu}{\varepsilon}}
\left(1+j\frac{\sigma}{2\omega\varepsilon}\right)
```

特点：衰减小，$\beta$ 接近理想介质，$E$ 和 $H$ 之间有小相位差。

## 良导体近似

条件：

```math
\frac{\sigma}{\omega\varepsilon}\gg1
```

近似：

```math
\alpha\approx\beta\approx\sqrt{\pi f\mu\sigma}
```

相速：

```math
v=\frac{\omega}{\beta}
\approx
\sqrt{\frac{2\omega}{\mu\sigma}}
```

波长：

```math
\lambda=\frac{2\pi}{\beta}
\approx
2\sqrt{\frac{\pi}{f\mu\sigma}}
```

本征阻抗：

```math
\eta_c\approx(1+j)\sqrt{\frac{\pi f\mu}{\sigma}}
```

相位角：

```math
\phi=45^\circ
```

也就是磁场滞后电场 $45^\circ$。

## 趋肤效应和趋肤深度

良导体中，电磁波进入材料后振幅按指数衰减：

```math
E(z)=E(0)e^{-\alpha z}
```

趋肤深度定义为振幅降到表面处 $1/e$ 的深度：

```math
\delta=\frac{1}{\alpha}
```

良导体中：

```math
\delta=\frac{1}{\sqrt{\pi f\mu\sigma}}
```

频率越高，$\delta$ 越小。高频电磁波只集中在导体表层。

## 例题 5.3.1：海水中传播

题目给：

```text
epsilon_r = 81
mu_r = 1
sigma = 4 S/m
E_x(z=0,t)=100cos(10^7πt) V/m
```

先算：

```math
\omega=10^7\pi\ \mathrm{rad/s}
```

```math
f=\frac{\omega}{2\pi}=5\times10^6\ \mathrm{Hz}
```

判断媒质：

```math
\frac{\sigma}{\omega\varepsilon}\approx180\gg1
```

海水按良导体处理。

衰减常数和相位常数：

```math
\alpha=\beta=\sqrt{\pi f\mu\sigma}=8.89\ \mathrm{Np/m}
```

本征阻抗：

```math
\eta_c=\pi e^{j\pi/4}\ \Omega
```

相速：

```math
v=\frac{\omega}{\beta}=3.53\times10^6\ \mathrm{m/s}
```

波长：

```math
\lambda=\frac{2\pi}{\beta}=0.707\ \mathrm{m}
```

趋肤深度：

```math
\delta=\frac{1}{\alpha}=0.112\ \mathrm{m}
```

电场衰减到 $1/1000$ 的距离：

```math
e^{-\alpha z}=\frac{1}{1000}
```

```math
z=\frac{\ln1000}{\alpha}
=\frac{6.908}{8.89}
=0.777\ \mathrm{m}
```

在 $z=0.8\ \mathrm{m}$：

```math
\vec E(0.8,t)=
\vec e_x0.082\cos(10^7\pi t-7.11)\ \mathrm{V/m}
```

磁场幅值：

```math
H_m=\frac{0.082}{|\eta_c|}
=\frac{0.082}{\pi}
=0.026\ \mathrm{A/m}
```

磁场相位滞后 $\pi/4$：

```math
\vec H(0.8,t)=
\vec e_y0.026\cos(10^7\pi t-7.11-\frac{\pi}{4})\ \mathrm{A/m}
```

平均功率密度：

```math
S_{\mathrm{av}}
=
\frac{100^2}{2\pi}e^{-2\cdot8.89\cdot0.8}\cos\frac{\pi}{4}
=0.75\ \mathrm{mW/m^2}
```

穿过 $1\ \mathrm{m^2}$：

```math
P_{\mathrm{av}}=0.75\ \mathrm{mW}
```

考点：先用 $\sigma/(\omega\varepsilon)$ 判断良导体，再套良导体近似。

## 例题 5.3.2：铜板屏蔽厚度

题目给频率范围：

```text
10 kHz 到 100 MHz
```

铜：

```math
\sigma=5.8\times10^7\ \mathrm{S/m}
```

屏蔽厚度要求：

```math
d>5\delta
```

趋肤深度：

```math
\delta=\frac{1}{\sqrt{\pi f\mu\sigma}}
```

$\delta$ 随频率增大而减小，所以要用最低频率 $10\ \mathrm{kHz}$ 算最厚要求。

```math
\delta_L=0.66\ \mathrm{mm}
```

```math
d_{\min}=5\delta_L=3.3\ \mathrm{mm}
```

考点：屏蔽频段题取最低频率，因为最低频率趋肤深度最大。

## 色散和群速

相速：

```math
v_p=\frac{\omega}{\beta}
```

若相速随频率变化，就有色散。

群速：

```math
v_g=\frac{d\omega}{d\beta}
```

也可写成：

```math
v_g=\frac{v_p}{1-\frac{\omega}{v_p}\frac{dv_p}{d\omega}}
```

判断：

| 条件 | 结论 |
| --- | --- |
| $\frac{dv_p}{d\omega}=0$ | 无色散，$v_g=v_p$ |
| $\frac{dv_p}{d\omega}<0$ | 正常色散，$v_g<v_p$ |
| $\frac{dv_p}{d\omega}>0$ | 反常色散，$v_g>v_p$ |

## 群速例题：海水中 100 kHz 信号

海水在 $100\ \mathrm{kHz}$ 下可按良导体处理。

良导体相速：

```math
v_p(\omega)\approx\sqrt{\frac{2\omega}{\mu\sigma}}
```

PPT 写成：

```math
v_p(\omega)=631\sqrt{\omega}
```

求导：

```math
\frac{dv_p}{d\omega}
=
\frac{631}{2\sqrt{\omega}}
```

代入群速公式，得到：

```math
v_g=10^6\ \mathrm{m/s}
```

而：

```math
v_p=5\times10^5\ \mathrm{m/s}
```

所以：

```text
v_g>v_p，属于反常色散。
```

考点：良导体中 $v_p\propto\sqrt{\omega}$，因此 $dv_p/d\omega>0$。

## 各向异性媒质：记结论

各向同性媒质中，$\varepsilon$ 或 $\mu$ 可当标量。

各向异性媒质中，$\varepsilon$ 或 $\mu$ 要写成张量。

磁化等离子体：外加恒定磁场让等离子体呈电各向异性，等效介电常数为张量。

沿外加磁场方向传播时，一个线极化波可分解为两个圆极化波：

```text
左旋圆极化波
右旋圆极化波
```

两个圆极化波相速不同。传播一段距离后重新合成，线极化面的方向会旋转。

这种现象叫：

```text
法拉第旋转效应
```

磁化铁氧体类似，只是主要表现为磁导率张量。它也会产生法拉第旋转，常用于微波器件。

## 做题总流程

### 理想介质题

1. 判断传播方向，看相位是 $\omega t-kz$ 还是 $\omega t+kz$。
2. 用 $v=1/\sqrt{\mu\varepsilon}$、$\lambda=v/f$、$\eta=\sqrt{\mu/\varepsilon}$。
3. 用右手关系确定 $E$、$H$、传播方向。
4. 功率题用 $S_{\mathrm{av}}=E_m^2/(2\eta)$。

### 极化题

1. 把 $E_x$、$E_y$ 都写成余弦形式。
2. 读振幅和初相。
3. 算 $\Delta\phi=\phi_y-\phi_x$。
4. 用表格判断线极化、圆极化、椭圆极化和旋向。

### 导电媒质题

1. 先算 $p=\sigma/(\omega\varepsilon)$。
2. $p\ll1$ 用弱导电近似，$p\gg1$ 用良导体近似。
3. 求 $\alpha$、$\beta$、$\eta_c$、$v$、$\lambda$、$\delta$。
4. 写瞬时场时记住 $e^{-\alpha z}$ 和磁场相位滞后。

## 易错点

- 把 $e^{-jkz}$ 和 $e^{jkz}$ 的传播方向看反。
- 用幅值算平均功率时漏掉 $\frac{1}{2}$。
- 极化题没有把 $\sin$ 化成 $\cos$ 就比较相位。
- 判断圆极化时只看相位差，忘记要求 $E_{xm}=E_{ym}$。
- 导电媒质题不先判断 $\sigma/(\omega\varepsilon)$，直接套错近似。
- 良导体中磁场滞后电场 $45^\circ$；理想介质中 $E$ 和 $H$ 同相位。
- 屏蔽厚度题要取最低频率对应的趋肤深度。
