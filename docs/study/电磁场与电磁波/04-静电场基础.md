# 静电场基础

## 这篇在解决什么问题

静电场研究静止电荷产生的电场。

它的核心线索是：

```text
电荷 -> 电场 -> 电势 -> 边界条件和能量
```

静电场最重要的两个性质是：

```math
\nabla\cdot\vec E=\frac{\rho}{\varepsilon_0}
```

```math
\nabla\times\vec E=0
```

前者说明电荷是电场的源，后者说明静电场是保守场。

## 库仑定律

两个静止点电荷之间的作用力为：

```math
\vec F
=
\frac{1}{4\pi\varepsilon_0}
\frac{q_1q_2}{R^2}\vec e_R
```

其中 $\vec e_R$ 是从源电荷指向受力电荷的单位矢量。

真空介电常数为：

```math
\varepsilon_0\approx 8.854\times 10^{-12}\ \text{F/m}
```

## 电场强度

电场强度定义为单位正试验电荷受到的力：

```math
\vec E=\lim_{q_0\to 0}\frac{\vec F}{q_0}
```

点电荷产生的电场为：

```math
\vec E
=
\frac{1}{4\pi\varepsilon_0}
\frac{q}{R^2}\vec e_R
```

多个电荷产生的电场满足叠加原理：

```math
\vec E=\sum_i \vec E_i
```

连续电荷分布则把求和换成积分。

## 高斯定理

真空静电场的高斯定理为：

```math
\oiint_S \vec E\cdot d\vec S
=
\frac{q}{\varepsilon_0}
```

微分形式为：

```math
\nabla\cdot\vec E=\frac{\rho}{\varepsilon_0}
```

它的物理意义是：闭合面内有多少净电荷，就决定有多少电场通量穿出闭合面。

## 什么时候用高斯定理算电场

高斯定理最适合对称性强的问题，例如：

- 球对称：点电荷、均匀带电球面、均匀带电球体。
- 柱对称：无限长带电直线、无限长带电圆柱。
- 平面对称：无限大均匀带电平面。

关键不是“看到闭合面就用”，而是要能选出一个高斯面，让 $\vec E$ 在面上容易积分。

## 静电场无旋

静电场的环路积分为：

```math
\oint_L \vec E\cdot d\vec l=0
```

对应微分形式：

```math
\nabla\times\vec E=0
```

这说明静电力做功与路径无关，只和起点、终点有关。

## 电势

因为静电场无旋，所以可以引入电势 $\varphi$：

```math
\vec E=-\nabla\varphi
```

电势差定义为：

```math
\varphi_A-\varphi_B
=
\int_A^B \vec E\cdot d\vec l
```

电场方向指向电势降低最快的方向。

## 泊松方程和拉普拉斯方程

把 $\vec E=-\nabla\varphi$ 代入高斯定律：

```math
\nabla\cdot\vec E=\frac{\rho}{\varepsilon}
```

得到泊松方程：

```math
\nabla^2\varphi=-\frac{\rho}{\varepsilon}
```

如果无自由电荷，即 $\rho=0$，得到拉普拉斯方程：

```math
\nabla^2\varphi=0
```

所以：

- 有电荷区域用泊松方程。
- 无电荷区域用拉普拉斯方程。

## 静电场的解题顺序

一般可以按这个顺序想：

1. 先判断电荷分布是否有对称性。
2. 有强对称性时，优先用高斯定理求 $\vec E$。
3. 如果要求电势，用 $\vec E=-\nabla\varphi$ 或积分求电势差。
4. 如果有边界和介质，转向边界条件和泊松/拉普拉斯方程。

## 易错点

- 电势是标量，电场是矢量，不要把二者混成一个量。
- 电场线从正电荷出发到负电荷终止，但电势不需要方向箭头。
- $\nabla\times\vec E=0$ 只对静电场成立；时变磁场存在时一般不成立。
- 高斯定理永远成立，但只有对称性足够好时才方便直接求电场。
