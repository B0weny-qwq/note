# 电磁场与电磁波分别是什么

## 这篇在解决什么问题

电磁场和电磁波不是两个完全独立的东西。

简单说：

- 电磁场是更大的概念，描述空间中电场和磁场的分布。
- 电磁波是电磁场的一种运动形式，是变化的电场和变化的磁场互相激发并向外传播。

如果只背公式，很容易把静电场、恒定磁场、电磁波混在一起。正确的顺序是：

```text
静止电荷 -> 静电场
恒定电流 -> 恒定磁场
变化电场和变化磁场 -> 电磁波
```

## 电磁场是什么

电磁场是电荷、电流以及它们的运动状态在空间中产生的场。

在课程里，电磁场常用四个矢量描述：

```math
\vec E,\quad \vec D,\quad \vec B,\quad \vec H
```

其中：

`电场强度 \vec E`
: 描述电荷在空间某点受到电场力的能力。

`电位移矢量 \vec D`
: 在介质中更方便地描述自由电荷产生的电场。

`磁感应强度 \vec B`
: 描述运动电荷或电流受到磁场力的能力。

`磁场强度 \vec H`
: 在介质中更方便地描述自由电流产生的磁场。

## 电磁场的核心方程

电磁场最核心的关系由 Maxwell 方程组概括：

```math
\nabla\cdot \vec D=\rho
```

```math
\nabla\times \vec E=-\frac{\partial \vec B}{\partial t}
```

```math
\nabla\cdot \vec B=0
```

```math
\nabla\times \vec H=\vec J+\frac{\partial \vec D}{\partial t}
```

四个式子可以这样读：

- 电荷是电场通量的源。
- 变化的磁场会激发旋涡状电场。
- 磁场没有单独的磁荷源，磁感线总是闭合的。
- 电流和变化的电场都会激发旋涡状磁场。

## 静态场和时变场

如果场量不随时间变化，就叫静态场或恒定场。

静电场满足：

```math
\nabla\times \vec E=0
```

所以静电场可以由电势表示：

```math
\vec E=-\nabla\varphi
```

恒定磁场满足：

```math
\nabla\cdot \vec B=0
```

说明磁场没有起点和终点，磁感线闭合。它的旋度与电流有关：

```math
\nabla\times \vec H=\vec J
```

## 电磁波是什么

电磁波是时变电磁场在空间中的传播。

关键不是“有电场”和“有磁场”，而是：

```text
变化的电场产生磁场，变化的磁场产生电场
```

这样电场和磁场就能互相维持，脱离局部电荷或电流源向远处传播。

在真空中，电磁波的传播速度为：

```math
c=\frac{1}{\sqrt{\mu_0\varepsilon_0}}
```

在均匀介质中，传播速度通常写成：

```math
v=\frac{1}{\sqrt{\mu\varepsilon}}
```

## 平面电磁波的基本特征

对沿某一方向传播的平面电磁波，电场、磁场、传播方向三者两两垂直：

```text
\vec E 方向、\vec H 方向、传播方向构成右手关系
```

常见关系为：

```math
\vec E \perp \vec H,\quad \vec E\perp \vec k,\quad \vec H\perp \vec k
```

其中 $\vec k$ 表示传播方向。

真空中电场和磁场大小满足：

```math
\frac{E}{H}=\eta_0
```

这里 $\eta_0$ 是真空波阻抗，约为：

```math
\eta_0\approx 377\ \Omega
```

## 怎么从电磁场过渡到电磁波

电磁波可以看成 Maxwell 方程组在无源区域中的自然结果。

无源区域中：

```math
\rho=0,\quad \vec J=0
```

此时仍然可能有场，因为时变项还在：

```math
\nabla\times \vec E=-\frac{\partial \vec B}{\partial t}
```

```math
\nabla\times \vec H=\frac{\partial \vec D}{\partial t}
```

这说明电磁波不需要一路都有电荷和电流“推着走”，它靠电场和磁场的时变耦合传播。

## 生活类比

电磁场像“空间里已经铺好的影响状态”。

电磁波像“这个影响状态发生扰动后向外传开的波动”。

比如池塘水面：

- 水面高度分布像一个场。
- 石子落下后向外扩散的水波像波。

对应到电磁学：

- 电荷和电流改变周围空间的电磁场。
- 电磁场随时间变化并向外传播，就是电磁波。

## 易错点

- 不要把电磁场只理解成电场加磁场的简单拼盘，时变情况下二者会互相耦合。
- 不要认为电磁波必须依附在导线里传播，无源空间中也可以传播。
- 静电场的电场线可以从正电荷出发到负电荷终止，但磁感线没有孤立起点和终点。
- 电磁波里的电场和磁场通常不是沿传播方向振动，而是横向振动。
