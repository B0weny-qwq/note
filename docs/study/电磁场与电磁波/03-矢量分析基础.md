# 矢量分析基础

## 这篇在解决什么问题

电磁场不是只在一个点上看数值，而是要看整个空间里的分布。

所以这门课一开始会反复用到三个运算：

```text
梯度：看标量场哪里变化最快
散度：看矢量场有没有源
旋度：看矢量场有没有绕着转
```

## 标量场和矢量场

`标量场`
: 空间每一点对应一个数，例如电势 $\varphi(x,y,z)$、温度 $T(x,y,z)$。

`矢量场`
: 空间每一点对应一个矢量，例如电场 $\vec E(x,y,z)$、磁场 $\vec B(x,y,z)$。

电磁场中最常见的矢量场是：

```math
\vec E,\quad \vec D,\quad \vec B,\quad \vec H,\quad \vec J
```

## Nabla 算符

直角坐标系中：

```math
\nabla
=
\vec e_x\frac{\partial}{\partial x}
+
\vec e_y\frac{\partial}{\partial y}
+
\vec e_z\frac{\partial}{\partial z}
```

它本身不是普通矢量，而是一个微分算符。它和标量或矢量组合后，分别形成梯度、散度、旋度。

## 梯度

标量场 $u$ 的梯度为：

```math
\nabla u
=
\vec e_x\frac{\partial u}{\partial x}
+
\vec e_y\frac{\partial u}{\partial y}
+
\vec e_z\frac{\partial u}{\partial z}
```

梯度的方向是标量场增长最快的方向，大小表示增长最快的变化率。

在静电场中：

```math
\vec E=-\nabla\varphi
```

意思是电场方向指向电势下降最快的方向。

## 通量和散度

矢量场 $\vec A$ 穿过曲面 $S$ 的通量为：

```math
\Phi=\iint_S \vec A\cdot d\vec S
```

如果闭合曲面的通量大于 0，可以理解为里面有源；小于 0，可以理解为里面有汇。

散度描述单位体积内的净流出强度：

```math
\nabla\cdot\vec A
=
\lim_{\Delta V\to 0}
\frac{\oiint_S \vec A\cdot d\vec S}{\Delta V}
```

电磁学里最典型的例子是：

```math
\nabla\cdot\vec D=\rho
```

这说明电荷密度是电位移矢量的源。

## 高斯定理

高斯定理把体积分和闭合曲面积分联系起来：

```math
\iiint_V \nabla\cdot\vec A\,dV
=
\oiint_S \vec A\cdot d\vec S
```

做题时可以这样判断：

- 如果题目有球面、柱面、平面对称，优先想到高斯定理。
- 如果要求某个闭合面内的总电荷或总通量，优先想到高斯定理。
- 如果场在高斯面上大小相等、方向一致，积分会大幅简化。

## 环量和旋度

矢量场 $\vec A$ 沿闭合曲线 $L$ 的环量为：

```math
\oint_L \vec A\cdot d\vec l
```

旋度描述单位面积上的环量强度：

```math
\nabla\times\vec A
```

直角坐标系中可用行列式记忆：

```math
\nabla\times\vec A
=
\begin{vmatrix}
\vec e_x & \vec e_y & \vec e_z\\
\frac{\partial}{\partial x} & \frac{\partial}{\partial y} & \frac{\partial}{\partial z}\\
A_x & A_y & A_z
\end{vmatrix}
```

电磁学里最典型的例子是：

```math
\nabla\times\vec H=\vec J
```

这说明电流密度会产生磁场的旋涡性。

## 斯托克斯定理

斯托克斯定理把曲面积分和边界曲线积分联系起来：

```math
\oint_L \vec A\cdot d\vec l
=
\iint_S (\nabla\times\vec A)\cdot d\vec S
```

做题时可以这样判断：

- 如果题目围绕一条闭合曲线积分，优先想到旋度。
- 如果题目出现“环路”“绕一圈”“安培环路”，优先想到斯托克斯定理或安培环路定律。
- 曲线方向和曲面法向要满足右手关系。

## 两个重要判断

如果：

```math
\nabla\cdot\vec A=0
```

说明这个场没有净源，场线不会在空间中凭空开始或结束。

如果：

```math
\nabla\times\vec A=0
```

说明这个场无旋，闭合环路积分通常为 0，可以尝试引入标量势。

## 易错点

- 散度看的是“从一个小体积净流出多少”，不是看矢量本身大不大。
- 旋度看的是“局部有没有绕圈趋势”，不是看场线是否弯曲。
- 高斯定理适合对称性强的问题，不是所有电场题都能直接用它秒算。
- 斯托克斯定理中的曲面可以换，但边界曲线必须相同。
