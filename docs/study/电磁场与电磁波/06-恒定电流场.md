# 恒定电流场

## 这篇在解决什么问题

恒定电流场研究电荷持续定向运动时形成的电流分布。

它处在静电场和磁场之间：

```text
电场推动电荷运动 -> 形成电流 -> 电流产生磁场
```

## 电流强度

电流强度定义为单位时间通过截面的电荷量：

```math
I=\frac{dq}{dt}
```

单位是安培：

```math
1\ \text{A}=1\ \text{C/s}
```

## 电流密度

电流密度 $\vec J$ 描述单位面积上通过的电流强弱和方向。

通过曲面 $S$ 的电流为：

```math
I=\iint_S \vec J\cdot d\vec S
```

如果载流粒子体电荷密度为 $\rho$，平均定向速度为 $\vec v$，则：

```math
\vec J=\rho\vec v
```

## 电流连续性方程

电荷守恒要求：

```math
\nabla\cdot\vec J+\frac{\partial\rho}{\partial t}=0
```

含义是：

- 如果某处电流净流出更多，那里电荷密度就会减少。
- 如果某处电荷密度不变，流入和流出必须平衡。

恒定电流中：

```math
\frac{\partial\rho}{\partial t}=0
```

所以：

```math
\nabla\cdot\vec J=0
```

这说明恒定电流场没有电流的净源或净汇。

## 导电介质中的欧姆定律

导电介质中，局部欧姆定律为：

```math
\vec J=\sigma\vec E
```

其中 $\sigma$ 是电导率。

也可写成：

```math
\vec E=\frac{1}{\sigma}\vec J
```

若用电阻率 $\rho_c$ 表示：

```math
\rho_c=\frac{1}{\sigma}
```

则：

```math
\vec E=\rho_c\vec J
```

## 恒定电流场的方程

恒定电流满足：

```math
\nabla\cdot\vec J=0
```

静态情况下电场仍无旋：

```math
\nabla\times\vec E=0
```

因此仍可引入电势：

```math
\vec E=-\nabla\varphi
```

代入 $\vec J=\sigma\vec E$：

```math
\vec J=-\sigma\nabla\varphi
```

若 $\sigma$ 为常数，则：

```math
\nabla^2\varphi=0
```

这和无源区域静电场的拉普拉斯方程形式相同。

## 导电介质边界条件

恒定电流在分界面处满足法向电流连续：

```math
J_{1n}=J_{2n}
```

静态电场仍满足切向电场连续：

```math
E_{1t}=E_{2t}
```

结合 $\vec J=\sigma\vec E$，可以得到：

```math
\frac{J_{1t}}{\sigma_1}
=
\frac{J_{2t}}{\sigma_2}
```

这说明不同导电率介质交界处，电流线会发生偏折。

## 功率损耗

电场推动电荷运动时会产生功率。

体功率密度为：

```math
p=\vec J\cdot\vec E
```

线性导电介质中：

```math
p=\sigma E^2
```

也可以写成：

```math
p=\frac{J^2}{\sigma}
```

总功率为：

```math
P=\int_V \vec J\cdot\vec E\,dV
```

这对应电路中的焦耳热。

## 和静电场的类比

恒定电流场和静电场很像：

| 静电场 | 恒定电流场 |
| --- | --- |
| $\vec D=\varepsilon\vec E$ | $\vec J=\sigma\vec E$ |
| $\nabla\cdot\vec D=\rho_f$ | $\nabla\cdot\vec J=0$ |
| 无源区 $\nabla^2\varphi=0$ | 均匀导体中 $\nabla^2\varphi=0$ |
| 电容 $C=q/U$ | 电导 $G=I/U$ |

## 易错点

- 恒定电流不是没有电荷运动，而是各处电荷密度不随时间积累。
- $\nabla\cdot\vec J=0$ 表示电流连续，不表示电流为 0。
- 导电介质中 $\vec J$ 和 $\vec E$ 同向只适用于各向同性线性介质。
- 恒定电流场可以用电势，但它和静电平衡不是一回事；导体内部可以存在电场来维持电流。
