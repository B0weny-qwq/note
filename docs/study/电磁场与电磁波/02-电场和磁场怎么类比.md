# 电场和磁场怎么类比

## 这篇在解决什么问题

电场和磁场都叫“场”，但它们的源、场线形状和数学描述不一样。

最重要的一句话是：

```text
电场更容易从“源和电势”理解，磁场更容易从“环流和闭合场线”理解。
```

## 一张对比表

| 角度 | 电场 | 磁场 |
| --- | --- | --- |
| 基本源 | 电荷 | 电流、运动电荷、变化电场 |
| 典型力 | $\vec F=q\vec E$ | $\vec F=q\vec v\times\vec B$ |
| 场线 | 可从正电荷出发，到负电荷终止 | 总是闭合曲线 |
| 通量特点 | 有电荷时通量不为零 | 任意闭合面总磁通为零 |
| 散度方程 | $\nabla\cdot\vec D=\rho$ | $\nabla\cdot\vec B=0$ |
| 旋度方程 | 静电场中 $\nabla\times\vec E=0$ | 恒定磁场中 $\nabla\times\vec H=\vec J$ |
| 势函数 | 静电场可用标量电势 $\varphi$ | 常用矢量磁位 $\vec A$ |
| 能量密度 | $w_e=\frac12\vec D\cdot\vec E$ | $w_m=\frac12\vec B\cdot\vec H$ |

## 从“源”来类比

电场的直接源是电荷。

点电荷产生的电场大小随距离平方衰减：

```math
\vec E
=
\frac{1}{4\pi\varepsilon_0}
\frac{q}{R^2}\vec e_R
```

磁场的直接源不是静止电荷，而是电流或运动电荷。

电流元产生磁场可用 Biot-Savart 定律描述：

```math
d\vec B
=
\frac{\mu_0}{4\pi}
\frac{I\,d\vec l\times\vec e_R}{R^2}
```

所以可以这样记：

```text
电荷静止时主要看电场，电荷定向运动形成电流后会产生磁场。
```

## 从“通量和散度”来类比

电场有源：

```math
\oint_S \vec D\cdot d\vec S=q
```

对应微分形式：

```math
\nabla\cdot\vec D=\rho
```

这说明闭合面包住净电荷时，电场通量不为零。

磁场无源：

```math
\oint_S \vec B\cdot d\vec S=0
```

对应微分形式：

```math
\nabla\cdot\vec B=0
```

这说明磁感线不会在空间中突然开始或结束。

## 从“环量和旋度”来类比

静电场无旋：

```math
\oint_L \vec E\cdot d\vec l=0
```

对应：

```math
\nabla\times\vec E=0
```

所以静电场是保守场，可以引入电势：

```math
\vec E=-\nabla\varphi
```

恒定磁场有旋，环量与穿过回路的电流有关：

```math
\oint_L \vec H\cdot d\vec l=I
```

对应：

```math
\nabla\times\vec H=\vec J
```

所以磁场更像绕着电流“打圈”的场。

## 从“势”来类比

静电场常用标量电势：

```math
\vec E=-\nabla\varphi
```

电势高低描述单位正电荷移动时电势能的变化。

磁场因为满足：

```math
\nabla\cdot\vec B=0
```

常引入矢量磁位：

```math
\vec B=\nabla\times\vec A
```

这个关系的意思是：磁场天然适合用“旋出来”的方式表示。

## 从“介质”来类比

电介质中，常用：

```math
\vec D=\varepsilon\vec E
```

其中 $\vec D$ 更关注自由电荷，$\vec E$ 更直接对应受力。

磁介质中，常用：

```math
\vec B=\mu\vec H
```

其中 $\vec H$ 更关注自由电流，$\vec B$ 更直接对应磁感应和受力。

类比记忆：

```text
电场：自由电荷 -> D，实际电场作用 -> E
磁场：自由电流 -> H，实际磁感应作用 -> B
```

## 从“边界条件”来类比

电场在介质分界面处：

```math
\vec n\cdot(\vec D_1-\vec D_2)=\rho_s
```

```math
\vec E_{1t}=\vec E_{2t}
```

意思是：

- 法向 $D$ 的跳变由面电荷决定。
- 切向 $E$ 连续。

磁场在介质分界面处：

```math
\vec n\cdot(\vec B_1-\vec B_2)=0
```

```math
\vec n\times(\vec H_1-\vec H_2)=\vec J_s
```

意思是：

- 法向 $B$ 连续。
- 切向 $H$ 的跳变由面电流决定。

## 从“能量”来类比

电场能量密度：

```math
w_e=\frac12\vec D\cdot\vec E
```

线性介质中：

```math
w_e=\frac12\varepsilon E^2
```

磁场能量密度：

```math
w_m=\frac12\vec B\cdot\vec H
```

线性介质中：

```math
w_m=\frac12\mu H^2
```

这说明场不只是计算力的工具，它本身也可以储存能量。

## 生活类比

电场像“坡度”。

电势 $\varphi$ 像海拔高度，电场 $\vec E=-\nabla\varphi$ 像坡最陡下降的方向。电荷放进去，会顺着电势变化受到力。

磁场像“绕流”。

电流像一根正在搅动周围空间的轴，磁感线围绕电流形成闭合环。判断方向时常用右手定则。

## 易错点

- 静电场可以写成 $\vec E=-\nabla\varphi$，但时变电磁场中一般不能只靠一个电势解决。
- 磁场无源指 $\nabla\cdot\vec B=0$，不是说磁场没有来源；它的来源主要体现在旋度方程里。
- $\vec D$ 和 $\vec E$ 不是同一个量，$\vec B$ 和 $\vec H$ 也不是同一个量。
- 电场线和磁感线都只是辅助想象，不是空间里真实存在的细线。
