# 电介质边界条件与电容

## 这篇在解决什么问题

真空里的静电场只需要看电荷和 $\vec E$。

进入电介质后，还要考虑介质被电场极化，于是引入：

```math
\vec D=\varepsilon_0\vec E+\vec P
```

线性各向同性介质中常写成：

```math
\vec D=\varepsilon\vec E
```

其中 $\vec D$ 主要负责和自由电荷联系，$\vec P$ 描述介质极化。

## 电极化

电介质放入电场后，介质内部的正负电荷会发生微小相对位移，形成大量电偶极子。

极化强度定义为单位体积内的电偶极矩：

```math
\vec P
=
\lim_{\Delta V\to 0}
\frac{\sum \vec p_i}{\Delta V}
```

极化会产生束缚电荷：

```math
\rho_p=-\nabla\cdot\vec P
```

介质表面的束缚面电荷密度为：

```math
\rho_{ps}=\vec P\cdot\vec n
```

## 电位移矢量

在介质中，高斯定律常写成：

```math
\oiint_S \vec D\cdot d\vec S=q_f
```

微分形式为：

```math
\nabla\cdot\vec D=\rho_f
```

这里的 $q_f$ 和 $\rho_f$ 指自由电荷。

这就是引入 $\vec D$ 的好处：它把介质极化带来的束缚电荷影响打包进材料参数里。

## 电介质中的基本方程

静电场仍满足：

```math
\nabla\times\vec E=0
```

线性介质中：

```math
\vec D=\varepsilon\vec E
```

所以无自由电荷区域可以写成：

```math
\nabla\cdot(\varepsilon\nabla\varphi)=0
```

如果 $\varepsilon$ 为常数，则退化为：

```math
\nabla^2\varphi=0
```

## 介质分界面的边界条件

设 $\vec n$ 从介质 2 指向介质 1，分界面上自由面电荷密度为 $\rho_s$。

法向 $D$ 的边界条件：

```math
\vec n\cdot(\vec D_1-\vec D_2)=\rho_s
```

如果分界面没有自由面电荷：

```math
D_{1n}=D_{2n}
```

切向 $E$ 的边界条件：

```math
\vec E_{1t}=\vec E_{2t}
```

等价地说：

```math
\vec n\times(\vec E_1-\vec E_2)=0
```

## 边界条件怎么记

电介质分界面可以这样记：

```text
法向看 D，切向看 E。
```

更具体地说：

- 法向 $D$ 是否跳变，由自由面电荷决定。
- 切向 $E$ 在静电场中连续。
- 电势 $\varphi$ 通常连续。
- 如果两侧介电常数不同，法向 $E$ 一般会改变。

## 电容

电容定义为：

```math
C=\frac{q}{U}
```

它表示导体系统储存电荷的能力。

求电容的常用步骤：

1. 假设导体带电量为 $q$。
2. 根据对称性或边界条件求出电场。
3. 由电场积分求电压 $U$。
4. 用 $C=q/U$ 得到电容。

## 静电能量

离散电荷系统的静电能可以写成：

```math
W_e=\frac12\sum_i q_i\varphi_i
```

连续分布时：

```math
W_e=\frac12\int_V \rho\varphi\,dV
```

用场来表示时，电场能量密度为：

```math
w_e=\frac12\vec D\cdot\vec E
```

线性介质中：

```math
w_e=\frac12\varepsilon E^2
```

## 易错点

- $\vec D$ 不是“另一种电场”，它是为了处理介质和自由电荷更方便引入的辅助量。
- 边界条件里要分清自由面电荷和束缚面电荷。
- $D_n$ 连续不代表 $E_n$ 连续，因为两侧 $\varepsilon$ 可能不同。
- 电容只由导体形状、相对位置和介质决定，和实际带多少电荷无关。
