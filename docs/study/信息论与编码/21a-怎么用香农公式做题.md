# 怎么用香农公式做题

## 这类题在考什么

这类题通常考：

```text
已知带宽和信噪比，求信道容量
```

本质上就是代香农公式。

## 先记公式

```math
C = B \log_2 \left(1+\frac{S}{N}\right)
```

参数解释：

- $C$：信道容量。
- $B$：带宽。
- $S$：信号功率。
- $N$：噪声功率。
- $S/N$：信噪比。

## 做题步骤

### 第一步：先看题目给的是不是直接信噪比

如果题目已经给了：

```text
S/N = 100
```

那最简单，直接代。

如果题目给的是：

- 信号功率 $S$
- 噪声功率 $N$

那先算：

```math
\frac{S}{N}
```

### 第二步：带宽单位看清楚

题目常给：

- `3000 Hz`
- `3 kHz`

本质是一样的，但代入时要统一。

### 第三步：代入公式

比如：

```math
B = 3000,\quad \frac{S}{N}=1000
```

则：

```math
C = 3000\log_2(1+1000)
```

## 结果怎么判断

记两个方向：

- 带宽越大，容量越大
- 信噪比越大，容量越大

但注意：

```text
带宽是线性影响
信噪比是对数影响
```

所以信噪比翻很多倍，容量不会同倍数暴涨。

## 最简单例子

如果：

```math
B=3000\text{ Hz},\quad \frac{S}{N}=1
```

则：

```math
C = 3000\log_2(2)=3000\text{ bit/s}
```

这题特别适合入门，因为：

```math
\log_2 2 = 1
```

一眼就能算。

## 易错点

- 别把 `S/N` 写成 `N/S`。
- 别忘了公式里是 `1 + S/N`。
- 对数底数是 2，不是 10。

## 注意点

- 题目如果问“理论极限”“最大可靠传输速率”，大概率就是香农公式。
- 简单题通常重点在代公式，不在复杂推导。
