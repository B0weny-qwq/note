# 怎么理解马尔可夫信源题

## 这类题在考什么

这类题通常考：

```text
当前符号是不是只和前面的少数状态有关
怎么从转移图或转移矩阵看出这种“有记忆”
```

小白最常遇到的是：

- 判断是不是一阶马尔可夫信源
- 根据转移概率图读状态转移关系
- 区分“有记忆信源”和“无记忆信源”

## 先记公式

一阶马尔可夫信源最核心的关系是：

```math
P(X_n \mid X_{n-1},X_{n-2},\ldots)=P(X_n \mid X_{n-1})
```

参数解释：

- $X_n$：当前时刻输出符号。
- $X_{n-1}$：前一个时刻输出符号。
- $X_{n-2},\ldots$：更早的历史符号。
- $P(\cdot|\cdot)$：条件概率。

它的意思是：

```text
当前输出只和前一个状态有关
不需要再看更久远的历史
```

## 做题步骤

### 第一步：先看题目给的是“图”还是“矩阵”

马尔可夫信源题常见两种给法：

- 状态转移图
- 状态转移矩阵

不管哪种给法，本质都在告诉你：

```text
当前状态转到下一个状态的概率是多少
```

### 第二步：先判断它是不是“有记忆”

如果当前符号概率会随着前一个状态变化，  
那它就不是无记忆信源，而是有记忆信源。

### 第三步：再判断记忆到哪一阶

如果只看前一个状态就够，  
那是一阶马尔可夫信源。

如果还要看前两个状态，  
那就是二阶。

### 第四步：把题目中的转移关系直接读出来

例如题目给：

```text
晴 -> 晴 概率 0.8
晴 -> 雨 概率 0.2
雨 -> 晴 概率 0.3
雨 -> 雨 概率 0.7
```

那你就可以直接说：

```text
当前天气只和前一天有关
这是一个一阶马尔可夫模型
```

## 最简单例子

如果题目给出转移矩阵：

```math
P=
\begin{bmatrix}
0.8 & 0.2\\
0.3 & 0.7
\end{bmatrix}
```

参数解释：

- 第 1 行表示当前状态为“晴”时的转移概率。
- 第 2 行表示当前状态为“雨”时的转移概率。

那就可以看出：

- 当前若是晴，下一时刻晴的概率是 $0.8$
- 当前若是雨，下一时刻晴的概率是 $0.3$

所以这是：

```text
一阶马尔可夫信源
```

## 易错点

- 马尔可夫“有记忆”，不等于“记住全部历史”。
- 别把信源有记忆和信道有记忆混在一起。
- 转移矩阵每一行的概率和应为 1。

## 注意点

- 这类题很多先考“会读图、会读矩阵”，不是一上来就考熵率推导。
- 小白先把“一阶 = 只看前一个状态”这句话记牢。
