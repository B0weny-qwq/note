# 怎么求互信息

## 这类题在考什么

互信息题考的是：

```text
知道 Y 以后，X 的不确定性减少了多少
```

也可以理解成：

```text
X 和 Y 共享了多少信息
```

## 先记公式

考试里最常用的是这两个：

```math
I(X;Y)=H(X)-H(X \mid Y)
```

```math
I(X;Y)=H(X)+H(Y)-H(X,Y)
```

## 做题步骤

### 第一步：先看题目给了什么

如果题目直接给了：

- $H(X)$
- $H(X \mid Y)$

那就直接用：

```math
I(X;Y)=H(X)-H(X \mid Y)
```

这是最省事的。

### 第二步：选最省事的公式

如果题目给的是联合概率分布表，  
通常要先求：

- $H(X)$
- $H(Y)$
- 或 $H(X,Y)$

再代公式。

## 最简单例子

如果题目给：

```math
H(X)=1.5,\quad H(X \mid Y)=0.4
```

那直接：

```math
I(X;Y)=1.5-0.4=1.1
```

## 结果怎么判断合理

记三个结论：

- $I(X;Y)\ge 0$
- 如果 $X$ 和 $Y$ 独立，则 $I(X;Y)=0$
- 如果 $Y$ 完全决定 $X$，则 $I(X;Y)=H(X)$

所以如果你算出负数，那一定错了。

## 小白做题最稳的方法

如果题目不是特别绕，优先找：

```text
有没有 H(X)
有没有 H(X|Y)
```

因为：

```text
互信息 = 原来有多不确定 - 知道之后还剩多少不确定
```

这个理解最直观，也最不容易代错。

## 易错点

- 互信息不是条件熵，别把 `-` 号方向写反。
- 独立时互信息为 0，不是熵为 0。
- 互信息是对称的，`I(X;Y)=I(Y;X)`。

## 注意点

- 简单题一般不会逼你从最原始定义双重求和硬算。
- 先看题目给的是熵、条件熵，还是联合分布表，再选公式。
