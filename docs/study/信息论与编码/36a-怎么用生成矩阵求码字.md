# 怎么用生成矩阵求码字

## 这类题在考什么

这类题通常考：

```text
已知信息组 m 和生成矩阵 G
求对应码字 c
```

这属于线性分组码里最基础的计算题。

## 先记公式

```math
c=mG
```

参数解释：

- $c$：码字。
- $m$：信息组或消息向量。
- $G$：生成矩阵。
- 运算都在模 2 下进行。

## 做题步骤

### 第一步：先看向量是行向量还是列向量

大多数教材写法是：

```math
mG
```

所以通常把 $m$ 写成行向量。

### 第二步：检查维数能不能乘

如果 $G$ 是 $k \times n$，  
那信息组 $m$ 应该是 $1 \times k$。

### 第三步：按矩阵乘法做，但结果对 2 取模

最关键的是：

```text
1 + 1 = 0
```

因为这里是模 2 运算。

### 第四步：把每一位都算出来

不要只写中间过程，  
最后要明确写出完整码字。

## 最简单例子

如果：

```math
m=[1\ 0]
```

```math
G=
\begin{bmatrix}
1 & 0 & 1\\
0 & 1 & 1
\end{bmatrix}
```

则：

```math
c=[1\ 0]
\begin{bmatrix}
1 & 0 & 1\\
0 & 1 & 1
\end{bmatrix}
=[1\ 0\ 1]
```

所以码字是：

```math
c=[1\ 0\ 1]
```

## 易错点

- 别把 $c=mG$ 写成 $Gm$。
- 别用普通整数加法，必须模 2。
- 别忘了最后把结果写成码字向量。

## 注意点

- 如果 $G$ 是系统码形式，结果里通常能直接看到原信息位。
- 这题和“校验矩阵验码字”是一对，一个负责生成，一个负责检查。
