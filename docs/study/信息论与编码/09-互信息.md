# 互信息

## 定义

互信息表示两个变量共享了多少信息。

记作：

```math
I(X;Y)
```

定义式：

```math
I(X;Y) = \sum_{i=1}^{n}\sum_{j=1}^{m} p(x_i,y_j)\log \frac{p(x_i \mid y_j)}{p(x_i)}
```

## 常用公式

```math
I(X;Y) = H(X) - H(X \mid Y)
```

```math
I(X;Y) = H(Y) - H(Y \mid X)
```

```math
I(X;Y) = H(X) + H(Y) - H(X,Y)
```

参数解释：

- $X, Y$：输入与输出，或两个相关随机变量。
- $x_i, y_j$：各自的可能取值。
- $p(x_i,y_j)$：联合概率。
- $p(x_i)$：边缘概率。
- $p(x_i \mid y_j)$：条件概率。
- $I(X;Y)$：$Y$ 对 $X$ 不确定度减少的平均量，也等价于共享信息量。

## 直觉理解

如果知道 `Y` 之后，`X` 的不确定性下降很多，那么 `X` 和 `Y` 的互信息就大。

## 详细例子

### 例 1：完全无关

如果 `X` 和 `Y` 独立：

```math
I(X;Y)=0
```

因为知道 `Y` 对猜 `X` 没帮助。

### 例 2：完全决定

如果 `Y=X`，也就是 `Y` 完全由 `X` 决定：

```math
I(X;Y)=H(X)
```

因为知道 `Y` 就等于知道 `X`。

## 生活类比

看天气预报和决定是否带伞之间存在互信息。  
如果天气预报很准，它能大幅减少你对“是否带伞”的不确定性，互信息就大。

## 易错点

- 互信息不是“交集大小”的直观面积，而是概率意义上的共享信息量。

## 注意点

- 第 3 章的信道容量本质上就是让互信息尽可能大。
