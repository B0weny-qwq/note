# 怎么判断失真函数怎么选

## 这类题在考什么

这类题通常考：

```text
什么场景该用什么失真函数
汉明失真和平方误差失真怎么区分
```

很多时候不是让你复杂计算，  
而是让你先选对“尺子”。

## 先记公式

汉明失真：

```math
d(x,y)=
\begin{cases}
0, & x=y\\
1, & x\ne y
\end{cases}
```

参数解释：

- $x$：原始符号。
- $y$：重构符号。

平方误差失真：

```math
d(x,y)=(x-y)^2
```

参数解释：

- $x,y$：原始值和重构值。

## 做题步骤

### 第一步：先看题目是离散符号还是连续数值

如果是：

- 字符
- 比特
- 分类标签

这类“对/错”明显的离散对象，  
优先想到汉明失真。

### 第二步：如果是连续量，优先看数值偏差

如果是：

- 电压
- 温度
- 图像灰度

这类连续量，  
更常用平方误差失真。

### 第三步：先问自己“只关心对错，还是关心差多远”

- 只关心对错：汉明失真
- 关心偏差大小：平方误差类

### 第四步：答题时把原因写出来

这比只写一个名字更稳。

## 最简单例子

如果题目是字符传输：

```text
A 重构成 A
A 重构成 B
```

这时只关心“对没对”，  
最自然的失真函数就是：

```text
汉明失真
```

如果题目是温度测量：

```text
真实温度 25.2
重构温度 25.0
```

这时显然更关心差了多少，  
更适合：

```text
平方误差失真
```

## 易错点

- 失真函数不是固定一种，要看业务场景。
- 汉明失真不是“所有通信题都默认用”的万能尺子。
- 平方误差失真不是用来判断字符对错的。

## 注意点

- 一旦失真函数选定，后面的平均失真和率失真分析都会跟着它走。
- 小白做题先学会选“合适的度量尺”，比上来套公式更重要。
