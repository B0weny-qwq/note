# 怎么区分信源编码和信道编码

## 这类题在考什么

这类题通常考：

```text
信源编码干什么
信道编码干什么
它们的目标为什么不一样
```

这是全课程最基础的区分题之一。

## 先记公式

这一节没有硬算公式，  
最关键的是记两句话：

```text
信源编码：去冗余，压缩
信道编码：加冗余，抗差错
```

## 做题步骤

### 第一步：先看目标是“省”还是“稳”

- 想少传一点，想到信源编码
- 想传得更可靠，想到信道编码

### 第二步：再看它对冗余的态度

- 信源编码尽量去掉冗余
- 信道编码主动加入冗余

### 第三步：答题时把“为什么”写出来

因为：

- 信源编码面向压缩效率
- 信道编码面向抗噪声和纠错

### 第四步：不要觉得它们矛盾

它们作用在通信系统的不同环节，  
目标不同，所以做法不同。

## 最简单例子

如果题目问：

```text
为什么信源编码去冗余，而信道编码又加冗余？
```

最稳的答法是：

```text
信源编码是为了压缩，提高表示效率；
信道编码是为了抵抗信道噪声，提高传输可靠性。
```

## 易错点

- 不要把两者都说成“编码所以差不多”。
- 信源编码不是纠错编码。
- 信道编码不是压缩编码。

## 注意点

- 一看到“压缩”优先想到信源编码，一看到“纠错/抗噪声”优先想到信道编码。
