# 考试大纲复习总览

这门课按考试大纲可以整理成六条主线：

```text
绪论 -> 信源与信息熵 -> 信道与信道容量 -> 信息率失真函数 -> 信源编码 -> 信道编码
```

复习时不要只按公式背，要按“研究对象、核心量、典型题型”去对齐。

## 第 1 章：绪论

大纲要求：

- 信息的概念和特征
- 信息论的形成和发展
- 信息论的研究内容
- 通信系统模型
- 信息论的应用

对应笔记：

- [什么是信息论与编码](./01-什么是信息论与编码.md)
- [信息、信号与信道的区别](./02-信息信号与信道的区别.md)
- [信息论的形成发展与应用](./01b-信息论的形成发展与应用.md)

考试抓法：

```text
信息论研究信息度量、压缩极限和可靠传输极限；
通信系统主线是信源、编码、信道、译码、信宿。
```

- 信源冗余度

自信息量看单个事件；

对应笔记：

- [失真矩阵与平均失真](./23-失真矩阵与平均失真.md)

```text

```

## 最短复习路线

时间很紧时按这个顺序过：

1. 先看本页，把六章边界分清。
2. 第 2 章重点背熵、条件熵、联合熵、互信息。
3. 第 3 章重点背信道容量、BSC、强对称信道、高斯信道。
4. 第 4 章重点背失真矩阵、平均失真、R(D) 性质。
5. 第 5 章重点会画码树、会做霍夫曼、会算平均码长。
6. 第 6 章重点会用 $d_{\min}$ 判断检错纠错、会用 $G/H$ 矩阵做题、会算伴随式。
