# 怎么求自信息量

## 这类题在考什么

这类题通常就考一句话：

```text
某个事件发生一次，带来了多少信息量？
```

本质上就是代公式。

## 先记公式

```math
I(x_i)=\log_2 \frac{1}{p(x_i)}=-\log_2 p(x_i)
```

参数解释：

- $x_i$：某个具体事件或符号。
- $p(x_i)$：这个事件发生的概率。
- $I(x_i)$：这个事件的自信息量。

## 做题步骤

### 第一步：先找概率

不要急着按计算器。  
先把题目里这个事件的概率单独找出来。

比如：

- 抛公平硬币正面：`p = 1/2`
- 骰子掷出 6：`p = 1/6`
- 某符号出现概率：题目直接给

### 第二步：代入公式

直接代：

```math
I(x)=-\log_2 p(x)
```

### 第三步：看结果大小合不合理

记一个判断：

- 概率越大，自信息量越小
- 概率越小，自信息量越大

如果你算出来“概率很小但信息量也很小”，那就是算错了。

## 最简单例子

### 例 1：公平硬币

正面概率是：

```math
p=1/2
```

所以：

```math
I(\text{正面})=-\log_2(1/2)=1\text{ bit}
```

### 例 2：骰子掷出 6

概率是：

```math
p=1/6
```

所以：

```math
I(6)=-\log_2(1/6)=\log_2 6
```

这说明它的信息量比硬币正面更大，因为它更少见。

## 考试时怎么快速判断

如果题目只让你比较大小，不一定要精算：

- `1/2` 的信息量小于 `1/4`
- `1/10` 的信息量大于 `1/5`

原因就一句话：

```text
越稀有，越“意外”，信息量越大
```

## 易错点

- 别把自信息量和熵混了，自信息量是“单个事件”，熵是“平均情况”。
- 对数底数默认看清楚，信息论考试里通常是 2。
- 概率必须在 0 到 1 之间，不能乱代频数。

## 注意点

- 这类题通常是送分题，关键不是难，而是别抄错概率。
- 看到“求某事件信息量”，就立刻想到 `-\log_2 p`。
