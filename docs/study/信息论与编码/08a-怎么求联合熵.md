# 怎么求联合熵

## 这类题在考什么

联合熵题考的是：

```text
把 X 和 Y 一起看，总共有多少平均不确定性
```

简单理解：

```text
一个变量不够，就两个一起算
```

## 先记公式

定义式：

```math
H(X,Y)=-\sum_{i=1}^{n}\sum_{j=1}^{m} p(x_i,y_j)\log_2 p(x_i,y_j)
```

考试里更常用的关系式：

```math
H(X,Y)=H(X)+H(Y \mid X)
```

或者：

```math
H(X,Y)=H(Y)+H(X \mid Y)
```

## 做题步骤

### 第一步：先看题目给的是概率表还是熵

如果题目直接给了：

- $H(X)$
- $H(Y|X)$

那直接相加。

### 第二步：如果给的是联合概率表，就按定义式算

也就是把每个格子的联合概率都拿出来，  
逐项做：

```text
p(xi,yj) × log p(xi,yj)
```

再全部加总。

### 第三步：看独立不独立

如果题目明确说：

```text
X 和 Y 独立
```

那就能直接写：

```math
H(X,Y)=H(X)+H(Y)
```

## 最简单例子

如果题目给：

```math
H(X)=1,\quad H(Y \mid X)=0.5
```

则：

```math
H(X,Y)=1+0.5=1.5
```

## 怎么判断结果合理

记两个判断：

- 联合熵不会小于单个变量熵太离谱
- 如果独立，联合熵就是两个熵相加

比如：

```math
H(X)=1,\quad H(Y)=1
```

如果还独立，那联合熵就该是：

```math
H(X,Y)=2
```

## 易错点

- 联合熵不是任何时候都等于 `H(X)+H(Y)`，只有独立时才成立。
- 条件熵和联合熵不要混着写。
- 用概率表时别漏格子。

## 注意点

- 简单题里先找有没有现成的关系式可用，别上来就双重求和。
- 一旦题目提到“两个变量一起”，先想到联合熵。
