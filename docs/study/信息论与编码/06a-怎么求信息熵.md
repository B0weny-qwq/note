# 怎么求信息熵

## 这类题在考什么

信息熵题本质上考的是：

```text
一个信源平均每次输出，带来多少信息量
```

和自信息量不同，它不是看某一个事件，  
而是看“所有事件按概率加权后的平均值”。

## 先记公式

```math
H(X)=-\sum_{i=1}^{n} p(x_i)\log_2 p(x_i)
```

参数解释：

- $X$：离散信源。
- $x_i$：第 $i$ 个可能输出符号。
- $p(x_i)$：符号 $x_i$ 的概率。
- $H(X)$：信源熵。

## 做题步骤

### 第一步：把所有概率列全

先检查题目是不是把所有符号概率都给齐了。  
如果没给齐，先补齐。

比如：

```text
p1 = 0.5, p2 = 0.3
```

那第三个概率可能要自己补：

```math
p_3 = 1 - 0.5 - 0.3 = 0.2
```

### 第二步：一项一项代进去

不要心急合并，先写成：

```math
H(X)=-(p_1\log_2 p_1 + p_2\log_2 p_2 + \cdots + p_n\log_2 p_n)
```

这样最不容易漏项。

### 第三步：看结果是否合理

记三条判断：

- 熵一定不小于 0
- 分布越平均，熵越大
- 分布越偏，熵越小

## 最简单例子

### 例 1：等概二元信源

```math
P(0)=1/2,\quad P(1)=1/2
```

则：

```math
H(X)= -\left(\frac12\log_2\frac12 + \frac12\log_2\frac12\right)=1
```

这是二元信源熵最大的情况。

### 例 2：偏置信源

```math
P(0)=0.9,\quad P(1)=0.1
```

此时：

```text
熵会小于 1
```

因为这个信源很偏，输出更容易猜。

## 考试时最常见的坑

### 坑 1：漏掉负号

很多人会写成：

```math
\sum p\log p
```

这是错的，前面一定有负号。

### 坑 2：概率没加到 1

如果题目给的概率和不是 1，先补概率，再算熵。

### 坑 3：把熵和自信息量混掉

熵是平均量。  
不是看某个符号，而是所有符号一起算。

## 怎么快速判断答案大概对不对

对于二元信源：

- 最平均时熵最大，为 `1 bit`
- 越偏离 `0.5/0.5`，熵越小

所以你一算出：

```text
概率是 0.9 / 0.1，结果却比 1 还大
```

那肯定错。

## 易错点

- 对数底数通常取 2。
- 别漏任何一个概率项。
- 熵不是“最大概率对应的信息量”，而是总平均。

## 注意点

- 题目一旦说“平均信息量”“平均不确定度”，优先想到熵。
- 简单题最稳的做法就是一项一项展开，别口算过头。
