# 怎么用最小码距判断检错和纠错

## 这类题在考什么

这类题通常考：

```text
已知 dmin，问最多能检几位错、最多能纠几位错
```

这是考试里特别高频的送分题。

## 先记公式

检错能力：

```math
e = d_{\min}-1
```

纠错能力：

```math
t = \left\lfloor \frac{d_{\min}-1}{2} \right\rfloor
```

## 做题步骤

### 第一步：先看最小码距是多少

题目如果给：

```math
d_{\min}=3
```

那后面就直接代。

### 第二步：先求检错能力

```math
e=d_{\min}-1
```

### 第三步：再求纠错能力

```math
t=\left\lfloor \frac{d_{\min}-1}{2} \right\rfloor
```

## 最简单例子

如果：

```math
d_{\min}=3
```

则：

```math
e=3-1=2
```

```math
t=\left\lfloor \frac{3-1}{2}\right\rfloor=1
```

所以：

- 最多能检出 2 位错
- 最多能纠正 1 位错

## 这题怎么快速记

直接记成一句：

```text
能检 dmin-1 位
能纠一半向下取整
```

## 为什么纠错比检错弱

因为“知道出错了”和“知道错在哪”不是一回事。

检错只要分辨：

```text
对 / 不对
```

纠错却要进一步分辨：

```text
到底是哪一位错了
```

所以纠错要求更强。

## 易错点

- 别把检错公式和纠错公式混掉。
- 纠错时要向下取整。
- `dmin=3` 时是纠 1 位，不是纠 2 位。

## 注意点

- 题目如果只问“单错纠正需要什么最小码距”，倒过来想就是 `dmin >= 3`。
- 这是第 6 章非常高频的基础题，必须一眼反应出来。
