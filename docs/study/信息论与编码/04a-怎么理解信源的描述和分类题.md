# 怎么理解信源的描述和分类题

## 这类题在考什么

这类题通常考：

```text
一个信源按什么标准分类
离散、连续、无记忆、有记忆怎么判断
```

这部分是后面所有章节的前置概念题。

## 先记公式

这一节重点是分类规则，不是复杂公式。  
最常记的几个分类维度是：

```text
按取值：离散 / 连续
按时间：离散时间 / 连续时间
按记忆性：无记忆 / 有记忆
```

## 做题步骤

### 第一步：先看取值是离散还是连续

如果只可能取若干符号，  
就是离散信源。

如果可取连续数值，  
就是连续信源。

### 第二步：再看时间上是逐拍输出还是连续变化

- 一个一个符号往外出，偏离散时间
- 随时间连续变化，偏连续时间

### 第三步：再看有没有记忆

- 当前输出和过去无关：无记忆
- 当前输出受过去影响：有记忆

### 第四步：答题时尽量按“维度”分类

不要一股脑只写一个标签。

## 最简单例子

比如键盘输入文字：

- 字符集合是离散的
- 输入通常按一个一个字符产生
- 如果简化看成每次独立，可视作离散无记忆信源

再比如语音波形：

- 幅度取值近似连续
- 随时间连续变化
- 前后样本强相关

所以它更像：

```text
连续、有记忆信源
```

## 易错点

- “离散/连续”和“有无记忆”是不同维度。
- 不要把时间连续和取值连续混为一谈。
- 一个信源完全可能同时是“离散时间 + 连续幅值”。

## 注意点

- 这类题看似简单，但后面很多公式能不能用，就取决于你前面分类有没有分对。
