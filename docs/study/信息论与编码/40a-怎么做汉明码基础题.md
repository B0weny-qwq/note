# 怎么做汉明码基础题

## 这类题在考什么

汉明码基础题通常考：

- 已知信息位数，求监督位数
- 判断它能不能单错纠正
- 收到码字后怎么定位 1 位错误

这部分就是典型的考试基础分。

## 先记公式

汉明码监督位常满足：

```math
2^r \ge k+r+1
```

也常写成：

```math
2^r \ge n+1
```

参数解释：

- $r$：监督位个数。
- $k$：信息位个数。
- $n$：码长，且 $n=k+r$。

另外，汉明码用于单错纠正时通常满足：

```math
d_{\min}=3
```

参数解释：

- $d_{\min}$：最小码距。

## 做题步骤

### 第一步：如果题目让求监督位，先列覆盖不等式

已知信息位数 $k$ 后，  
逐个试 $r=2,3,4,\dots$

直到满足：

```math
2^r \ge k+r+1
```

### 第二步：如果题目问能不能单错纠正，先看最小码距

单错纠正最基本条件是：

```math
d_{\min}\ge 3
```

汉明码正是最经典的单错纠正码。

### 第三步：如果题目给了收码，通常去求伴随式

也就是算：

```math
S=rH^T
```

如果伴随式对应某一列，  
就说明那一位出错。

### 第四步：定位错误位后，把那一位翻转

这一题最朴素的纠错动作就是：

```text
0 变 1
1 变 0
```

## 最简单例子

如果信息位数为：

```math
k=4
```

试：

```math
r=2 \Rightarrow 2^2=4<4+2+1=7
```

不够。

再试：

```math
r=3 \Rightarrow 2^3=8\ge 4+3+1=8
```

成立。

所以 4 位信息至少需要：

```math
r=3
```

也就是得到常见的 $(7,4)$ 汉明码。

## 易错点

- 别把条件写成 $2^r \ge k+1$，少了监督位本身。
- 别把“能检错”和“能纠错”混在一起。
- 纠错时不要只说“有错”，还要指出哪一位错。

## 注意点

- 一看到 $(7,4)$、单错纠正、监督位个数，优先想到汉明码。
- 汉明码基础题往往和信息位、监督位、伴随式连着考。
