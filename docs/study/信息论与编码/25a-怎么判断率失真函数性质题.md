# 怎么判断率失真函数性质题

## 这类题在考什么

这类题通常考：

```text
R(D) 随 D 怎么变
它会不会小于 0
什么时候会变成 0
```

本质上是标准性质判断题。

## 先记公式

最常记的几个结论是：

```math
R(D)\ge 0
```

```math
R(D)\ \text{单调不增}
```

并且当失真放宽到一定程度后，可能有：

```math
R(D)=0
```

参数解释：

- $R(D)$：允许平均失真不超过 $D$ 时所需的最小码率。
- $D$：允许失真。

## 做题步骤

### 第一步：先抓最大方向关系

最重要的一条就是：

```text
D 越大
R(D) 越小或不增
```

### 第二步：再判断非负性

码率不可能是负的，  
所以：

```math
R(D)\ge 0
```

### 第三步：看到“失真很宽松”就想到可能降到 0

如果题目允许的失真已经大到几乎无所谓，  
那就可能无需再传额外信息。

### 第四步：如果题目提到曲线形状，再补“下凸”

很多教材会给出：

```text
R(D) 一般是下凸函数
```

## 最简单例子

如果题目问：

```text
当 D 增大时，R(D) 增大还是减小？
```

直接答：

```text
减小或不增
```

如果题目问：

```text
R(D) 会不会小于 0？
```

直接答：

```text
不会
```

## 易错点

- 允许失真更多，不是更难压缩，而是更容易压缩。
- `R(D)=0` 不等于“系统没有输出”。
- 性质题很多不需要你去推一般公式。
