# 联合熵

## 定义

联合熵表示两个变量一起提供的平均信息量。

记作：

```math
H(X,Y)
```

定义式可写成：

```math
H(X,Y) = -\sum_{i=1}^{n}\sum_{j=1}^{m} p(x_i, y_j)\log p(x_i, y_j)
```

## 直觉理解

如果只知道一个变量不够，那把两个变量一起看，信息就更多。

## 公式关系

最常用关系：

```math
H(X,Y) = H(X) + H(Y \mid X)
```

也可以写成：

```math
H(X,Y) = H(Y) + H(X \mid Y)
```

参数解释：

- $X, Y$：两个离散随机变量。
- $x_i, y_j$：对应变量的可能取值。
- $p(x_i, y_j)$：$X=x_i, Y=y_j$ 同时发生的联合概率。
- $H(X,Y)$：两个变量一起看的平均不确定度。

## 详细例子

抛两次硬币：

- 第一次结果是 `X`
- 第二次结果是 `Y`

如果两次独立，且都均匀：

```math
H(X)=1,\quad H(Y)=1,\quad H(X,Y)=2
```

因为知道第一次并不能帮你预测第二次。

## 生活类比

一个人的“姓名”不够唯一，再加上“班级”，联合起来信息更多，更容易唯一确定这个人。

## 易错点

- 联合熵不是简单把两个熵机械相加，是否相加成立要看独立性。

## 注意点

- 一旦变量相关，联合熵一般小于两个单独熵的简单相加。
