# 率失真函数的性质

## 单调不增

允许失真越大，最低码率不可能反而更高，所以：

```math
R(D)\ \text{单调不增}
```

参数解释：

- $R(D)$：率失真函数。
- $D$：允许失真。
- “单调不增”表示允许失真越大，最低码率不会反而上升。

## 下凸

$R(D)$ 一般是下凸函数。  
这表示在不同失真策略之间做混合时，整体表现具有良好的凸性结构。

## Dmax

当失真容忍度大到某个程度后，系统几乎可以“不认真还原”也满足要求，这时：

```math
R(D) = 0
```

这里对应的是失真已经放宽到某个上限后，无需再传额外信息也能满足要求。

这个阈值通常记作 `Dmax`。

## 详细例子

如果你给同学传一张头像：

- 要求看清五官，码率不能太低
- 只要求分清“是不是个人”，码率可以低很多
- 如果只要求“随便给个色块都行”，那几乎不用传什么信息

## 生活类比

像做手绘临摹：

- 要求百分百还原，工作量大
- 允许简笔画风格，工作量下降
- 只要能看出大概轮廓，甚至几笔就够

## 易错点

- `R(D)=0` 不代表系统没输出，而是“无需传递额外信息也能满足失真要求”。

## 注意点

- 这些性质能帮助你在做题时先判断答案是否合理。
