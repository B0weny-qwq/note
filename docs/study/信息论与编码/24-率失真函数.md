# 率失真函数

## 定义

率失真函数表示：在平均失真不超过 `D` 的条件下，最少需要多少码率。

公式：

```math
R(D) = \min_{P_D} I(X;Y)
```

课件里同时给出约束条件：

```math
P_D = \left\{ p(y_j \mid x_i)\; \middle|\; D \le D_0 \right\}
```

也就是在满足平均失真约束的所有编码信道里，取互信息最小值。

参数解释：

- $R(D)$：允许平均失真不超过 $D$ 时所需的最小码率。
- $I(X;Y)$：输入与重构输出之间的互信息。
- $P_D$：满足失真约束的编码信道集合。
- $p(y_j \mid x_i)$：输入 $x_i$ 被重构成 $y_j$ 的条件概率。
- $D$：允许的平均失真上限。

## 直觉理解

如果允许失真变大，压缩就能更狠，所需码率就会下降。  
如果要求完全不失真，码率就必须更高。

## 图像意义

`R(D)` 曲线通常是单调下降的：

- `D` 越小，要求越严格，`R(D)` 越高
- `D` 越大，允许失真越多，`R(D)` 越低

## 详细例子

你发一张照片给同学：

- 如果要求原图级别，一点都不能糊，文件就很大
- 如果允许“肉眼看起来差不多”，就可以压得很小

这就是率失真思想。

## 生活类比

搬家时装衣服：

- 不允许一点褶皱，就得平铺，占很多箱子
- 允许轻微压皱，就能压进更小空间

这里“压皱程度”像失真，“箱子体积”像码率。

## 易错点

- `R(D)` 是“最小互信息”，不是“任意满足条件的互信息”。
