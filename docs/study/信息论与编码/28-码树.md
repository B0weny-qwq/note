# 码树

## 码树是什么

码树是用树形结构表示码字集合的方法。

在二元码里：

- 每个分支可标 0 或 1
- 从根到叶子的一条路径就是一个码字

## 为什么有用

码树能直接看出：

- 哪些码字互为前缀
- 是否是即时码
- 如何构造 Shannon/Fano/Huffman 码

## 详细例子

如果：

- `A -> 0`
- `B -> 10`
- `C -> 11`

那么码树上：

- `A` 在左支叶节点
- `B` 和 `C` 在右子树的两个叶节点

因为叶节点不再延长，所以这是即时码。

## 生活类比

像岔路地图：

- 从起点走不同岔路，最后到不同终点
- 每条完整路线就像一个码字

## 易错点

- 叶节点下面如果还继续长分支，通常就意味着存在前缀关系问题。

## 注意点

- 课件里很多唯一可译码和即时码判断题，最快方法就是画码树。
