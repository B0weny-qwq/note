# 怎么判断是不是循环码

## 这类题在考什么

这类题通常考：

```text
给你一组码字
问它是不是循环码
```

本质上考的是：

```text
码字循环移位后
还在不在原来的码字集合里
```

## 先记公式

循环码最核心的判定思想是：

```text
任一码字经过循环移位后
仍然是码字
```

如果题目强调它是二元线性分组码，  
那就在这基础上再检查循环性质。

## 做题步骤

### 第一步：先确认码字长度一致

长度不一致，  
那连分组码都谈不上，更不用说循环码。

### 第二步：任取一个码字做循环移位

例如：

```text
101100
```

循环右移一位可得到：

```text
010110
```

### 第三步：检查移位后的结果还在不在码字集合里

如果不在，  
那它就不是循环码。

### 第四步：多检查几个码字或说明一般性

简单题里，找出一个反例通常就够了。  
因为循环码要求对所有码字都成立。

## 最简单例子

若某组码字中有：

```text
101100
```

把它循环移位后得到：

```text
010110
```

如果这个新串不在给定码字集合中，  
那就可以直接写：

```text
该码不是循环码
```

## 易错点

- 普通左移右移和循环移位不是一回事。
- 只看一个码字成立，不代表整体就是循环码。
- 循环码首先还是线性分组码的一类。

## 注意点

- 题目如果只让你“判断”，最稳的是找反例。
- 一旦进入后续计算题，往往要切到多项式视角。
