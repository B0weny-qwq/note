# 怎么理解高斯信道题

## 这类题在考什么

这类题通常考：

```text
什么叫加性高斯白噪声信道
题目一旦给出这个模型该想到什么
```

最基础的题一般是：

- 认出 AWGN 模型
- 说清各个量是什么意思
- 判断容量和带宽、信噪比的关系

## 先记公式

高斯白噪声信道的基本模型是：

```math
Y=X+N
```

参数解释：

- $X$：发送信号。
- $Y$：接收信号。
- $N$：噪声。

如果题目进一步考容量，通常会联想到：

```math
C=B\log_2\left(1+\frac{S}{N}\right)
```

参数解释：

- $C$：信道容量。
- $B$：带宽。
- $S$：信号功率。
- $N$：噪声功率。

## 做题步骤

### 第一步：先认出“加性高斯白噪声”这几个词

- `加性`：噪声是直接加上去的
- `高斯`：噪声服从高斯分布
- `白噪声`：在频域上统计特性比较均匀

### 第二步：把模型读成“收到的是信号加噪声”

看到：

```math
Y=X+N
```

先不要慌，先翻成人话：

```text
接收端拿到的不是纯净信号
而是原信号叠加了随机扰动
```

### 第三步：如果题目给了带宽和信噪比，马上想到香农公式

这类题常常不是单独考高斯分布性质，  
而是借 AWGN 场景去考：

```text
容量怎么算
```

### 第四步：会判断两个基本方向

- 带宽越大，容量越大
- 信噪比越大，容量越大

## 最简单例子

如果题目说：

```text
某信道满足 Y = X + N，
其中 N 是高斯白噪声
```

那你至少要立刻反应出：

```text
这是 AWGN 信道
接收信号等于发送信号加噪声
```

如果再给：

```math
B=3000\text{ Hz},\quad \frac{S}{N}=1
```

则：

```math
C=3000\log_2(1+1)=3000\text{ bit/s}
```

## 易错点

- `Y=X+N` 里的 $N$ 是噪声，不是噪声功率。
- 香农公式里的 $N$ 指噪声功率，别和上面的随机变量混掉。
- 连续信道不等于“无限容量”。

## 注意点

- 这类题最先做的是认模型，不是上来就背一堆高斯分布细节。
- 一看到 AWGN，很多基础题其实都会落回香农公式。
