# 怎么求平均失真

## 这类题在考什么

平均失真题考的是：

```text
长期平均下来，重构结果和原结果差了多少
```

它的思路和“平均码长”很像，  
本质也是加权平均。

## 先记公式

```math
D=\sum_{i=1}^{n}\sum_{j=1}^{m} p(x_i,y_j)d(x_i,y_j)
```

也常写成：

```math
D=\sum_{i=1}^{n}\sum_{j=1}^{m} p(x_i)p(y_j \mid x_i)d(x_i,y_j)
```

## 做题步骤

### 第一步：先看失真矩阵

先搞清楚：

```text
哪个组合失真是 0
哪个组合失真是 1
哪个组合是别的值
```

### 第二步：再看概率

看题目给的是：

- 联合概率 `p(x_i,y_j)`
- 还是源概率和条件概率

### 第三步：按“概率 × 失真”逐项求和

就是：

```text
每一种情况发生的概率
乘上它带来的失真
最后全加起来
```

## 最简单例子

如果是二元汉明失真：

```text
对了失真 0
错了失真 1
```

并且题目给：

```math
p(0,1)=0.2,\quad p(1,0)=0.1
```

那平均失真就是：

```math
D = 0.2\times1 + 0.1\times1 = 0.3
```

因为正确的那些项都乘 0，不贡献失真。

## 小白做题时最稳的想法

先别急着套大公式，先想：

```text
哪些情况算错了
哪些情况没错
```

如果是汉明失真，这样做特别快。

## 易错点

- 失真矩阵不是概率矩阵。
- 别把 `p(y|x)` 直接当平均失真。
- 对于失真为 0 的项，不要忘了它乘出来就是 0。

## 注意点

- 汉明失真题最常见，往往就是“错的概率之和”。
- 题目一旦给失真矩阵和转移概率表，就想到平均失真。
