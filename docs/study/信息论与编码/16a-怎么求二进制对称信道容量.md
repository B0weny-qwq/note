# 怎么求二进制对称信道容量

## 这类题在考什么

这类题通常考：

```text
已知交叉概率 p
求二进制对称信道的容量
```

这是第 3 章里非常典型的基础计算题。

## 先记公式

```math
C=1-H(p)
```

其中：

```math
H(p)=-p\log_2 p-(1-p)\log_2(1-p)
```

参数解释：

- $C$：二进制对称信道每次使用的容量，单位通常是 bit/符号。
- $p$：交叉概率，也就是发 0 收成 1 或发 1 收成 0 的概率。
- $H(p)$：二元熵函数。

## 做题步骤

### 第一步：先认出这是不是 BSC

二进制对称信道的特点是：

- 只有两个输入符号
- 错转概率左右对称

通常转移关系是：

```math
P(1|0)=P(0|1)=p
```

### 第二步：先写容量公式

一认出是 BSC，就先写：

```math
C=1-H(p)
```

### 第三步：把题目给的交叉概率代进去

如果题目给：

```math
p=0.1
```

那就先算：

```math
H(0.1)=-0.1\log_2 0.1-0.9\log_2 0.9
```

再用：

```math
C=1-H(0.1)
```

### 第四步：会看两个极端情况

- 当 $p=0$ 时，信道完全无错，容量最大
- 当 $p=0.5$ 时，输出完全乱掉，容量为 0

## 最简单例子

如果：

```math
p=0
```

则：

```math
H(0)=0
```

所以：

```math
C=1-0=1
```

意思是每传 1 个二进制符号，最多能可靠传 1 bit 信息。

再比如：

```math
p=0.5
```

则：

```math
H(0.5)=1
```

所以：

```math
C=1-1=0
```

## 易错点

- 别把 $p$ 看成“正确概率”，这里它是交叉概率。
- 别忘了先算 $H(p)$，再算 $1-H(p)$。
- 当 $p=0.5$ 时容量是 0，不是 0.5。

## 注意点

- 如果题目只问趋势，也要会答：$p$ 越大，容量越小。
- BSC 容量题最值钱的不是死算，而是会认模型、会看极端情况。
