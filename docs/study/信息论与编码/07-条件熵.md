# 条件熵

## 定义

条件熵表示：在已知另一个变量之后，当前变量还剩下多少不确定性。

公式：

```math
H(X \mid Y) = \sum_{j=1}^{m} p(y_j) H(X \mid y_j)
```

也常展开写成：

```math
H(X \mid Y) = -\sum_{j=1}^{m}\sum_{i=1}^{n} p(x_i, y_j)\log p(x_i \mid y_j)
```

参数解释：

- $X, Y$：两个离散随机变量。
- $x_i$：$X$ 的第 $i$ 个可能取值。
- $y_j$：$Y$ 的第 $j$ 个可能取值。
- $p(y_j)$：事件 $Y=y_j$ 的概率。
- $p(x_i, y_j)$：联合概率。
- $p(x_i \mid y_j)$：在已知 $Y=y_j$ 时，$X=x_i$ 的条件概率。
- $H(X \mid y_j)$：条件 $Y=y_j$ 下，$X$ 的条件熵。

## 直觉理解

知道更多背景信息后，通常更容易猜测结果，所以不确定性会下降。

## 详细例子

假设 `X` 是“今天是否带伞”，`Y` 是“天气是否下雨”。

- 如果不知道天气，`X` 比较难猜。
- 如果已经知道在下大雨，`X` 更容易猜成“带伞”。

于是：

```math
H(X \mid Y) < H(X)
```

## 生活类比

不知道上下文时，很难猜一句话的下一个字。  
如果知道前面已经出现“信息”，你更容易猜后面是“论”。  
这就是条件信息减少了不确定性。

## 重要结论

```math
H(X \mid Y) \le H(X)
```

## 易错点

- 不是所有知道的信息都能大幅降低不确定性，关键看 `Y` 和 `X` 是否相关。

## 注意点

- 后面求互信息时，条件熵是最核心的组成部分。
