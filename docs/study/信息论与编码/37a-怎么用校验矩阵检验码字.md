# 怎么用校验矩阵检验码字

## 这类题在考什么

这类题通常考：

```text
给你一个向量 r
问它是不是合法码字
```

本质上是做一次校验：

```text
它有没有满足全部监督关系
```

## 先记公式

```math
rH^T=S
```

对于合法码字，应满足：

```math
rH^T=0
```

参数解释：

- $r$：待检验的向量或收到的码字。
- $H$：校验矩阵。
- $H^T$：校验矩阵的转置。
- $S$：伴随式。

## 做题步骤

### 第一步：先把收到的向量写成行向量

比如题目给：

```math
r=[1\ 0\ 1\ 1]
```

那就按行向量去乘。

### 第二步：把校验矩阵转置

公式用的是：

```math
rH^T
```

不是直接乱写成 `rH`。

### 第三步：按模 2 运算

所有加法乘法都在模 2 下做。  
最容易错的就是这一步。

### 第四步：看结果是不是全零

- 若 $S=0$，说明它满足校验关系，可能是合法码字
- 若 $S\ne 0$，说明它不是合法码字，或收码中有错

## 最简单例子

如果算得：

```math
rH^T=[0\ 0\ 0]
```

那就可以写：

```text
该向量通过校验，属于合法码字
```

如果算得：

```math
rH^T=[1\ 0\ 1]
```

那就说明：

```text
它没有通过校验
```

## 易错点

- 别忘了转置。
- 别忘了模 2。
- `rH^T=0` 说明“通过校验”，不代表现实里绝对无错。

## 注意点

- 这题和“怎么求伴随式”几乎是一类题。
- 如果题目继续追问错在第几位，就要把伴随式和错误图样对应起来。
