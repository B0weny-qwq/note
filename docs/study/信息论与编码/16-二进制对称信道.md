# 二进制对称信道

## 定义

二进制对称信道 `BSC` 是最经典的离散信道模型：

- 输入只有 `0` 和 `1`
- 输出也只有 `0` 和 `1`
- `0` 变错成 `1` 的概率，和 `1` 变错成 `0` 的概率相同

这个错误概率通常记作 `p`。

## 为什么叫“对称”

因为两个符号被错误翻转的待遇完全一样，没有谁更容易出错。

## 详细例子

假设：

```text
发送 0 -> 以 0.9 概率收到 0，以 0.1 概率收到 1
发送 1 -> 以 0.9 概率收到 1，以 0.1 概率收到 0
```

这就是误码率 `p=0.1` 的 BSC。

## 容量直觉

- `p=0` 时完全不出错，容量最大
- `p=0.5` 时输出像随机猜，容量降到 0

## 生活类比

把 BSC 想成一个经常把“是/否”听错的对讲机：

- 有时“是”被听成“否”
- 有时“否”被听成“是”
- 两种错听概率一样

## 易错点

- BSC 不是“只会出错 0 或 1 次”，而是每个比特独立地以概率 `p` 出错。

## 注意点

- BSC 是后面讲强对称信道、离散无记忆信道的基础模板。
