# 重要公式速查

这页只放考试中最常用、最值得单独背的公式。  
每个公式按四件事记：

```text
它定义什么 -> 公式怎么写 -> 表示什么意义 -> 做题什么时候用
```

## 第 2 章：信源与信息熵

### 自信息量

定义：某个事件 $x_i$ 发生后带来的信息量。

公式：

```math
I(x_i)=-\log_b p(x_i)
```

常用二进制单位时：

```math
I(x_i)=-\log_2 p(x_i)
```

意义：

- 概率越小，事件越意外，信息量越大。
- 概率越大，事件越常见，信息量越小。

怎么用：

```text
题目问“某个具体事件的信息量是多少”时，用自信息量。
```

### 离散信源熵

定义：离散信源平均每发出一个符号带来的平均信息量。

公式：

```math
H(X)=-\sum_{i=1}^{n}p(x_i)\log_b p(x_i)
```

二进制单位时：

```math
H(X)=-\sum_{i=1}^{n}p(x_i)\log_2 p(x_i)
```

意义：

- 熵是平均不确定性。
- 熵也是无失真压缩平均码长的理论下限。

怎么用：

```text
题目给一组符号概率，让你求平均信息量或信源熵时，用它。
```

### 等概离散信源最大熵

定义：当 $n$ 个符号等概率时，离散信源熵达到最大。

公式：

```math
H_{\max}(X)=\log_b n
```

二进制单位时：

```math
H_{\max}(X)=\log_2 n
```

意义：

- 越均匀，越难预测，熵越大。
- 完全等概时不确定性最大。

怎么用：

```text
题目问最大熵、冗余度或均匀分布时，经常用它。
```

### 条件熵

定义：已知 $Y$ 后，$X$ 仍然剩下的平均不确定性。

公式：

```math
H(X|Y)=-\sum_i\sum_j p(x_i,y_j)\log_b p(x_i|y_j)
```

也可以写成：

```math
H(X|Y)=H(X,Y)-H(Y)
```

意义：

- 条件熵表示“知道一部分信息以后还剩多少不确定”。
- 如果 $Y$ 完全决定 $X$，则 $H(X|Y)=0$。

怎么用：

```text
互信息题、联合分布表题、条件概率题里经常用。
```

### 联合熵

定义：两个随机变量合在一起的不确定性。

公式：

```math
H(X,Y)=-\sum_i\sum_j p(x_i,y_j)\log_b p(x_i,y_j)
```

链式关系：

```math
H(X,Y)=H(X)+H(Y|X)
```

```math
H(X,Y)=H(Y)+H(X|Y)
```

意义：

- 联合熵不是简单相加，除非两个变量独立。
- 它描述一对随机变量整体平均需要多少信息来确定。

怎么用：

```text
题目给联合概率表，让你求 H(X,Y) 或互信息时用。
```

### 互信息

定义：两个随机变量之间共享的信息量。

常用公式：

```math
I(X;Y)=H(X)-H(X|Y)
```

```math
I(X;Y)=H(Y)-H(Y|X)
```

```math
I(X;Y)=H(X)+H(Y)-H(X,Y)
```

原始定义：

```math
I(X;Y)=\sum_i\sum_j p(x_i,y_j)\log_b\frac{p(x_i,y_j)}{p(x_i)p(y_j)}
```

意义：

- 互信息表示知道 $Y$ 后，$X$ 的不确定性平均减少多少。
- $I(X;Y)=I(Y;X)$，互信息是对称的。

怎么用：

```text
题目问“两个变量相关多少”“知道输出后获得多少输入信息”时，用互信息。
```

### 熵之间的常用关系

公式：

```math
H(X,Y)=H(X)+H(Y|X)=H(Y)+H(X|Y)
```

```math
I(X;Y)=H(X)+H(Y)-H(X,Y)
```

```math
H(X|Y)\le H(X)
```

意义：

- 条件不会增加不确定性。
- 联合熵、条件熵、互信息经常互相转化。

怎么用：

```text
题目给的量不直接匹配目标公式时，用这些关系换元。
```

### 离散无记忆序列信源熵

定义：无记忆信源连续发出 $n$ 个符号时，整个序列的熵。

公式：

```math
H(X^n)=nH(X)
```

意义：

- 无记忆表示每个符号独立同分布。
- 序列总熵等于单符号熵乘以符号数。

怎么用：

```text
题目说“无记忆信源”“长度为 n 的序列”时，优先用这个。
```

### 熵率

定义：长序列平均每个符号的信息量。

公式：

```math
H_\infty=\lim_{n\to\infty}\frac{1}{n}H(X_1,X_2,\ldots,X_n)
```

意义：

- 熵率适合描述有记忆信源。
- 有记忆时，总熵通常不能简单写成 $nH(X)$。

怎么用：

```text
题目出现马尔可夫信源、有记忆序列信源时，用熵率思想。
```

### 连续信源微分熵

定义：连续随机变量的不确定性度量。

公式：

```math
h(X)=-\int f(x)\log_b f(x)\,dx
```

意义：

- 微分熵是连续版熵，但它可以为负。
- 连续情形下，互信息通常比单独的微分熵更有直接工程意义。

怎么用：

```text
题目给概率密度函数 f(x)，让你求连续信源熵时使用。
```

### 连续互信息

定义：连续变量 $X$ 和 $Y$ 之间共享的信息量。

公式：

```math
I(X;Y)=h(X)-h(X|Y)
```

也可写成：

```math
I(X;Y)=h(X)+h(Y)-h(X,Y)
```

意义：

- 虽然微分熵本身可能为负，但互信息仍然非负。
- 连续信道、高斯信道里常用互信息分析容量。

怎么用：

```text
题目问连续变量间信息关系时，用连续互信息。
```

### 信源冗余度

定义：信源实际熵与最大可能熵之间的差距。

绝对冗余度：

```math
R=H_{\max}(X)-H(X)
```

相对冗余度：

```math
\gamma=1-\frac{H(X)}{H_{\max}(X)}
```

意义：

- 冗余度越大，说明可压缩空间越大。
- 真实文本、语音、图像通常都有冗余。

怎么用：

```text
题目给最大熵和实际熵，让你算冗余度时用。
```

## 第 3 章：信道与信道容量

### 信道转移概率

定义：发送某个输入符号后，接收成某个输出符号的概率。

公式：

```math
p(y_j|x_i)
```

信道矩阵：

```math
P(Y|X)=
\begin{bmatrix}
p(y_1|x_1) & p(y_2|x_1) & \cdots\\
p(y_1|x_2) & p(y_2|x_2) & \cdots\\
\vdots & \vdots & \ddots
\end{bmatrix}
```

意义：

- 每一行表示固定输入下输出的概率分布。
- 每一行概率和应为 1。

怎么用：

```text
题目给转移概率表时，先把它看成 P(Y|X)。
```

### 信道疑义度

定义：收到输出 $Y$ 后，对输入 $X$ 仍然存在的不确定性。

公式：

```math
H(X|Y)
```

意义：

- 疑义度越小，输出越能说明输入。
- 无噪声时通常疑义度较小甚至为 0。

怎么用：

```text
题目问接收端对发送端还剩多少不确定时，用 H(X|Y)。
```

### 噪声熵

定义：已知输入 $X$ 后，输出 $Y$ 仍然存在的不确定性。

公式：

```math
H(Y|X)
```

意义：

- 噪声熵反映信道随机扰动程度。
- 无噪声信道中，$H(Y|X)=0$。

怎么用：

```text
题目问信道本身引入多少随机性时，用 H(Y|X)。
```

### 信道互信息

定义：输入 $X$ 通过信道后，被输出 $Y$ 保留下来的信息量。

公式：

```math
I(X;Y)=H(X)-H(X|Y)
```

也可写成：

```math
I(X;Y)=H(Y)-H(Y|X)
```

意义：

- 互信息越大，信道传递的有效信息越多。
- 信道容量就是互信息在输入分布上的最大值。

怎么用：

```text
题目给输入分布和信道矩阵，让你算“本次输入分布下传了多少信息”时用。
```

### 信道容量定义

定义：给定信道中，单位信道使用能可靠传输的最大平均信息量。

公式：

```math
C=\max_{p(x)} I(X;Y)
```

意义：

- 容量不是随便一个互信息，而是所有输入分布中最大的互信息。
- 它是可靠传输的理论上限。

怎么用：

```text
题目问“信道最多能传多少”“可靠传输上限”时，用容量。
```

### 离散无记忆信道序列容量

定义：无记忆信道连续使用 $n$ 次时的总容量。

公式：

```math
C_n=nC
```

意义：

- 无记忆信道每次使用相互独立。
- 总容量等于单次容量乘以使用次数。

怎么用：

```text
题目说 DMC 连续使用 n 次时，可以用这个思路。
```

### 二进制对称信道容量

定义：交叉概率为 $p$ 的 BSC 的容量。

公式：

```math
C=1-H(p)
```

其中：

```math
H(p)=-p\log_2p-(1-p)\log_2(1-p)
```

意义：

- $p=0$ 时完全无错，$C=1$。
- $p=0.5$ 时输出像随机猜，$C=0$。

怎么用：

```text
题目出现 BSC、交叉概率 p、二进制对称信道容量时，直接用。
```

### 强对称信道容量

定义：行互为排列、列和相等的强对称离散信道容量。

公式：

```math
C=\log_2 m-H(\text{任一行})
```

其中 $m$ 是输出符号个数。

意义：

- 强对称信道中，等概输入通常达到容量。
- 容量可以由输出最大不确定性减去一行噪声不确定性得到。

怎么用：

```text
题目让判断强对称信道并求容量时，用这个。
```

### 高斯白噪声信道容量

定义：带宽受限、加性高斯白噪声信道的容量。

公式：

```math
C=B\log_2\left(1+\frac{S}{N}\right)
```

也常写成：

```math
C=B\log_2(1+\mathrm{SNR})
```

意义：

- 带宽越大，容量越大。
- 信噪比越大，容量越大。
- 功率提升带来的是对数增长，不是线性增长。

怎么用：

```text
题目给带宽 B 和信噪比 S/N，让你求最大可靠传输速率时用。
```

### 信源与信道匹配

定义：判断信源产生的信息率是否能被信道可靠承载。

无失真匹配：

```math
R_s\le C
```

有失真匹配：

```math
R(D)\le C
```

意义：

- 信源信息率不能超过信道容量。
- 允许失真时，比较的是率失真函数和信道容量。

怎么用：

```text
题目问“能不能通过该信道可靠传输”时，比较信息率和容量。
```

## 第 4 章：信息率失真函数

### 失真函数

定义：原始符号 $x$ 被重构成 $y$ 时的损失代价。

公式：

```math
d(x,y)
```

意义：

- 失真函数是衡量“还原得差多少”的尺子。
- 不同业务场景要选不同失真函数。

怎么用：

```text
题目问“用什么衡量失真”或给原符号和重构符号时，用失真函数。
```

### 汉明失真

定义：只关心符号是否相同的失真。

公式：

```math
d(x,y)=
\begin{cases}
0,&x=y\\
1,&x\ne y
\end{cases}
```

意义：

- 对了代价 0。
- 错了代价 1。

怎么用：

```text
离散符号、字符、二元信源判断对错时常用。
```

### 平方误差失真

定义：用数值差的平方衡量失真。

公式：

```math
d(x,y)=(x-y)^2
```

意义：

- 差得越远，惩罚越大。
- 适合连续量，例如温度、电压、图像灰度。

怎么用：

```text
连续数值重构题常用。
```

### 平均失真

定义：长期平均每次重构造成的失真。

公式：

```math
\bar D=\sum_i\sum_j p(x_i)p(y_j|x_i)d(x_i,y_j)
```

也可写成：

```math
\bar D=\sum_i\sum_j p(x_i,y_j)d(x_i,y_j)
```

意义：

- 平均失真必须结合概率和失真矩阵一起算。
- 不能只看失真矩阵本身。

怎么用：

```text
题目给信源概率、重构概率、失真矩阵时，用它算平均损失。
```

### 率失真函数

定义：平均失真不超过 $D$ 时，最少需要的码率。

公式：

```math
R(D)=\min_{p(y|x):\bar D\le D} I(X;Y)
```

意义：

- $R(D)$ 是满足失真约束下的最小互信息。
- 它给有损压缩的理论下限。

怎么用：

```text
题目问“允许失真 D 时最低码率是多少”时，用率失真函数。
```

### 率失真函数性质

非负性：

```math
R(D)\ge 0
```

单调不增：

```math
D_1<D_2 \Rightarrow R(D_1)\ge R(D_2)
```

下凸性：

```math
R(\lambda D_1+(1-\lambda)D_2)\le \lambda R(D_1)+(1-\lambda)R(D_2)
```

意义：

- 允许失真越多，最低码率不会更高。
- 失真足够宽松时，$R(D)$ 可以降到 0。

怎么用：

```text
判断题、选择题问 R(D) 曲线性质时，优先用这些。
```

### 等概二元信源的率失真函数

定义：等概二元信源在汉明失真下的 $R(D)$。

公式：

```math
R(D)=1-H(D),\quad 0\le D\le \frac12
```

当：

```math
D\ge\frac12
```

有：

```math
R(D)=0
```

意义：

- 不允许失真时，$R(0)=1$。
- 允许一半错误时，可以不传额外信息。

怎么用：

```text
题目出现等概二元信源、汉明失真、求 R(D) 时直接用。
```

## 第 5 章：信源编码

### 平均码长

定义：每个信源符号平均需要多少码元表示。

公式：

```math
L=\sum_i p_i l_i
```

意义：

- $p_i$ 是符号概率。
- $l_i$ 是符号码长。
- 平均码长是加权平均，不是普通平均。

怎么用：

```text
题目给符号概率和码字长度，让你评价编码时，用它。
```

### 编码效率

定义：编码接近理论下限的程度。

公式：

```math
\eta=\frac{H(X)}{L}
```

意义：

- 效率越接近 1，编码越接近熵极限。
- 一般有 $\eta\le 1$。

怎么用：

```text
题目问编码效率、压缩效果时，用它。
```

### 编码冗余度

定义：编码效率没有达到理论极限的剩余比例。

公式：

```math
\gamma=1-\eta
```

意义：

- 冗余度越小，编码越紧凑。
- 冗余度越大，说明平均码长离熵越远。

怎么用：

```text
题目让由编码效率求冗余度时，用它。
```

### 香农编码码长

定义：香农编码中每个符号应取的码长。

公式：

```math
l_i=\left\lceil -\log_2 p_i \right\rceil
```

意义：

- 概率越大，码长越短。
- 概率越小，码长越长。

怎么用：

```text
做香农编码题第一步求码长时用。
```

### 香农编码累积概率

定义：按概率从大到小排序后，第 $i$ 个符号前面所有概率之和。

公式：

```math
F_i=\sum_{k=1}^{i-1}p_k
```

意义：

- 香农编码用累积概率的二进制展开生成码字。
- 取前 $l_i$ 位作为码字。

怎么用：

```text
做香农编码具体码字时，用累积概率。
```

### 克拉夫特不等式

定义：判断一组码长是否可能构成即时码的必要充分条件。

二元码公式：

```math
\sum_i 2^{-l_i}\le 1
```

$D$ 元码公式：

```math
\sum_i D^{-l_i}\le 1
```

意义：

- 满足不等式，存在对应的即时码。
- 不满足则不可能构造这样的即时码。

怎么用：

```text
题目给一组码长，问能否构成即时码时用。
```

### 霍夫曼编码平均码长

定义：霍夫曼编码完成后评价编码效果的平均码长。

公式：

```math
L=\sum_i p_i l_i
```

意义：

- 霍夫曼编码在单符号离散无记忆信源的前缀码中平均码长最短。
- 具体码字可能不唯一，但平均码长可相同。

怎么用：

```text
构造霍夫曼树后，用它算最终平均码长。
```

### 算术编码区间更新

定义：算术编码每读入一个符号，就在当前区间内缩小到该符号对应子区间。

若当前区间为：

```math
[L,U)
```

宽度为：

```math
W=U-L
```

符号累计概率区间为：

```math
[F_{\text{low}},F_{\text{high}})
```

更新公式：

```math
L'=L+W F_{\text{low}}
```

```math
U'=L+W F_{\text{high}}
```

意义：

- 每读一个符号，区间变窄。
- 最终区间内任意一个数都可以代表整个序列。

怎么用：

```text
算术编码入门题让你逐步缩小区间时用。
```

## 第 6 章：信道编码

### 信息位、监督位和码长

定义：码字由信息位和监督位组成。

公式：

```math
n=k+r
```

意义：

- $n$ 是码长。
- $k$ 是信息位数。
- $r$ 是监督位数。

怎么用：

```text
线性分组码、汉明码题里先分清 n、k、r。
```

### 编码效率

定义：信道编码中信息位占码字总长度的比例。

公式：

```math
R=\frac{k}{n}
```

意义：

- 码率越高，冗余越少。
- 码率越低，通常保护能力更强，但开销更大。

怎么用：

```text
题目问 (n,k) 码的码率时直接用。
```

### 生成矩阵编码

定义：用生成矩阵把信息组映射成码字。

公式：

```math
c=mG
```

意义：

- $m$ 是信息位向量。
- $G$ 是生成矩阵。
- $c$ 是生成出的码字。
- 二进制线性码中运算对 2 取模。

怎么用：

```text
题目给 m 和 G，让你求码字时用。
```

### 系统码生成矩阵

定义：系统码中，码字直接包含原信息位。

常见形式：

```math
G=[I_k\;P]
```

意义：

- $I_k$ 保留信息位。
- $P$ 生成监督位。

怎么用：

```text
题目问系统线性分组码或由 G 求 H 时常用。
```

### 校验矩阵校验

定义：用校验矩阵判断一个向量是不是合法码字。

公式：

```math
cH^T=0
```

意义：

- 结果为 0，说明满足校验关系。
- 结果非 0，说明不是合法码字或发生了可检测错误。

怎么用：

```text
题目给码字和 H，让你判断是否合法时用。
```

### 系统码校验矩阵

定义：若生成矩阵为 $G=[I_k\;P]$，对应校验矩阵可写为：

公式：

```math
H=[P^T\;I_r]
```

在二进制域中，减号和加号等价。

意义：

- $G$ 负责生成码字。
- $H$ 负责检查码字。
- 二者满足正交关系。

怎么用：

```text
题目由系统生成矩阵求校验矩阵时用。
```

### 伴随式

定义：接收向量经过校验矩阵得到的校验结果。

公式：

```math
S=rH^T
```

若：

```math
r=c+e
```

则：

```math
S=eH^T
```

意义：

- 伴随式只和错误图样有关。
- 可纠错范围内，可以用伴随式定位错误。

怎么用：

```text
题目让检验收到码字、判断错在哪一位时用。
```

### 汉明距离

定义：两个等长码字中不同位置的个数。

公式：

```math
d(c_i,c_j)=w(c_i-c_j)
```

其中 $w(\cdot)$ 是汉明重量。

意义：

- 距离越大，码字越不容易互相混淆。
- 线性码中，任意两个码字距离等于它们差向量的重量。

怎么用：

```text
题目问两个码字相差几位或求最小码距时用。
```

### 最小码距

定义：码字集合中任意两个不同码字之间的最小汉明距离。

公式：

```math
d_{\min}=\min_{i\ne j}d(c_i,c_j)
```

线性码中也可写成：

```math
d_{\min}=\min_{c\ne0}w(c)
```

意义：

- 最小码距决定检错纠错能力。
- 线性码只需要看非零码字最小重量。

怎么用：

```text
题目让判断检错纠错能力时，先求 d_min。
```

### 检错能力

定义：保证能检测最多多少位错误。

公式：

```math
d_{\min}\ge t+1
```

等价地：

```math
t_{\text{detect}}=d_{\min}-1
```

意义：

- 最小码距为 $d_{\min}$ 的码，可以检测最多 $d_{\min}-1$ 位错误。

怎么用：

```text
题目问最多能检几位错时用。
```

### 纠错能力

定义：保证能纠正最多多少位错误。

公式：

```math
d_{\min}\ge 2t+1
```

等价地：

```math
t_{\text{correct}}=\left\lfloor\frac{d_{\min}-1}{2}\right\rfloor
```

意义：

- 纠错比检错要求更高。
- 因为错误球之间不能重叠。

怎么用：

```text
题目问最多能纠几位错时用。
```

### 汉明码监督位关系

定义：汉明码中监督位数量要足够区分所有单错位置和无错情况。

公式：

```math
2^r\ge n+1
```

若 $n=k+r$，则常写成：

```math
2^r\ge k+r+1
```

意义：

- $2^r$ 个伴随式状态要覆盖 $n$ 个单错位置和 1 个无错状态。
- 这是汉明码能单错纠正的基本条件。

怎么用：

```text
题目给信息位数 k，让你求至少几个监督位时用。
```

### 循环码多项式表示

定义：把长度为 $n$ 的码字写成多项式。

若码字为：

```text
c=(c_{n-1},c_{n-2},\ldots,c_0)
```

则：

```math
c(x)=c_{n-1}x^{n-1}+c_{n-2}x^{n-2}+\cdots+c_1x+c_0
```

意义：

- 循环移位可以用多项式乘 $x$ 后模 $x^n-1$ 表示。
- 循环码适合用多项式运算描述。

怎么用：

```text
题目让把码字写成多项式，或判断循环码时用。
```

### 循环码生成多项式

定义：循环码中用来生成所有码字的多项式。

公式：

```math
c(x)=m(x)g(x)
```

通常要求：

```math
g(x)\mid x^n-1
```

意义：

- $g(x)$ 决定循环码结构。
- 所有合法码字都是 $g(x)$ 的倍式。

怎么用：

```text
题目给生成多项式，让你生成码字或判断是否合法时用。
```

### 信道编码定理

定义：有噪声信道中可靠通信的理论边界。

公式关系：

```math
R<C
```

意义：

- 当传输率低于信道容量时，存在编码使错误概率任意小。
- 当传输率高于信道容量时，不能实现任意可靠传输。

怎么用：

```text
概念题问“有噪声信道为什么仍能可靠通信”时，用它。
```
