# 率失真函数计算

## 这部分在考什么

大纲写了 `R(D)` 计算，但入门考试通常不会要求完整推一般情形。常见题型是：

- 会写定义
- 会判断曲线性质
- 会做二元对称信源加汉明失真的简单结果
- 会说明离散和连续情形的计算对象不同

## 先记定义

率失真函数定义为：

```math
R(D)=\min I(X;Y)
```

约束条件是平均失真不超过给定值：

```math
\bar D \le D
```

意思是：

```text
所有满足失真要求的重构方案里，选互信息最小的那个
```

## 通用计算步骤

### 第一步：确定信源分布

先写清：

```math
p(x_i)
```

也就是每个原始符号出现的概率。

### 第二步：确定失真函数

离散信源通常给失真矩阵：

```math
d(x_i,y_j)
```

连续信源常见平方误差：

```math
d(x,y)=(x-y)^2
```

### 第三步：写平均失真

离散情形：

```math
\bar D=\sum_i\sum_j p(x_i)p(y_j|x_i)d(x_i,y_j)
```

### 第四步：在约束下最小化互信息

```math
I(X;Y)=H(X)-H(X|Y)
```

或：

```math
I(X;Y)=H(Y)-H(Y|X)
```

然后在满足 $\bar D\le D$ 的重构信道中取最小。

## 二元对称信源常用结果

如果信源是等概二元信源：

```math
P(0)=P(1)=\frac12
```

失真采用汉明失真：

```math
d(x,y)=
\begin{cases}
0,&x=y\\
1,&x\ne y
\end{cases}
```

则率失真函数为：

```math
R(D)=1-H(D),\quad 0\le D\le \frac12
```

其中：

```math
H(D)=-D\log_2D-(1-D)\log_2(1-D)
```

当：

```math
D\ge\frac12
```

有：

```math
R(D)=0
```

## 结果怎么判断

检查三个方向：

- $D=0$ 时不允许失真，$R(D)$ 最大
- $D$ 增大时，$R(D)$ 不增
- 失真宽松到一定程度后，$R(D)$ 可以降到 0

## 离散和连续的区别

`离散信源`
: 常用求和、失真矩阵、转移概率 $p(y_j|x_i)$。

`连续信源`
: 常用积分、概率密度、平方误差失真。

连续高斯信源在平方误差下有专门结果，简单题通常更重视性质判断，不会强行推导。

## 易错点

- 不要把 $R(D)$ 当成某一个随便选的互信息。
- 不要忘记平均失真约束。
- 二元结果里的 $H(D)$ 是二元熵函数，不是信源熵随便代。

## 注意点

- 考试遇到 `D 增大，R(D) 怎么变`，直接答单调不增。
- 遇到等概二元信源和汉明失真，优先想到 $R(D)=1-H(D)$。
