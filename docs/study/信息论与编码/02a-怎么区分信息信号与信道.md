# 怎么区分信息信号与信道

## 这类题在考什么

这类题通常考：

```text
信息是什么
信号是什么
信道是什么
三者之间是什么关系
```

这是典型基础概念送分题。

## 先记公式

这一节没有必须代数值的核心公式，  
最稳的定义式理解是：

```text
信息：消息中让人不确定度减少的内容
信号：信息的物理载体
信道：信号传播的通道
```

## 做题步骤

### 第一步：先区分“内容”和“载体”

- 信息是内容
- 信号是承载内容的形式

### 第二步：再区分“载体”和“传播路径”

- 信号是你拿来传的东西
- 信道是它经过的路

### 第三步：答题时尽量连成一句关系链

```text
信息通过信号来表示
信号通过信道来传输
```

### 第四步：不要把三者互相替代着写

这是最常见失分点。

## 最简单例子

比如你打电话说一句：

```text
“明天考试”
```

那么：

- “明天考试”这个内容是信息
- 你的声音波形是信号
- 电话线路或无线链路是信道

## 易错点

- 信息不是电压波形本身。
- 信号不是信道。
- 信道不是“消息内容”。

## 注意点

- 概念题就按“内容 - 载体 - 通道”这条线去记，最稳。
