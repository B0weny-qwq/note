# 怎么理解离散无记忆信道题

## 这类题在考什么

这类题通常考：

```text
什么叫离散无记忆信道
怎么从概率表看出“离散”和“无记忆”
```

最常见的是概念判断题。

## 先记公式

离散无记忆信道最核心的条件可以写成：

```math
P(y_n \mid x_n,x_{n-1},y_{n-1},\ldots)=P(y_n \mid x_n)
```

参数解释：

- $x_n$：当前输入符号。
- $y_n$：当前输出符号。
- $x_{n-1},y_{n-1},\ldots$：过去的输入输出历史。

它的意思是：

```text
当前输出只由当前输入决定
和过去发生过什么无关
```

## 做题步骤

### 第一步：先看符号是不是离散的

如果输入输出是：

- `0/1`
- `A/B/C`

这种有限或可数符号集，  
那就有“离散”的特征。

### 第二步：再看当前输出是否只和当前输入有关

如果题目描述里强调：

```text
每次传输互不影响
上一位错没错不影响下一位
```

那就是无记忆。

### 第三步：把转移概率表读成“当前输入到当前输出”的规则

比如：

```math
P(1|0)=0.1,\quad P(0|0)=0.9
```

这表示发送 `0` 时，  
有 $0.1$ 的概率被错收成 `1`。

### 第四步：会和“有记忆信道”做区分

如果当前误码率会随着前一次状态变化，  
那就不是无记忆信道了。

## 最简单例子

如果一个二元信道满足：

```math
P(0|0)=0.9,\quad P(1|0)=0.1
```

```math
P(0|1)=0.2,\quad P(1|1)=0.8
```

并且题目说明“每次传输彼此独立”，  
那它就是：

```text
离散无记忆信道
```

## 易错点

- “无记忆”说的是信道，不是说输入信源没有记忆。
- 别把信道转移概率和信源先验概率混掉。
- 题目里只要强调“当前只和当前有关”，就要想到无记忆。

## 注意点

- BSC、强对称信道这些都可以看成离散无记忆信道的具体例子。
- 小白先把“离散”和“无记忆”拆开理解，别混成一句空话。
