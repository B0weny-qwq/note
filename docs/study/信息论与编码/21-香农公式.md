# 香农公式

## 公式

对限时、限频、限功率的加性高斯白噪声信道：

```math
C = B \log_2 \left(1 + \frac{S}{N}\right)
```

其中：

- $C$：信道容量
- $B$：带宽
- $S$：信号平均功率
- $N$：噪声平均功率
- $S/N$：信噪比

参数解释：

- $C$：单位时间内可可靠传输的最大信息率。
- $B$：信道可用带宽。
- $S$：发送信号的平均功率。
- $N$：带宽 $B$ 内的噪声平均功率。
- $S/N$：信噪比，表示有用信号相对噪声的强弱。

课件里还会把噪声功率写成带宽与噪声功率谱的乘积，即：

```math
N = n_0 B
```

其中：

- $n_0$：噪声单边功率谱密度。
- $B$：信道带宽。

## 每个参数怎么理解

`带宽 B`
: 信道可用的频率范围，越宽通常越能承载更多信息。

`信噪比 S/N`
: 有效信号强度相对于噪声强度的比例。

## 详细例子

如果：

- 带宽 $B = 3000\ \text{Hz}$
- 信噪比 $S/N = 1000$

就可以代入公式估算理论最大可传信息率。

这里的意义不是“你实际一定达到”，而是“理论上不可能长期可靠超过它”。

## 生活类比

一条公路：

- 车道数像带宽
- 路况清晰程度像信噪比

车道更多、干扰更少，单位时间通过能力就更高。

## 易错点

- 提高功率不代表容量无限增长，因为是对数增长。
- 带宽变宽也不是白送，工程上要付成本。

## 注意点

- 香农公式是通信系统上限意识的核心公式之一。
