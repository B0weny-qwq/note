# 生成多项式

## 定义

在循环码里，码字常用多项式表示。  
生成多项式是构造循环码的核心工具。

## 它有什么用

通过生成多项式，可以：

- 构造合法码字
- 推导系统循环码
- 设计编码电路

## 详细例子

把二进制串看成多项式，例如：

```math
1011 \rightarrow x^3 + x + 1
```

然后按多项式规则做模 2 运算，就能构造循环码。

## 生活类比

像做饼干用模具：

- 模具形状决定每次压出来的形状
- 生成多项式就像这套模具规则

## 易错点

- 多项式加法在这里是模 2，不是普通整数加法。

## 注意点

- 一旦看到循环码题，最好先切换到多项式视角。
