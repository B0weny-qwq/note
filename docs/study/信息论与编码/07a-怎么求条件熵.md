# 怎么求条件熵

## 这类题在考什么

条件熵题考的是：

```text
已知 Y 以后，X 还剩多少不确定性
```

也就是“知道一点信息后，剩下还要猜多少”。

## 先记公式

最常用的是：

```math
H(X \mid Y)=H(X,Y)-H(Y)
```

如果题目给了条件概率和联合概率，也可以用：

```math
H(X \mid Y)=-\sum_{j=1}^{m}\sum_{i=1}^{n} p(x_i,y_j)\log_2 p(x_i \mid y_j)
```

## 做题步骤

### 第一步：先看题目给了什么

常见有两种：

- 给熵：`H(X,Y)`、`H(Y)` 之类
- 给联合概率表

### 第二步：优先选最省事的公式

如果题目已经给了：

```math
H(X,Y),\ H(Y)
```

那直接：

```math
H(X \mid Y)=H(X,Y)-H(Y)
```

这是最稳的。

### 第三步：如果只给概率表，就先求联合熵或边缘熵

先从表里求出：

- $H(X,Y)$
- $H(Y)$

再代公式。

## 最简单例子

如果题目给：

```math
H(X,Y)=2.0,\quad H(Y)=0.8
```

那直接有：

```math
H(X \mid Y)=2.0-0.8=1.2
```

## 怎么判断结果合理

记一个最重要的判断：

```math
H(X \mid Y)\le H(X)
```

也就是知道更多信息后，不确定性不会变得更大。

如果你算出来条件熵比原熵还大，通常就是错了。

## 易错点

- 条件熵不是互信息，别和 `I(X;Y)` 混掉。
- 别把 `H(X|Y)=H(X)-H(Y)` 乱写出来，这一般不成立。
- 如果题目用的是对数底数 2，结果单位是 bit。

## 注意点

- 简单题里最推荐的路线是：`先求联合熵，再减边缘熵`。
- 一看到“已知 Y 后 X 还剩多少不确定性”，优先想到条件熵。
