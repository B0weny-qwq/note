# 信道分类与数学模型

## 这部分在考什么

信道题不只考“什么是信道”，还会考：

```text
信道怎么分类
信道用什么数学模型描述
看到转移概率表要怎么读
```

## 信道分类

### 按输入输出取值分类

`离散信道`
: 输入和输出都是离散符号，例如 0、1、A、B。

`连续信道`
: 输入或输出是连续取值，例如电压、幅度、相位。

### 按有无记忆分类

`无记忆信道`
: 当前输出只和当前输入有关，和过去输入输出无关。

`有记忆信道`
: 当前输出还会受到过去输入或过去状态影响。

### 按噪声情况分类

`无噪声信道`
: 输出能完全反映输入。

`有噪声信道`
: 输出可能被扰动、翻转、失真或叠加噪声。

### 按时间特性分类

`恒参信道`
: 统计规律不随时间明显变化。

`变参信道`
: 信道特性会随时间变化，例如移动无线信道。

## 离散单符号信道模型

离散信道通常用转移概率描述：

```math
p(y_j \mid x_i)
```

意思是：

```text
发送 x_i 时，接收成 y_j 的概率
```

所有转移概率可以排成矩阵：

```math
P(Y|X)=
\begin{bmatrix}
p(y_1|x_1) & p(y_2|x_1) & \cdots\\
p(y_1|x_2) & p(y_2|x_2) & \cdots\\
\vdots & \vdots & \ddots
\end{bmatrix}
```

每一行对应一个输入符号，所以每一行概率和为 1。

## 离散序列信道模型

如果输入是序列：

```text
X^n=(X_1,X_2,\ldots,X_n)
```

输出是：

```text
Y^n=(Y_1,Y_2,\ldots,Y_n)
```

一般信道可写为：

```math
p(y^n \mid x^n)
```

如果是离散无记忆信道，则可以拆成：

```math
p(y^n \mid x^n)=\prod_{k=1}^{n}p(y_k \mid x_k)
```

这表示每一拍的传输互不影响。

## 连续信道模型

连续信道常用条件概率密度描述：

```math
f(y \mid x)
```

最常见模型是加性噪声信道：

```math
Y=X+N
```

如果噪声 $N$ 是高斯白噪声，就得到高斯白噪声信道。

## 做题时怎么读

看到信道题，先问四个问题：

1. 输入输出是离散还是连续？
2. 当前输出是否只和当前输入有关？
3. 题目给的是转移概率矩阵、条件概率密度，还是信道框图？
4. 题目要求判断类型、求互信息，还是求容量？

## 易错点

- 信道转移概率不是信源先验概率。
- 转移矩阵每一行通常对应“固定输入时输出怎么分布”。
- 无记忆不是“没有噪声”，而是“不受过去影响”。

## 注意点

- 第 3 章容量题，本质上都离不开信道数学模型。
- BSC、强对称信道、准对称信道、高斯信道都是具体模型。
