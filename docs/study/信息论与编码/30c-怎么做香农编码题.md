# 怎么做香农编码题

## 这类题在考什么

这类题通常考：

```text
已知一组符号概率
用香农编码构造码字
```

有时还会顺带问：

- 平均码长
- 编码效率

## 先记公式

香农编码最关键的两步公式是：

码长满足：

```math
-\log_2 p(x_i) \le K_i < -\log_2 p(x_i)+1
```

参数解释：

- $p(x_i)$：第 $i$ 个符号的概率。
- $K_i$：第 $i$ 个符号的码长。

再定义累积概率：

```math
F_i=\sum_{j=1}^{i-1}p(x_j)
```

参数解释：

- $F_i$：第 $i$ 个符号前面所有符号概率之和。

## 做题步骤

### 第一步：按概率从大到小排序

这一步几乎是所有信源编码题的起手式。  
不先排序，后面很容易乱。

### 第二步：先求每个符号的码长

用概率去决定码长。  
小白做题时，通常直接取：

```math
K_i=\lceil -\log_2 p(x_i)\rceil
```

参数解释：

- $\lceil \cdot \rceil$：向上取整。

### 第三步：求每个符号的累积概率

也就是把当前符号前面所有概率先加起来。  
这一项决定它的码字从哪段二进制小数开始截。

### 第四步：把累积概率写成二进制小数

这是香农编码最有辨识度的一步。

### 第五步：按对应码长截取前 $K_i$ 位

截出来的前 $K_i$ 位，  
就是该符号的香农码字。

## 最简单例子

若概率为：

```math
0.4,\ 0.3,\ 0.2,\ 0.1
```

先排序后求码长：

```math
K_1=2,\quad K_2=2,\quad K_3=3,\quad K_4=4
```

再求累积概率：

```math
F_1=0,\quad F_2=0.4,\quad F_3=0.7,\quad F_4=0.9
```

把这些累积概率写成二进制小数后，  
按各自码长截取，可得到一组香农码：

```text
x1 -> 00
x2 -> 01
x3 -> 101
x4 -> 1110
```

## 易错点

- 香农编码不是只算码长，后面还要用累积概率写具体码字。
- 累积概率写对了，截位数截错，整题也会错。
- 概率一定先排序，再统一往下做。

## 注意点

- 香农编码题最容易失分的地方不是公式，而是二进制小数截取。
