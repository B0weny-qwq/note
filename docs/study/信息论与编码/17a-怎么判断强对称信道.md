# 怎么判断强对称信道

## 这类题在考什么

这类题通常考：

```text
给你一个转移概率矩阵
问它是不是强对称信道
```

重点不在计算，  
而在看矩阵结构是不是足够规整。

## 先记公式

强对称信道最核心的结构特征可以记成两句：

```text
每一行互为排列
每一列也互为排列
```

也就是：

- 每个输入符号看到的输出概率结构一样
- 每个输出符号对应的输入概率结构也一样

## 做题步骤

### 第一步：先看每一行是不是同一组数的重排

如果第一行是：

```text
0.7, 0.2, 0.1
```

那别的行也应该只是这三个数换位置。

### 第二步：再看每一列是不是同一组数的重排

这一步不能省。  
只看行对称还不够，列也要满足类似结构。

### 第三步：只要有一行或一列破坏规则，就不是强对称

强对称的要求是很严格的，  
不是“看着差不多”就算。

### 第四步：如果它是二元完全对称结构，优先联想到 BSC

BSC 就是最简单的强对称信道。

## 最简单例子

如果矩阵是：

```math
\begin{bmatrix}
0.8 & 0.2\\
0.2 & 0.8
\end{bmatrix}
```

那：

- 两行互为排列
- 两列也互为排列

所以它是：

```text
强对称信道
```

## 易错点

- 只看行，不看列，容易误判。
- “数值接近”不等于“互为排列”。
- 强对称是严格结构概念，不是感觉题。

## 注意点

- 这类题很多是选择题、判断题，先抓结构，不要急着算容量。
- 如果题目后面继续问容量，常常可以利用等概输入这个结论。
