# 怎么把码字写成多项式

## 这类题在考什么

这类题通常考：

```text
把一个二进制串改写成多项式
或者反过来把多项式写回二进制串
```

这是循环码和生成多项式题的最基础入口。

## 先记公式

一个二进制串：

```math
b_{n-1}b_{n-2}\cdots b_1b_0
```

对应的多项式写成：

```math
b_{n-1}x^{n-1}+b_{n-2}x^{n-2}+\cdots+b_1x+b_0
```

参数解释：

- $b_i$：第 $i$ 位二进制系数，只能取 0 或 1。
- $x^i$：对应第 $i$ 次幂项。

## 做题步骤

### 第一步：先看最高位对应哪个幂次

最左边那一位对应最高次幂，  
最右边对应常数项。

### 第二步：遇到 1 就写出该项，遇到 0 就跳过

因为系数为 0 的项不用写。

### 第三步：按从高次到低次写完整

不要漏项，也不要把次序写反。

### 第四步：如果要反写回二进制串，就把缺的项补 0

这一步在反向题里特别重要。

## 最简单例子

如果码字是：

```math
1011
```

那从左到右对应：

- 第 1 位：$x^3$
- 第 2 位：$0x^2$
- 第 3 位：$x$
- 第 4 位：$1$

所以多项式是：

```math
x^3+x+1
```

反过来，如果题目给：

```math
x^4+x^2+1
```

那二进制串应写成：

```math
10101
```

## 易错点

- 左边是高次项，右边是低次项，别反过来。
- 缺少的幂次要补 0，不是直接略过去。
- 这里的系数是 0 或 1，不是普通十进制系数。

## 注意点

- 这是做循环码题前的基础动作，必须熟。
- 一旦切到多项式视角，后面的除法、生成多项式、校验多项式都会顺很多。
