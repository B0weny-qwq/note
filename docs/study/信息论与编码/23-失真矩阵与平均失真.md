# 失真矩阵与平均失真

## 失真矩阵是什么

对于离散信源，可以把每一对 `(原符号, 重构符号)` 的失真写成一个矩阵。

比如：

```math
d(x_i, y_j)
```

每个元素表示原来是 $x_i$、重构成 $y_j$ 时的代价。

参数解释：

- $x_i$：离散信源的第 $i$ 个输入符号。
- $y_j$：编码或重构后的第 $j$ 个输出符号。
- $d(x_i,y_j)$：输入 $x_i$ 被重构为 $y_j$ 时的单次失真。

## 平均失真是什么

平均失真就是长期平均下来，每次重构大概损失了多少。

可以理解为：

```math
D = E[d(X, Y)]
```

对离散信源，课件里的展开式是：

```math
D = \sum_{i=1}^{n}\sum_{j=1}^{m} p(x_i, y_j)\, d(x_i, y_j)
```

也可以写成：

```math
D = \sum_{i=1}^{n}\sum_{j=1}^{m} p(x_i)\, p(y_j \mid x_i)\, d(x_i, y_j)
```

参数解释：

- $D$：平均失真。
- $X, Y$：输入与输出随机变量。
- $p(x_i,y_j)$：联合概率。
- $p(y_j \mid x_i)$：编码信道从 $x_i$ 到 $y_j$ 的转移概率。

## 详细例子

假设源符号只有 `0` 和 `1`，重构符号也只有 `0` 和 `1`，汉明失真矩阵就是：

```text
0 1
1 0
```

含义：

- 原来是 0，重构成 0，失真 0
- 原来是 0，重构成 1，失真 1
- 原来是 1，重构成 0，失真 1
- 原来是 1，重构成 1，失真 0

如果系统中错一半、对一半，那么平均失真就是 0.5。

## 生活类比

老师批改选择题：

- 答对得 0 罚分
- 答错得 1 罚分

全班平均罚分，就相当于平均失真。

## 易错点

- 失真矩阵不是转移概率矩阵，它表示的是“代价”，不是“发生概率”。

## 注意点

- 平均失真必须结合概率分布一起算，不能只看矩阵值本身。
