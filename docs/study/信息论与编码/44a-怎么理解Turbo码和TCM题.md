# 怎么理解Turbo码和TCM题

## 这类题在考什么

这类题通常考：

```text
Turbo 码是什么
TCM 是什么
它们各自强调哪件事
```

这部分更偏概念理解和选择题。

## 先记公式

这类题一般没有必须现场代入的基础公式，  
最稳的是记两句话：

```text
Turbo 码：强调强纠错和迭代译码
TCM：强调把编码和调制联合设计
```

## 做题步骤

### 第一步：先分清两个对象在解决什么问题

- Turbo 码更关注纠错性能
- TCM 更关注编码和调制的一体化设计

### 第二步：看到“迭代译码”就优先想到 Turbo 码

这是它最有辨识度的关键词。

### 第三步：看到“编码和调制联合”就想到 TCM

不是先编码再随便调制，  
而是一开始就一起设计。

### 第四步：答题时别把它们和汉明码、循环码混成一层

它们是后续更现代的编码思路。

## 最简单例子

如果题目问：

```text
哪种方法通过迭代译码提升性能，并接近香农极限？
```

答案就是：

```text
Turbo 码
```

如果题目问：

```text
哪种方法把纠错编码直接融入调制点设计？
```

答案就是：

```text
TCM
```

## 易错点

- Turbo 码不是卷积码的简单别名。
- TCM 不是“先编码再调制”的普通串联。
- 这部分多半考理解，不是考你现场推导复杂算法。

## 注意点

- 课程复习里，这一节通常属于拓展了解题。
- 小白先抓关键词：`迭代译码` 对 Turbo，`联合设计` 对 TCM。
