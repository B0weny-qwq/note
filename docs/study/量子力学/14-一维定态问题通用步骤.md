# 一维定态问题通用步骤

## 基本方程

一维定态 Schrödinger 方程为：

```math
-\frac{\hbar^2}{2m}\frac{d^2\psi}{dx^2}
+
U(x)\psi
=
E\psi
```

整理：

```math
\frac{d^2\psi}{dx^2}
+
\frac{2m}{\hbar^2}(E-U)\psi=0
```

## 第一步：按势能分区

如果 $U(x)$ 是分段常数，要先按不同区域写方程。

例如：

```text
I 区: x<0
II 区: 0<x<a
III 区: x>a
```

每个区域分别写通解。

## 第二步：判断解的形式

当：

```math
E>U
```

定义：

```math
k=\frac{\sqrt{2m(E-U)}}{\hbar}
```

方程为振荡解：

```math
\psi=Ae^{ikx}+Be^{-ikx}
```

当：

```math
E<U
```

定义：

```math
\kappa=\frac{\sqrt{2m(U-E)}}{\hbar}
```

方程为指数解：

```math
\psi=Ce^{\kappa x}+De^{-\kappa x}
```

## 第三步：列边界条件

常用物理条件：

1. 波函数单值。
2. 波函数有限。
3. 波函数连续。
4. 势能有限突变处，导数连续。

有限势垒边界：

```math
\psi_1(x_0)=\psi_2(x_0)
```

```math
\psi_1'(x_0)=\psi_2'(x_0)
```

如果是无限高势垒，墙外波函数为 0，墙处通常要求波函数为 0，但导数不一定连续。

## 第四步：求本征值

边界条件会给出一组线性方程。要有非零解，系数行列式必须为 0：

```math
\det M(E)=0
```

这一步会给出允许能量 $E_n$。

## 第五步：归一化

求出本征函数后，利用：

```math
\int|\psi_n(x)|^2dx=1
```

确定归一化常数。

## 难点注释

- 微分方程本身通常不难，真正难的是边界条件。
- 束缚态要求无穷远处波函数不发散。
- 散射态通常不能归一化为 1，而要用概率流定义反射、透射。

## 易错点

- 势能有限突变处导数连续；无限突变处不能照搬。
- 禁阻区不是波函数为 0，而是指数衰减。
- 先求能量本征值，再写对应本征函数，最后归一化。
