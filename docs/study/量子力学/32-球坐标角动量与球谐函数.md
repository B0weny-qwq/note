# 球坐标角动量与球谐函数

## Lz 算符

在球坐标中：

```math
\hat L_z
=
-i\hbar\frac{\partial}{\partial\phi}
```

设：

```math
\hat L_zY(\theta,\phi)=m\hbar Y(\theta,\phi)
```

则：

```math
-i\hbar\frac{\partial Y}{\partial\phi}
=
m\hbar Y
```

整理：

```math
\frac{\partial Y}{\partial\phi}=imY
```

所以 $\phi$ 方向解为：

```math
Y\propto e^{im\phi}
```

单值性要求：

```math
Y(\theta,\phi+2\pi)=Y(\theta,\phi)
```

因此：

```math
m=0,\pm1,\pm2,\cdots
```

## L² 与 Lz 的共同本征函数

球谐函数 $Y_l^m(\theta,\phi)$ 满足：

```math
\hat L^2Y_l^m
=
\hbar^2l(l+1)Y_l^m
```

以及：

```math
\hat L_zY_l^m
=
m\hbar Y_l^m
```

量子数取值：

```math
l=0,1,2,\cdots
```

```math
m=-l,-l+1,\cdots,l-1,l
```

## 正交归一

球谐函数满足：

```math
\int
Y_l^{m*}(\theta,\phi)
Y_{l'}^{m'}(\theta,\phi)
d\Omega
=
\delta_{ll'}\delta_{mm'}
```

其中：

```math
d\Omega=\sin\theta\,d\theta\,d\phi
```

## 物理意义

角动量大小：

```math
|\vec L|
=
\sqrt{l(l+1)}\hbar
```

z 方向投影：

```math
L_z=m\hbar
```

对于固定 $l$，共有：

```math
2l+1
```

个不同的 $m$ 状态。

## 难点注释

- $m$ 的整数性来自 $\phi$ 方向的单值性。
- $|m|\le l$ 来自 $\theta$ 方程解的有限性。
- 球谐函数是中心势问题角向部分的标准答案。

## 易错点

- $l$ 不能为负数。
- $m$ 不是任意整数，必须满足 $-l\le m\le l$。
- $L^2$ 的本征值是 $\hbar^2l(l+1)$，不是 $\hbar^2l^2$。
