# 力学量平均值与算符夹心

## 位置相关量的平均值

若力学量只是位置函数 $f(x)$，概率密度为 $|\psi(x)|^2$，则：

```math
\langle f(x)\rangle
=
\int \psi^*(x)f(x)\psi(x)dx
```

也就是：

```math
\langle f(x)\rangle
=
\int f(x)|\psi(x)|^2dx
```

## 一般力学量

对于一般力学量 $F$，用算符 $\hat F$ 表示：

```math
\langle F\rangle
=
\int\psi^*(x)\hat F\psi(x)dx
```

Dirac 记号写成：

```math
\langle F\rangle
=
\langle\psi|\hat F|\psi\rangle
```

这叫算符夹心形式。

## 动量平均值

动量算符为：

```math
\hat p_x=-i\hbar\frac{\partial}{\partial x}
```

所以：

```math
\langle p_x\rangle
=
\int
\psi^*(x)
\left(
-i\hbar\frac{\partial}{\partial x}
\right)
\psi(x)dx
```

注意算符先作用在右侧的 $\psi$ 上。

## 动能平均值

动能算符：

```math
\hat T=\frac{\hat p^2}{2m}
=
-\frac{\hbar^2}{2m}\frac{d^2}{dx^2}
```

因此：

```math
\langle T\rangle
=
\int
\psi^*(x)
\left(
-\frac{\hbar^2}{2m}\frac{d^2}{dx^2}
\right)
\psi(x)dx
```

## 非归一化态

如果 $\psi$ 未归一化，平均值要写成：

```math
\langle F\rangle
=
\frac{\langle\psi|\hat F|\psi\rangle}
{\langle\psi|\psi\rangle}
```

即：

```math
\langle F\rangle
=
\frac{\int\psi^*\hat F\psi dx}
{\int\psi^*\psi dx}
```

## 难点注释

- 算符作用顺序很重要，微分算符通常作用在右侧波函数上。
- 若 $\hat F$ 是厄米算符，$\langle F\rangle$ 必为实数。
- 算符夹心形式统一了波动力学和矩阵力学。

## 易错点

- 不要把动量平均值写成 $\int p|\psi|^2dx$，因为位置表象中 $p$ 不是普通变量。
- 未归一化态一定要除以范数。
- 二阶导数作用对象不要搞错。
