# 概率流密度与连续性方程

## 要证明什么

概率密度为：

```math
\rho=\psi^*\psi
```

概率守恒的局域形式应写成：

```math
\frac{\partial\rho}{\partial t}
+
\nabla\cdot\vec J=0
```

其中 $\vec J$ 是概率流密度。

## 写出 Schrödinger 方程

设势能 $U$ 为实函数：

```math
i\hbar\frac{\partial\psi}{\partial t}
=
-\frac{\hbar^2}{2m}\nabla^2\psi
+
U\psi
```

其复共轭为：

```math
-i\hbar\frac{\partial\psi^*}{\partial t}
=
-\frac{\hbar^2}{2m}\nabla^2\psi^*
+
U\psi^*
```

## 构造相减

第一式左乘 $\psi^*$：

```math
i\hbar\psi^*\frac{\partial\psi}{\partial t}
=
-\frac{\hbar^2}{2m}\psi^*\nabla^2\psi
+
U\psi^*\psi
```

第二式左乘 $\psi$：

```math
-i\hbar\psi\frac{\partial\psi^*}{\partial t}
=
-\frac{\hbar^2}{2m}\psi\nabla^2\psi^*
+
U\psi\psi^*
```

第一式减去第二式，势能项抵消：

```math
i\hbar
\left(
\psi^*\frac{\partial\psi}{\partial t}
+
\psi\frac{\partial\psi^*}{\partial t}
\right)
=
-\frac{\hbar^2}{2m}
\left(
\psi^*\nabla^2\psi
-
\psi\nabla^2\psi^*
\right)
```

左边括号就是：

```math
\frac{\partial}{\partial t}(\psi^*\psi)
=
\frac{\partial\rho}{\partial t}
```

## 整理成散度

利用恒等式：

```math
\nabla\cdot(\psi^*\nabla\psi-\psi\nabla\psi^*)
=
\psi^*\nabla^2\psi-\psi\nabla^2\psi^*
```

可得：

```math
\frac{\partial\rho}{\partial t}
+
\nabla\cdot
\left[
\frac{\hbar}{2mi}
(\psi^*\nabla\psi-\psi\nabla\psi^*)
\right]
=0
```

因此：

```math
\vec J
=
\frac{\hbar}{2mi}
(\psi^*\nabla\psi-\psi\nabla\psi^*)
```

## 难点注释

- 势能项抵消依赖于 $U$ 是实函数。
- 连续性方程表达的是局域守恒：某处概率减少，必须流到别处。
- 对全空间积分且边界流为 0，就得到总概率不随时间变化。

## 易错点

- 不要把概率流写成 $\rho v$，那是经典流体直觉，不是一般量子定义。
- 注意前因子是 $\frac{\hbar}{2mi}$。
- 推导中是“方程与共轭方程相减”，不是相加。
