# 无限深势阱正交归一与展开

## 本征函数

一维无限深势阱 $0<x<L$ 的归一化本征函数为：

```math
\psi_n(x)=\sqrt{\frac{2}{L}}\sin\frac{n\pi x}{L}
```

能量为：

```math
E_n=\frac{n^2\pi^2\hbar^2}{2mL^2}
```

## 正交性

要证明：

```math
\int_0^L\psi_m^*(x)\psi_n(x)dx=\delta_{mn}
```

先看未归一化的积分：

```math
I_{mn}=\int_0^L
\sin\frac{m\pi x}{L}
\sin\frac{n\pi x}{L}dx
```

使用恒等式：

```math
\sin A\sin B
=
\frac12[\cos(A-B)-\cos(A+B)]
```

当 $m\ne n$ 时：

```math
I_{mn}
=
\frac12
\int_0^L
\left[
\cos\frac{(m-n)\pi x}{L}
-
\cos\frac{(m+n)\pi x}{L}
\right]dx
```

积分后两项都在边界给出 $\sin$ 的整数倍 $\pi$，所以为 0：

```math
I_{mn}=0,\quad m\ne n
```

当 $m=n$ 时：

```math
I_{nn}
=
\int_0^L\sin^2\frac{n\pi x}{L}dx
=
\frac{L}{2}
```

乘上归一化系数：

```math
\int_0^L\psi_m^*\psi_n dx
=
\frac{2}{L}I_{mn}
=
\delta_{mn}
```

## 任意初态展开

如果初态 $\Psi(x,0)$ 在势阱内，可以展开为：

```math
\Psi(x,0)=\sum_{n=1}^{\infty}c_n\psi_n(x)
```

展开系数为：

```math
c_n=\int_0^L\psi_n^*(x)\Psi(x,0)dx
```

## 含时演化

每个定态只附加相位：

```math
\Psi(x,t)
=
\sum_{n=1}^{\infty}
c_n\psi_n(x)e^{-iE_nt/\hbar}
```

测量能量得到 $E_n$ 的概率为：

```math
P(E_n)=|c_n|^2
```

## 难点注释

- 正交性来自边界条件和不同波数的完整周期性。
- 展开系数是投影，不是随便配出来的常数。
- 若初态不是能量本征态，概率密度一般随时间变化。

## 易错点

- 不要漏掉归一化系数 $\sqrt{2/L}$。
- $c_n$ 可能是复数。
- $\sum |c_n|^2=1$ 要求初态已归一化。
