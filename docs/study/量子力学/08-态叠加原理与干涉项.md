# 态叠加原理与干涉项

## 态叠加原理

如果 $\psi_1$ 和 $\psi_2$ 是某个量子体系的两个可能状态，那么它们的线性组合：

```math
\psi=c_1\psi_1+c_2\psi_2
```

也是这个体系的可能状态。

这里 $c_1,c_2$ 一般是复数。

## 叠加态的概率密度

概率密度为：

```math
|\psi|^2=\psi^*\psi
```

先写共轭：

```math
\psi^*=c_1^*\psi_1^*+c_2^*\psi_2^*
```

于是：

```math
|\psi|^2
=
(c_1^*\psi_1^*+c_2^*\psi_2^*)
(c_1\psi_1+c_2\psi_2)
```

展开：

```math
|\psi|^2
=
|c_1|^2|\psi_1|^2
+
|c_2|^2|\psi_2|^2
+
c_1^*c_2\psi_1^*\psi_2
+
c_1c_2^*\psi_1\psi_2^*
```

后两项就是干涉项。

## 和经典混合的区别

如果只是经典概率混合，比如“有概率 $|c_1|^2$ 走路径 1，有概率 $|c_2|^2$ 走路径 2”，概率应写成：

```math
P_{\text{mix}}
=
|c_1|^2|\psi_1|^2
+
|c_2|^2|\psi_2|^2
```

它没有干涉项。

叠加态保留相对相位，所以会出现条纹。

## 正交归一时的系数条件

如果 $\psi_1,\psi_2$ 正交归一：

```math
\langle\psi_i|\psi_j\rangle=\delta_{ij}
```

归一化要求：

```math
|c_1|^2+|c_2|^2=1
```

## 难点注释

- “叠加”不是“有时在这个态、有时在那个态”的无知概率。
- 干涉项来自相对相位，是量子叠加最关键的可观测效果。
- 一旦测量路径信息，状态会改变，干涉项可能消失。

## 易错点

- 不要把 $c_i$ 当成概率，$|c_i|^2$ 才对应概率。
- 只有在正交归一基下，才可直接说 $|c_i|^2$ 是测得对应态的概率。
- 复共轭展开时不要漏掉 $c_i^*$。
