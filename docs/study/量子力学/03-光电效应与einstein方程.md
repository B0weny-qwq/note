# 光电效应与 Einstein 方程

## 实验现象

光电效应有几个经典波动理论不好解释的现象：

1. 存在截止频率，低于该频率无论光强多大都没有光电子。
2. 光电子最大动能和光频率成线性关系。
3. 光电流几乎瞬时产生。
4. 光强主要影响光电子数量，不决定单个电子最大动能。

## 光量子假设

Einstein 认为光由一份一份光子组成。频率为 $\nu$ 的光子能量为：

```math
E=h\nu
```

光子动量大小为：

```math
p=\frac{E}{c}=\frac{h}{\lambda}
```

## 能量守恒推导

一个电子吸收一个光子，光子能量先用来克服金属逸出功 $W$，剩余部分变成电子最大初动能：

```math
h\nu=W+K_{\max}
```

因此：

```math
K_{\max}=h\nu-W
```

如果用遏止电压 $U_0$ 刚好阻止最快电子到达阳极，则电场做功等于最大动能：

```math
K_{\max}=eU_0
```

所以：

```math
eU_0=h\nu-W
```

这说明 $U_0$ 与 $\nu$ 是线性关系。

## 截止频率

刚好能逸出时，最大动能为 0：

```math
K_{\max}=0
```

所以：

```math
h\nu_0=W
```

即：

```math
\nu_0=\frac{W}{h}
```

对应红限波长：

```math
\lambda_0=\frac{c}{\nu_0}=\frac{hc}{W}
```

## 光强的作用

光强增大，意味着单位时间、单位面积入射光子数增多。若频率已经超过截止频率，则逸出的电子数增多，光电流增大。

但是单个光子的能量仍然是 $h\nu$，所以最大动能不由光强决定。

## 难点注释

- 光电效应中最关键的量是单个光子的能量，不是光束总能量。
- 截止频率说明电子不能慢慢积累来自经典波的能量。
- 斜率 $h/e$ 可以用于测普朗克常数。

## 易错点

- 不要写成 $h\nu=K_{\max}-W$。
- 截止频率由材料逸出功决定，不是由光强决定。
- 遏止电压越大，说明电子最大动能越大。
