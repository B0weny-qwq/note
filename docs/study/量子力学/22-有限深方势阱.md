# 有限深方势阱

## 模型

取对称有限深势阱：

```math
U(x)=
\begin{cases}
0,& |x|<a\\
U_0,& |x|\ge a
\end{cases}
```

讨论束缚态：

```math
0<E<U_0
```

## 各区解的形式

阱内 $|x|<a$，有：

```math
k=\frac{\sqrt{2mE}}{\hbar}
```

波函数是振荡形式。

阱外 $|x|\ge a$，有：

```math
\kappa=\frac{\sqrt{2m(U_0-E)}}{\hbar}
```

束缚态要求无穷远处衰减，所以阱外是指数衰减。

## 利用奇偶性

势能是偶函数：

```math
U(x)=U(-x)
```

所以 Hamilton 与宇称算符对易，本征函数可选为奇函数或偶函数。

## 偶宇称解

阱内取：

```math
\psi_{\text{in}}=A\cos kx
```

阱外取：

```math
\psi_{\text{out}}=Be^{-\kappa |x|}
```

在 $x=a$ 处连续：

```math
A\cos ka=Be^{-\kappa a}
```

导数连续：

```math
-Ak\sin ka=-B\kappa e^{-\kappa a}
```

两式相除：

```math
k\tan(ka)=\kappa
```

这就是偶态能量本征值方程。

## 奇宇称解

阱内取：

```math
\psi_{\text{in}}=A\sin kx
```

同样匹配可得：

```math
-k\cot(ka)=\kappa
```

这是奇态能量本征值方程。

## 为什么叫超越方程

方程：

```math
k\tan(ka)=\kappa
```

和：

```math
-k\cot(ka)=\kappa
```

中 $k,\kappa$ 都依赖能量 $E$。它们一般不能解析解出，只能画图或数值求根。

## 难点注释

- 有限深势阱墙外波函数不为 0，而是有指数尾巴。
- 奇偶性可以把求解工作减半。
- 束缚态个数有限，取决于势阱宽度和深度。

## 易错点

- 不要把有限深势阱当成无限深势阱，边界处 $\psi$ 不必为 0。
- 势能有限突变处导数连续。
- 偶态和奇态对应不同超越方程。
