# 隧道效应：E 小于 U0

## 物理现象

经典力学中，如果粒子能量小于势垒高度：

```math
E<U_0
```

粒子不能穿过势垒。

量子力学中，势垒内波函数不为 0，只是指数衰减，因此存在非零透射概率，这就是隧道效应。

## 势垒内方程

矩形势垒中，II 区势能为 $U_0$。定态方程：

```math
-\frac{\hbar^2}{2m}\frac{d^2\psi}{dx^2}
+
U_0\psi
=
E\psi
```

整理：

```math
\frac{d^2\psi}{dx^2}
=
\frac{2m(U_0-E)}{\hbar^2}\psi
```

定义：

```math
\kappa=\frac{\sqrt{2m(U_0-E)}}{\hbar}
```

则：

```math
\frac{d^2\psi}{dx^2}=\kappa^2\psi
```

势垒内通解：

```math
\psi_{II}=Ce^{\kappa x}+De^{-\kappa x}
```

## 厚高势垒近似

完整透射系数由边界匹配得到。若势垒足够厚或足够高：

```math
\kappa a\gg 1
```

透射系数主要呈指数衰减：

```math
D\propto e^{-2\kappa a}
```

代入 $\kappa$：

```math
D\propto
\exp\left[
-2a\frac{\sqrt{2m(U_0-E)}}{\hbar}
\right]
```

## 任意势垒的 WKB 近似

对一般势垒 $V(x)$，在禁阻区 $V(x)>E$，透射系数近似为：

```math
D\simeq
\exp\left[
-2\int_a^b
\sqrt{\frac{2m(V(x)-E)}{\hbar^2}}dx
\right]
```

其中 $a,b$ 是经典转折点：

```math
V(a)=E,\quad V(b)=E
```

## 难点注释

- 禁阻区不是波函数为 0，而是波函数指数衰减。
- 指数里含有粒子质量，质量越大越难隧穿。
- 势垒越宽、越高，透射概率越小。

## 应用

- $\alpha$ 衰变。
- 扫描隧道显微镜。
- 场致电子发射。
- 半导体隧穿器件。

## 易错点

- 不要说粒子“借能量越过势垒”，更准确说法是波函数在势垒另一侧仍有非零振幅。
- $D$ 是概率量，指数很小但通常不严格为 0。
- WKB 指数中是 $V(x)-E$，不是 $E-V(x)$。
