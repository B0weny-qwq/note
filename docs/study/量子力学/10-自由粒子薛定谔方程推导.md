# 自由粒子薛定谔方程推导

## 推导入口

自由粒子的 de Broglie 平面波写作：

```math
\psi(x,t)=Ae^{\frac{i}{\hbar}(px-Et)}
```

非相对论自由粒子的能量关系为：

```math
E=\frac{p^2}{2m}
```

目标是找到一个波动方程，让这个平面波满足它。

## 时间导数给出能量

对时间求导：

```math
\frac{\partial\psi}{\partial t}
=
\frac{-iE}{\hbar}\psi
```

两边乘以 $i\hbar$：

```math
i\hbar\frac{\partial\psi}{\partial t}=E\psi
```

这提示能量算符为：

```math
\hat E=i\hbar\frac{\partial}{\partial t}
```

## 空间二阶导数给出动量平方

先求一阶导：

```math
\frac{\partial\psi}{\partial x}
=
\frac{ip}{\hbar}\psi
```

再求二阶导：

```math
\frac{\partial^2\psi}{\partial x^2}
=
-\frac{p^2}{\hbar^2}\psi
```

所以：

```math
-\hbar^2\frac{\partial^2\psi}{\partial x^2}=p^2\psi
```

这提示动量算符为：

```math
\hat p=-i\hbar\frac{\partial}{\partial x}
```

## 代入能量关系

由：

```math
E=\frac{p^2}{2m}
```

作用到波函数上：

```math
E\psi=\frac{p^2}{2m}\psi
```

替换为微分形式：

```math
i\hbar\frac{\partial\psi}{\partial t}
=
-\frac{\hbar^2}{2m}
\frac{\partial^2\psi}{\partial x^2}
```

这就是一维自由粒子的含时 Schrödinger 方程。

## 三维形式

三维中二阶导数换成 Laplace 算子：

```math
i\hbar\frac{\partial\psi}{\partial t}
=
-\frac{\hbar^2}{2m}\nabla^2\psi
```

## 难点注释

- 这不是从 Newton 方程严格推导，而是从平面波、de Broglie 关系和非相对论能量关系构造相容方程。
- 方程必须是线性的，因为量子态满足叠加原理。
- 这里的自由粒子没有势能项。

## 易错点

- 空间部分要用二阶导，因为能量含 $p^2$。
- 时间部分是一阶导，因为能量本身对应 $E$。
- 不要漏掉负号：$\partial_x^2\psi=-(p^2/\hbar^2)\psi$。
