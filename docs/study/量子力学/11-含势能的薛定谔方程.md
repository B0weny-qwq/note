# 含势能的薛定谔方程

## 经典 Hamilton 量

非相对论粒子在势场中的总能量为：

```math
H=\frac{p^2}{2m}+U(\vec r,t)
```

量子力学中，把能量和动量换成算符：

```math
\hat E=i\hbar\frac{\partial}{\partial t}
```

```math
\hat{\vec p}=-i\hbar\nabla
```

## 动能算符

动量平方算符为：

```math
\hat p^2
=
(-i\hbar\nabla)\cdot(-i\hbar\nabla)
=
-\hbar^2\nabla^2
```

所以动能算符为：

```math
\hat T
=
\frac{\hat p^2}{2m}
=
-\frac{\hbar^2}{2m}\nabla^2
```

## 势能算符

如果势能只依赖位置和时间，势能算符通常就是乘法算符：

```math
\hat U\psi=U(\vec r,t)\psi
```

## Hamilton 算符

因此：

```math
\hat H
=
-\frac{\hbar^2}{2m}\nabla^2
+
U(\vec r,t)
```

含时 Schrödinger 方程为：

```math
i\hbar\frac{\partial}{\partial t}\Psi(\vec r,t)
=
\hat H\Psi(\vec r,t)
```

写开：

```math
i\hbar\frac{\partial\Psi}{\partial t}
=
\left[
-\frac{\hbar^2}{2m}\nabla^2
+
U(\vec r,t)
\right]\Psi
```

## 难点注释

- 势能项是乘法，动能项是微分，这是两者在位置表象中的重要区别。
- Hamilton 算符控制态随时间演化。
- 如果 $U$ 不显含时间，可以进一步分离变量得到定态方程。

## 易错点

- 不要把势能项写成单独数字，它要乘在波函数上。
- $\nabla^2$ 是 Laplace 算子，不是梯度。
- 势能若显含时间，一般不能简单使用定态分离。
