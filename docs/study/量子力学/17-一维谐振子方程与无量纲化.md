# 一维谐振子方程与无量纲化

## 势能形式

一维线性谐振子的势能为：

```math
U(x)=\frac12m\omega^2x^2
```

定态 Schrödinger 方程：

```math
-\frac{\hbar^2}{2m}\frac{d^2\psi}{dx^2}
+
\frac12m\omega^2x^2\psi
=
E\psi
```

## 化成标准形式

两边乘以 $2m/\hbar^2$：

```math
-\frac{d^2\psi}{dx^2}
+
\frac{m^2\omega^2}{\hbar^2}x^2\psi
=
\frac{2mE}{\hbar^2}\psi
```

整理：

```math
\frac{d^2\psi}{dx^2}
+
\left(
\frac{2mE}{\hbar^2}
-
\frac{m^2\omega^2}{\hbar^2}x^2
\right)\psi=0
```

引入无量纲变量：

```math
\xi=\sqrt{\frac{m\omega}{\hbar}}x
```

则：

```math
\frac{d}{dx}
=
\sqrt{\frac{m\omega}{\hbar}}\frac{d}{d\xi}
```

```math
\frac{d^2}{dx^2}
=
\frac{m\omega}{\hbar}\frac{d^2}{d\xi^2}
```

再定义：

```math
\lambda=\frac{2E}{\hbar\omega}
```

方程化为：

```math
\frac{d^2\psi}{d\xi^2}
+
(\lambda-\xi^2)\psi=0
```

## 无穷远行为

当 $|\xi|$ 很大时，$\lambda$ 相对 $\xi^2$ 可忽略，方程近似为：

```math
\frac{d^2\psi}{d\xi^2}-\xi^2\psi\simeq0
```

物理解应在无穷远衰减，典型因子为：

```math
e^{-\xi^2/2}
```

于是设：

```math
\psi(\xi)=e^{-\xi^2/2}H(\xi)
```

## 难点注释

- 引入 $\xi$ 是为了把长度尺度统一到 $\sqrt{\hbar/(m\omega)}$。
- $e^{+\xi^2/2}$ 会发散，不能归一化。
- 后续能级量子化来自 $H(\xi)$ 必须截断成多项式。

## 易错点

- $\lambda=2E/(\hbar\omega)$，不是 $E/(\hbar\omega)$。
- 无量纲化时二阶导要多出因子 $m\omega/\hbar$。
- 衰减因子是 $e^{-\xi^2/2}$。
