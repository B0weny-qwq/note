# 定态 Schrödinger 方程

## 什么时候能分离变量

含时 Schrödinger 方程为：

```math
i\hbar\frac{\partial}{\partial t}\Psi(\vec r,t)
=
\left[
-\frac{\hbar^2}{2m}\nabla^2+U(\vec r)
\right]\Psi(\vec r,t)
```

这里要求势能不显含时间：

```math
U=U(\vec r)
```

## 分离变量

设：

```math
\Psi(\vec r,t)=\psi(\vec r)T(t)
```

代入方程：

```math
i\hbar\psi(\vec r)\frac{dT}{dt}
=
T(t)
\left[
-\frac{\hbar^2}{2m}\nabla^2+U(\vec r)
\right]\psi(\vec r)
```

两边除以 $\psi T$：

```math
i\hbar\frac{1}{T}\frac{dT}{dt}
=
\frac{1}{\psi}
\left[
-\frac{\hbar^2}{2m}\nabla^2+U(\vec r)
\right]\psi
```

左边只依赖 $t$，右边只依赖 $\vec r$，所以二者都等于常数。记为 $E$。

## 时间部分

时间方程：

```math
i\hbar\frac{dT}{dt}=ET
```

解为：

```math
T(t)=e^{-iEt/\hbar}
```

## 空间部分

空间方程：

```math
\left[
-\frac{\hbar^2}{2m}\nabla^2+U(\vec r)
\right]\psi(\vec r)
=
E\psi(\vec r)
```

定义 Hamilton 算符：

```math
\hat H=
-\frac{\hbar^2}{2m}\nabla^2+U(\vec r)
```

得到定态 Schrödinger 方程：

```math
\hat H\psi=E\psi
```

## 定态波函数

完整含时波函数为：

```math
\Psi(\vec r,t)=\psi(\vec r)e^{-iEt/\hbar}
```

概率密度：

```math
|\Psi(\vec r,t)|^2
=
|\psi(\vec r)|^2
```

所以定态的概率密度不随时间变化。

## 难点注释

- 定态不是波函数完全不随时间变，而是只多一个整体相位因子。
- 多个不同能量定态叠加后，相对相位随时间变化，一般不再是定态。
- 定态方程本质上是 Hamilton 算符的本征值方程。

## 易错点

- 势能显含时间时不能这样直接分离。
- $E$ 是分离常数，也是能量本征值。
- $e^{-iEt/\hbar}$ 的符号不要写反。
