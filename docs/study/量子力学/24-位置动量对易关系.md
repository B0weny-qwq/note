# 位置动量对易关系

## 算符定义

一维位置算符：

```math
\hat x=x
```

一维动量算符：

```math
\hat p_x=-i\hbar\frac{d}{dx}
```

对易子定义：

```math
[\hat A,\hat B]=\hat A\hat B-\hat B\hat A
```

## 让算符作用到任意函数上

取任意可微函数 $f(x)$。

先算：

```math
\hat x\hat p_x f
=
x\left(-i\hbar\frac{df}{dx}\right)
=
-i\hbar x f'
```

再算：

```math
\hat p_x\hat x f
=
-i\hbar\frac{d}{dx}(xf)
```

用乘积求导：

```math
\frac{d}{dx}(xf)=f+xf'
```

所以：

```math
\hat p_x\hat x f
=
-i\hbar(f+xf')
```

## 相减

```math
[\hat x,\hat p_x]f
=
\hat x\hat p_xf-\hat p_x\hat xf
```

代入：

```math
[\hat x,\hat p_x]f
=
-i\hbar xf'
-
[-i\hbar(f+xf')]
```

化简：

```math
[\hat x,\hat p_x]f
=
i\hbar f
```

因为对任意 $f$ 都成立：

```math
[\hat x,\hat p_x]=i\hbar
```

## 三维形式

```math
[\hat x_i,\hat p_j]=i\hbar\delta_{ij}
```

并且：

```math
[\hat x_i,\hat x_j]=0
```

```math
[\hat p_i,\hat p_j]=0
```

## 难点注释

- 不能只移动符号，必须让对易子作用到任意函数上。
- 不对易来自乘积求导多出来的 $f$。
- 这个基本对易关系直接导致位置-动量不确定性。

## 易错点

- 顺序不能反：$[\hat p_x,\hat x]=-i\hbar$。
- 不要漏掉任意函数 $f$。
- $x$ 和 $p_x$ 是算符，不是经典变量。
