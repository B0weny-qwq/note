# 氢原子库仑场分离变量

## 势能

类氢离子中电子受到中心库仑势：

```math
V(r)=
-\frac{Ze^2}{4\pi\varepsilon_0r}
```

Hamilton 算符：

```math
\hat H
=
-\frac{\hbar^2}{2\mu}\nabla^2
+
V(r)
```

其中 $\mu$ 是约化质量：

```math
\mu=\frac{m_em_N}{m_e+m_N}
```

## 为什么用球坐标

势能只依赖 $r$，不依赖角度：

```math
V=V(r)
```

所以系统有旋转对称性，可以同时对角化：

```math
\hat H,\quad \hat L^2,\quad \hat L_z
```

设波函数分离为：

```math
\psi(r,\theta,\phi)
=
R(r)Y_l^m(\theta,\phi)
```

其中 $Y_l^m$ 是球谐函数。

## 径向方程结构

球坐标中 Laplace 算子可以分为径向部分和角向部分。利用：

```math
\hat L^2Y_l^m
=
\hbar^2l(l+1)Y_l^m
```

可得到径向方程：

```math
\left[
-\frac{\hbar^2}{2\mu}
\frac{1}{r^2}\frac{d}{dr}
\left(r^2\frac{d}{dr}\right)
+
\frac{\hbar^2l(l+1)}{2\mu r^2}
+
V(r)
\right]R(r)
=
ER(r)
```

其中：

```math
\frac{\hbar^2l(l+1)}{2\mu r^2}
```

是角动量带来的有效离心势项。

## 常用变换

常令：

```math
u(r)=rR(r)
```

可把径向方程化成类似一维形式：

```math
\left[
-\frac{\hbar^2}{2\mu}\frac{d^2}{dr^2}
+
\frac{\hbar^2l(l+1)}{2\mu r^2}
+
V(r)
\right]u(r)
=
Eu(r)
```

## 难点注释

- 离心势不是额外加入的势能，而是角向动能在径向方程中的表现。
- 球谐函数解决角向问题，径向方程决定能级和径向分布。
- 约化质量让两体问题化成等效单体问题。

## 易错点

- 不要直接把电子质量写死，精确处理应用约化质量 $\mu$。
- $\psi=R(r)Y_l^m$ 中 $R$ 和 $Y$ 分别处理径向和角向。
- 径向概率不是 $|R|^2dr$，而是 $|R|^2r^2dr$ 乘角向积分。
