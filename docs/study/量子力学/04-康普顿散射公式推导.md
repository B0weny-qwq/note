# 康普顿散射公式推导

## 物理图像

入射 X 射线或 $\gamma$ 射线光子与近似静止的自由电子碰撞。碰撞后：

- 光子方向改变，频率变小，波长变大。
- 电子获得反冲动量和能量。

实验结论是散射光波长改变只和散射角有关：

```math
\Delta\lambda=\lambda-\lambda_0
```

## 守恒方程

入射光子能量：

```math
E_0=h\nu_0=\frac{hc}{\lambda_0}
```

散射后光子能量：

```math
E=h\nu=\frac{hc}{\lambda}
```

光子动量大小：

```math
p_0=\frac{h}{\lambda_0},\quad p=\frac{h}{\lambda}
```

能量守恒：

```math
h\nu_0+m_ec^2=h\nu+E_e
```

动量守恒：

```math
\vec p_e=\vec p_0-\vec p
```

因此电子反冲动量平方为：

```math
p_e^2=p_0^2+p^2-2p_0p\cos\theta
```

## 使用相对论关系

电子能量动量关系：

```math
E_e^2=p_e^2c^2+m_e^2c^4
```

由能量守恒：

```math
E_e=h\nu_0-h\nu+m_ec^2
```

把它代入相对论关系：

```math
(h\nu_0-h\nu+m_ec^2)^2
=p_e^2c^2+m_e^2c^4
```

展开并消去两边的 $m_e^2c^4$，再用：

```math
p_e^2c^2
=
\left(\frac{h\nu_0}{c}\right)^2c^2
+
\left(\frac{h\nu}{c}\right)^2c^2
-
2\frac{h\nu_0}{c}\frac{h\nu}{c}c^2\cos\theta
```

化简得到：

```math
m_ec^2h(\nu_0-\nu)
=
h^2\nu_0\nu(1-\cos\theta)
```

两边除以 $h\nu_0\nu m_ec$：

```math
\frac{c}{\nu}-\frac{c}{\nu_0}
=
\frac{h}{m_ec}(1-\cos\theta)
```

因为 $\lambda=c/\nu$，所以：

```math
\lambda-\lambda_0
=
\frac{h}{m_ec}(1-\cos\theta)
```

## 最终公式

```math
\Delta\lambda
=
\lambda-\lambda_0
=
\lambda_C(1-\cos\theta)
```

其中：

```math
\lambda_C=\frac{h}{m_ec}
```

称为电子的 Compton 波长。

## 难点注释

- 这里必须用电子的相对论能量动量关系，不能直接用 $\frac12mv^2$。
- 散射角 $\theta$ 是光子散射角，不是电子反冲角。
- 结果不含入射波长以外的材料参数，说明这是光子和自由电子碰撞的几何结果。

## 易错点

- 不要把 $\Delta\lambda$ 写成和电子出射角有关。
- $\theta=0$ 时没有波长改变。
- $\theta=\pi$ 时波长改变最大，为 $2h/(m_ec)$。
