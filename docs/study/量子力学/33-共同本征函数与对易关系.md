# 共同本征函数与对易关系

## 问题

什么时候两个力学量可以同时具有确定值？

量子力学中的判据和算符对易有关。

如果：

```math
[\hat A,\hat B]=0
```

则两个力学量通常可以选取共同本征函数系。

## 非简并情况推导

设：

```math
\hat A\phi=a\phi
```

且：

```math
[\hat A,\hat B]=0
```

则：

```math
\hat A(\hat B\phi)
=
\hat B(\hat A\phi)
```

代入 $\hat A\phi=a\phi$：

```math
\hat A(\hat B\phi)
=
\hat B(a\phi)
=
a\hat B\phi
```

这说明 $\hat B\phi$ 仍然是 $\hat A$ 的本征值 $a$ 对应的本征函数。

如果 $a$ 非简并，则该本征空间只有一维，所以：

```math
\hat B\phi=b\phi
```

于是 $\phi$ 同时也是 $\hat B$ 的本征函数。

## 简并情况

如果 $a$ 是简并本征值，$\hat B\phi$ 只能说明仍在同一个简并子空间中。

这时需要在该简并子空间内把 $\hat B$ 再对角化，才能选出共同本征函数。

## 完全集

为了唯一标记一个态，常常需要一组两两对易的力学量，称为力学量完全集。

氢原子中常用：

```math
\{\hat H,\hat L^2,\hat L_z\}
```

共同本征态记作：

```math
\psi_{nlm}
```

满足：

```math
\hat H\psi_{nlm}=E_n\psi_{nlm}
```

```math
\hat L^2\psi_{nlm}=\hbar^2l(l+1)\psi_{nlm}
```

```math
\hat L_z\psi_{nlm}=m\hbar\psi_{nlm}
```

## 难点注释

- 对易是能否同时测量的重要条件。
- 非简并情况下推导最简单；简并时要在子空间内继续选基。
- 完全集的作用是把态标记到足够细。

## 易错点

- 不要把“对易”理解成普通数字乘法交换，它是算符作用顺序的问题。
- 共同本征函数存在通常还要考虑完备性和定义域。
- 有共同本征函数但不完备时，不能直接推出完整物理结论。
