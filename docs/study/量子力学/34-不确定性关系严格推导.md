# 不确定性关系严格推导

## 定义偏差算符

对力学量 $F$，平均值为：

```math
\langle F\rangle=\langle\psi|\hat F|\psi\rangle
```

定义偏差算符：

```math
\Delta\hat F=\hat F-\langle F\rangle
```

方差为：

```math
(\Delta F)^2
=
\langle(\Delta\hat F)^2\rangle
```

同理：

```math
\Delta\hat G=\hat G-\langle G\rangle
```

## 构造两个态矢

定义：

```math
|\alpha\rangle=\Delta\hat F|\psi\rangle
```

```math
|\beta\rangle=\Delta\hat G|\psi\rangle
```

则：

```math
\langle\alpha|\alpha\rangle=(\Delta F)^2
```

```math
\langle\beta|\beta\rangle=(\Delta G)^2
```

## 使用 Cauchy-Schwarz 不等式

```math
\langle\alpha|\alpha\rangle
\langle\beta|\beta\rangle
\ge
|\langle\alpha|\beta\rangle|^2
```

即：

```math
(\Delta F)^2(\Delta G)^2
\ge
\left|
\langle
\Delta\hat F\Delta\hat G
\rangle
\right|^2
```

## 拆成对易子和反对易子

任意两个算符乘积可写成：

```math
\Delta\hat F\Delta\hat G
=
\frac12[\Delta\hat F,\Delta\hat G]
+
\frac12\{\Delta\hat F,\Delta\hat G\}
```

其中：

```math
[A,B]=AB-BA
```

```math
\{A,B\}=AB+BA
```

常用下界只保留对易子给出的虚部，得到 Robertson 形式：

```math
\Delta F\,\Delta G
\ge
\frac12
\left|
\langle[\hat F,\hat G]\rangle
\right|
```

## 位置和动量

利用：

```math
[\hat x,\hat p_x]=i\hbar
```

可得：

```math
\Delta x\,\Delta p_x
\ge
\frac12|i\hbar|
```

即：

```math
\Delta x\,\Delta p_x
\ge
\frac{\hbar}{2}
```

## 难点注释

- 不确定性来自态空间内积结构和非对易关系，不只是仪器误差。
- 严格 Schrödinger-Robertson 关系还包含反对易协方差项；常用式是一个下界。
- 若两个算符对易，下界可能为 0，但不代表所有态都同时确定。

## 易错点

- 不要把 $\Delta F$ 当成算符，它通常表示标准差；偏差算符应写 $\Delta\hat F$。
- 下界中是对易子的期望值。
- 位置动量关系中的系数是 $\hbar/2$。
