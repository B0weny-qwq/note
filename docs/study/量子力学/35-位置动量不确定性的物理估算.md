# 位置动量不确定性的物理估算

## 基本关系

位置和动量满足：

```math
\Delta x\,\Delta p_x\ge\frac{\hbar}{2}
```

这意味着粒子位置越局域，动量分布越宽。

## 局域在尺度 a 内

如果粒子被限制在长度尺度 $a$ 内，可估计：

```math
\Delta x\sim a
```

则：

```math
\Delta p_x\gtrsim\frac{\hbar}{2a}
```

## 最小动能量级

动能大致为：

```math
K\sim\frac{(\Delta p_x)^2}{2m}
```

代入估算：

```math
K_{\min}
\sim
\frac{1}{2m}
\left(\frac{\hbar}{2a}\right)^2
```

即：

```math
K_{\min}
\sim
\frac{\hbar^2}{8ma^2}
```

这说明局域化会带来非零动能。

## 与无限深势阱联系

无限深势阱宽度为 $L$ 时，精确基态能量：

```math
E_1=\frac{\pi^2\hbar^2}{2mL^2}
```

它和不确定性估算：

```math
\frac{\hbar^2}{mL^2}
```

在量级上一致。

## 物理理解

如果粒子完全静止，动量不确定度很小；但如果又要求它被限制在很小区域内，就违反不确定性关系。

所以束缚态基态能量不能为 0。

## 难点注释

- 不确定性估算只能给数量级，精确系数由具体势能和边界决定。
- 质量越大，局域化带来的最小动能越小，所以宏观物体量子效应不明显。
- 零点能不是热运动能量，而是量子态本身的限制。

## 易错点

- 不要用不确定性关系直接代替精确求解。
- $\Delta x$ 是标准差量级，不一定等于盒子宽度。
- 能量估算中动量应平方。
