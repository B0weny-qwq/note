# 氢原子能级与量子数

## 能级结果

解库仑势场中的径向方程，并要求波函数有限、可归一化，会得到离散能级：

```math
E_n
=
-
\frac{\mu Z^2e^4}
{2(4\pi\varepsilon_0)^2\hbar^2}
\frac{1}{n^2}
```

其中：

```math
n=1,2,3,\cdots
```

对氢原子 $Z=1$，且 $\mu\approx m_e$：

```math
E_n=-\frac{13.6\text{ eV}}{n^2}
```

## Bohr 半径

氢原子长度尺度为：

```math
a_0
=
\frac{4\pi\varepsilon_0\hbar^2}{\mu e^2}
```

数值约为：

```math
a_0=0.529\times 10^{-10}\text{ m}
```

## 三个量子数

氢原子态通常写作：

```math
\psi_{nlm}(r,\theta,\phi)
=
R_{nl}(r)Y_l^m(\theta,\phi)
```

三个量子数取值：

```math
n=1,2,3,\cdots
```

```math
l=0,1,2,\cdots,n-1
```

```math
m=-l,-l+1,\cdots,l
```

## 对应本征方程

能量：

```math
\hat H\psi_{nlm}=E_n\psi_{nlm}
```

角动量平方：

```math
\hat L^2\psi_{nlm}
=
\hbar^2l(l+1)\psi_{nlm}
```

角动量 z 分量：

```math
\hat L_z\psi_{nlm}
=
m\hbar\psi_{nlm}
```

## 简并

能量只依赖 $n$，不依赖 $l,m$：

```math
E=E_n
```

这是库仑势的特殊简并。

对给定 $n$，总态数为：

```math
\sum_{l=0}^{n-1}(2l+1)=n^2
```

若考虑电子自旋，还会再乘以 2。

## 难点注释

- $n$ 决定能量主尺度。
- $l$ 决定角动量大小。
- $m$ 决定角动量在 z 方向投影。
- 电子云图像来自 $|\psi_{nlm}|^2$，不是经典轨道。

## 易错点

- $l$ 最大只能到 $n-1$。
- $m$ 的取值依赖 $l$，不是依赖 $n$ 直接取到 $\pm n$。
- 氢原子能级的 $n$ 从 1 开始。
