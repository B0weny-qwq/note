# 动量算符厄米性

## 厄米性的定义

算符 $\hat F$ 厄米，意味着：

```math
\langle\psi|\hat F\phi\rangle
=
\langle\hat F\psi|\phi\rangle
```

对动量算符：

```math
\hat p=-i\hbar\frac{d}{dx}
```

要检查：

```math
\langle\psi|\hat p\phi\rangle
\stackrel{?}{=}
\langle\hat p\psi|\phi\rangle
```

## 从左边开始

```math
\langle\psi|\hat p\phi\rangle
=
\int_{-\infty}^{\infty}
\psi^*
\left(-i\hbar\frac{d\phi}{dx}\right)dx
```

即：

```math
\langle\psi|\hat p\phi\rangle
=
-i\hbar\int\psi^*\phi' dx
```

## 分部积分

```math
\int\psi^*\phi'dx
=
[\psi^*\phi]_{-\infty}^{\infty}
-
\int(\psi^*)'\phi dx
```

所以：

```math
\langle\psi|\hat p\phi\rangle
=
-i\hbar[\psi^*\phi]_{-\infty}^{\infty}
+
i\hbar\int(\psi^*)'\phi dx
```

注意：

```math
(\hat p\psi)^*
=
\left(-i\hbar\psi'\right)^*
=
i\hbar(\psi^*)'
```

因此：

```math
i\hbar\int(\psi^*)'\phi dx
=
\int(\hat p\psi)^*\phi dx
=
\langle\hat p\psi|\phi\rangle
```

如果边界项为 0：

```math
[\psi^*\phi]_{-\infty}^{\infty}=0
```

则：

```math
\langle\psi|\hat p\phi\rangle
=
\langle\hat p\psi|\phi\rangle
```

动量算符在该函数空间中是厄米的。

## 边界项什么时候为 0

常见情况：

- 束缚态波函数在无穷远处趋于 0。
- 盒归一化中使用周期性边界条件。
- 波函数满足使边界项消失的物理边界条件。

## 难点注释

- 厄米性不是只看算符形式，还要看函数空间和边界条件。
- 分部积分后的边界项是关键。
- 连续谱平面波要用 $\delta$ 归一化或盒归一化谨慎处理。

## 易错点

- 不要忘记复共轭会让 $-i$ 变成 $+i$。
- 不要默认所有微分算符都厄米。
- 边界条件不同，算符定义域也不同。
