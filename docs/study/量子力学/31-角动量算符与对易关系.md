# 角动量算符与对易关系

## 从经典定义开始

经典角动量：

```math
\vec L=\vec r\times\vec p
```

量子力学中：

```math
\hat{\vec p}=-i\hbar\nabla
```

所以：

```math
\hat{\vec L}
=
\hat{\vec r}\times\hat{\vec p}
=
-i\hbar\vec r\times\nabla
```

## 直角坐标分量

```math
\hat L_x
=
-i\hbar
\left(
y\frac{\partial}{\partial z}
-
z\frac{\partial}{\partial y}
\right)
```

```math
\hat L_y
=
-i\hbar
\left(
z\frac{\partial}{\partial x}
-
x\frac{\partial}{\partial z}
\right)
```

```math
\hat L_z
=
-i\hbar
\left(
x\frac{\partial}{\partial y}
-
y\frac{\partial}{\partial x}
\right)
```

角动量平方：

```math
\hat L^2=\hat L_x^2+\hat L_y^2+\hat L_z^2
```

## 对易关系

角动量分量满足：

```math
[\hat L_x,\hat L_y]=i\hbar\hat L_z
```

```math
[\hat L_y,\hat L_z]=i\hbar\hat L_x
```

```math
[\hat L_z,\hat L_x]=i\hbar\hat L_y
```

统一写作：

```math
[\hat L_i,\hat L_j]
=
i\hbar\epsilon_{ijk}\hat L_k
```

其中 $\epsilon_{ijk}$ 是 Levi-Civita 符号。

## 与 L 平方的对易

角动量平方与各分量对易：

```math
[\hat L^2,\hat L_i]=0
```

所以可以同时确定：

```math
L^2
```

和某一个分量，例如：

```math
L_z
```

但不能同时确定 $L_x,L_y,L_z$ 三个分量。

## 难点注释

- 三个角动量分量彼此不对易，所以不能同时都有确定值。
- $\hat L^2$ 和 $\hat L_z$ 对易，因此常用它们共同标记态。
- 角动量对易关系是三维旋转对称性的代数表达。

## 易错点

- 不要把 $L=\sqrt{l(l+1)}\hbar$ 误写成 $l\hbar$。
- $m\hbar$ 是 $L_z$ 的本征值，不是总角动量大小。
- 对易关系中循环顺序决定正负号。
