# 算符与力学量

## 算符是什么

算符是一种作用在函数上的运算规则。它把一个函数变成另一个函数：

```math
\hat F\psi=\phi
```

量子力学中，可观测力学量用算符表示。

例如动量算符：

```math
\hat{\vec p}=-i\hbar\nabla
```

能量算符：

```math
\hat E=i\hbar\frac{\partial}{\partial t}
```

Hamilton 算符：

```math
\hat H=-\frac{\hbar^2}{2m}\nabla^2+U(\vec r)
```

## 本征值方程

如果：

```math
\hat F\psi_n=f_n\psi_n
```

则 $\psi_n$ 是算符 $\hat F$ 的本征函数，$f_n$ 是对应本征值。

物理含义是：如果体系处于 $\psi_n$ 状态，测量力学量 $F$ 必定得到 $f_n$。

## 线性算符

量子态满足叠加原理，所以力学量算符通常要求线性：

```math
\hat F(c_1\psi_1+c_2\psi_2)
=
c_1\hat F\psi_1+c_2\hat F\psi_2
```

如果算符不线性，叠加态的作用结果就无法保持量子理论的基本结构。

## 厄米算符

可观测量的测量结果必须是实数，因此可观测力学量用厄米算符表示：

```math
\hat F^\dagger=\hat F
```

厄米算符的重要性质：

- 本征值为实数。
- 不同本征值对应的本征函数正交。
- 本征函数可构成展开态的基底。

## 难点注释

- 算符不是测量值本身；本征值才是可能测量结果。
- 一般态不是某算符本征态时，测量结果有概率分布。
- 同一个态可以在不同算符的本征基下展开。

## 易错点

- 不要把 $\hat F\psi$ 写成普通乘法，除非 $\hat F$ 本来就是乘法算符。
- 动量算符在位置表象中是微分算符。
- 可观测量算符必须考虑定义域和边界条件。
