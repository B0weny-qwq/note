# 谐振子本征函数与节点

## 本征函数

一维谐振子的归一化本征函数为：

```math
\psi_n(x)
=
\frac{1}{\sqrt{2^nn!}}
\left(\frac{m\omega}{\pi\hbar}\right)^{1/4}
e^{-m\omega x^2/(2\hbar)}
H_n\left(\sqrt{\frac{m\omega}{\hbar}}x\right)
```

其中 $H_n$ 是 Hermite 多项式。

## 前几个 Hermite 多项式

```math
H_0(\xi)=1
```

```math
H_1(\xi)=2\xi
```

```math
H_2(\xi)=4\xi^2-2
```

```math
H_3(\xi)=8\xi^3-12\xi
```

## 能级

```math
E_n=\hbar\omega\left(n+\frac12\right),
\quad n=0,1,2,\cdots
```

## 正交归一

谐振子本征函数满足：

```math
\int_{-\infty}^{\infty}
\psi_m^*(x)\psi_n(x)dx
=
\delta_{mn}
```

因此任意可接受波函数可以展开为：

```math
\Psi(x,0)=\sum_{n=0}^{\infty}c_n\psi_n(x)
```

## 奇偶性

势能：

```math
U(x)=\frac12m\omega^2x^2
```

关于 $x=0$ 对称，所以本征函数具有确定奇偶性。

- $n$ 偶数时，$\psi_n$ 是偶函数。
- $n$ 奇数时，$\psi_n$ 是奇函数。

## 节点数

第 $n$ 个谐振子本征函数有 $n$ 个节点。

例如：

- $n=0$ 基态无节点。
- $n=1$ 有 1 个节点。
- $n=2$ 有 2 个节点。

## 难点注释

- Gaussian 因子控制无穷远处衰减。
- Hermite 多项式控制节点结构。
- 奇偶性常用于判断矩阵元是否为 0。

## 易错点

- 不要忘记归一化系数中的 $2^nn!$。
- $H_n$ 的自变量是 $\xi=\sqrt{m\omega/\hbar}x$。
- 节点数和量子数 $n$ 相同，但能量还多一个 $\frac12$。
