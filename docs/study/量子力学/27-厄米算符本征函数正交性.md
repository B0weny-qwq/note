# 厄米算符本征函数正交性

## 定理

设 $\hat F$ 是厄米算符：

```math
\hat F^\dagger=\hat F
```

若：

```math
\hat F\psi_m=\lambda_m\psi_m
```

```math
\hat F\psi_n=\lambda_n\psi_n
```

且：

```math
\lambda_m\ne\lambda_n
```

则：

```math
\langle\psi_m|\psi_n\rangle=0
```

## 证明

先计算：

```math
\langle\psi_m|\hat F\psi_n\rangle
```

由 $\hat F\psi_n=\lambda_n\psi_n$：

```math
\langle\psi_m|\hat F\psi_n\rangle
=
\lambda_n\langle\psi_m|\psi_n\rangle
```

利用厄米性：

```math
\langle\psi_m|\hat F\psi_n\rangle
=
\langle\hat F\psi_m|\psi_n\rangle
```

由 $\hat F\psi_m=\lambda_m\psi_m$，且本征值为实数：

```math
\langle\hat F\psi_m|\psi_n\rangle
=
\lambda_m\langle\psi_m|\psi_n\rangle
```

所以：

```math
\lambda_n\langle\psi_m|\psi_n\rangle
=
\lambda_m\langle\psi_m|\psi_n\rangle
```

整理：

```math
(\lambda_n-\lambda_m)
\langle\psi_m|\psi_n\rangle=0
```

因为 $\lambda_n\ne\lambda_m$，所以：

```math
\langle\psi_m|\psi_n\rangle=0
```

## 简并情况

如果多个不同本征函数对应同一个本征值，叫简并。

此时上面的证明不能推出它们自动正交，因为：

```math
\lambda_n-\lambda_m=0
```

但可以在简并子空间中用 Gram-Schmidt 方法重新选取一组正交归一基。

## 难点注释

- 正交性只直接适用于不同本征值。
- 简并态可以选成正交的，但不是自动必须正交。
- 正交归一基使任意态展开和测量概率计算变得简单。

## 易错点

- 不要说“所有本征函数天然正交”，简并情况要额外处理。
- 厄米性是证明正交性的关键。
- 连续谱中 Kronecker $\delta$ 要换成 Dirac $\delta$。
