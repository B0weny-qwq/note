# 波函数统计解释与归一化

## 波函数描述什么

波函数 $\psi(\vec r,t)$ 描述量子体系的状态。它本身不是概率，而是概率幅。

Born 统计解释说：粒子在位置 $\vec r$ 附近出现的概率密度为：

```math
\rho(\vec r,t)=|\psi(\vec r,t)|^2
```

因为波函数一般是复函数，所以更完整地写作：

```math
\rho(\vec r,t)
=
\psi^*(\vec r,t)\psi(\vec r,t)
```

## 归一化条件

粒子在全空间某处出现的总概率应为 1：

```math
\int_{\mathbb R^3}|\psi(\vec r,t)|^2d^3r=1
```

如果是一维问题：

```math
\int_{-\infty}^{\infty}|\psi(x,t)|^2dx=1
```

## 非归一化波函数怎么处理

如果原来的波函数 $\psi$ 没有归一化，可以引入常数 $c$：

```math
\Phi(\vec r,t)=c\psi(\vec r,t)
```

要求：

```math
\int|\Phi|^2d^3r=1
```

代入：

```math
|c|^2\int|\psi|^2d^3r=1
```

所以：

```math
|c|^{-2}=\int|\psi|^2d^3r
```

若取 $c$ 为正实数：

```math
c=\frac{1}{\sqrt{\int|\psi|^2d^3r}}
```

## 整体相位不影响概率

若：

```math
\psi'(\vec r,t)=e^{i\alpha}\psi(\vec r,t)
```

则：

```math
|\psi'|^2
=
(e^{i\alpha}\psi)^*(e^{i\alpha}\psi)
=
e^{-i\alpha}e^{i\alpha}\psi^*\psi
=|\psi|^2
```

所以整体相位不可观测。

## 难点注释

- 概率密度必须是非负实数，所以用 $\psi^*\psi$。
- 不是所有波函数都能归一化。例如无限空间中的平面波不能归一化为 1。
- 整体相位不影响概率，但相对相位会影响干涉。

## 易错点

- 不要把 $\psi$ 当成概率。
- 不要用 $\psi^2$ 代替 $|\psi|^2$。
- 归一化常数可以含相位，但通常取正实数即可。
