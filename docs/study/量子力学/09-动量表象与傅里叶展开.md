# 动量表象与傅里叶展开

## 表象是什么意思

同一个量子态可以用不同变量表示：

- 用位置 $\vec r$ 作自变量，得到位置表象波函数 $\psi(\vec r,t)$。
- 用动量 $\vec p$ 作自变量，得到动量表象波函数 $\phi(\vec p,t)$。

它们描述的是同一个态，只是展开基底不同。

## 动量本征平面波

具有确定动量 $\vec p$ 的自由粒子，对应平面波：

```math
u_{\vec p}(\vec r)
=
\frac{1}{(2\pi\hbar)^{3/2}}
e^{i\vec p\cdot\vec r/\hbar}
```

它满足：

```math
\hat{\vec p}u_{\vec p}=\vec p\,u_{\vec p}
```

其中：

```math
\hat{\vec p}=-i\hbar\nabla
```

## 从动量表象到位置表象

一般波函数可以看成不同动量平面波的叠加：

```math
\psi(\vec r,t)
=
\int \phi(\vec p,t)u_{\vec p}(\vec r)d^3p
```

这里 $\phi(\vec p,t)$ 就是动量空间中的概率幅。

## 从位置表象到动量表象

利用平面波基底的正交归一关系，可得逆变换：

```math
\phi(\vec p,t)
=
\int u_{\vec p}^*(\vec r)\psi(\vec r,t)d^3r
```

写开就是：

```math
\phi(\vec p,t)
=
\frac{1}{(2\pi\hbar)^{3/2}}
\int e^{-i\vec p\cdot\vec r/\hbar}
\psi(\vec r,t)d^3r
```

这就是带有 $\hbar$ 尺度的傅里叶变换。

## 概率守恒

傅里叶变换保持总概率：

```math
\int|\psi(\vec r,t)|^2d^3r
=
\int|\phi(\vec p,t)|^2d^3p
```

如果 $\psi$ 归一化，则 $\phi$ 也归一化。

## 难点注释

- 平面波是连续谱本征函数，严格上归一化到 Dirac $\delta$ 函数。
- $|\phi(\vec p,t)|^2d^3p$ 表示动量落在 $d^3p$ 范围内的概率。
- 位置越局域，动量表象越分散，这是不确定性的傅里叶来源。

## 易错点

- 不要把 $\phi(\vec p,t)$ 理解成另一个物理粒子，它只是同一态的另一种表示。
- 变换常数要和所用约定一致。
- 连续谱时求和要改成积分。
