# 矩形势垒：E 大于 U0

## 势垒模型

设矩形势垒为：

```math
U(x)=
\begin{cases}
0,&x<0\\
U_0,&0<x<a\\
0,&x>a
\end{cases}
```

讨论粒子从左侧入射，且：

```math
E>U_0
```

## 各区波函数

I 区：

```math
\psi_I=Ae^{ik_1x}+Be^{-ik_1x}
```

其中第一项是入射波，第二项是反射波。

II 区：

```math
\psi_{II}=Ce^{ik_2x}+De^{-ik_2x}
```

III 区：

```math
\psi_{III}=Fe^{ik_1x}
```

这里不写 $e^{-ik_1x}$，因为假设右侧没有从无穷远入射的波。

波数为：

```math
k_1=\frac{\sqrt{2mE}}{\hbar}
```

```math
k_2=\frac{\sqrt{2m(E-U_0)}}{\hbar}
```

## 边界条件

在 $x=0$：

```math
\psi_I(0)=\psi_{II}(0)
```

```math
\psi_I'(0)=\psi_{II}'(0)
```

在 $x=a$：

```math
\psi_{II}(a)=\psi_{III}(a)
```

```math
\psi_{II}'(a)=\psi_{III}'(a)
```

解这四个方程可得到反射振幅 $B/A$ 和透射振幅 $F/A$。

## 透射系数

概率流密度：

```math
J=\frac{\hbar}{2mi}(\psi^*\psi'-\psi\psi'^*)
```

因为左右两侧势能相同，入射波和透射波波数相同，所以透射系数可写成：

```math
D=\frac{|F|^2}{|A|^2}
```

最终结果：

```math
D
=
\frac{1}
{1+
\frac{(k_1^2-k_2^2)^2}{4k_1^2k_2^2}
\sin^2(k_2a)}
```

反射系数：

```math
R=1-D
```

## 共振透射

当：

```math
\sin(k_2a)=0
```

即：

```math
k_2a=n\pi
```

有：

```math
D=1
```

这叫共振透射。

## 难点注释

- 即使 $E>U_0$，也不一定完全透射，量子粒子仍可能反射。
- 反射来自边界处波函数匹配导致的相位和振幅调整。
- 共振透射可以理解为势垒内多次反射后透射波相位相干增强。

## 易错点

- 透射系数一般要考虑速度比；这里只因左右区域势能相同，速度比为 1。
- 不要把 $D$ 写成 $F/A$，应是概率流比。
- $R+D=1$ 表示概率流守恒。
