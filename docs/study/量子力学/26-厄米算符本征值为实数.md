# 厄米算符本征值为实数

## 定理

如果：

```math
\hat F^\dagger=\hat F
```

且：

```math
\hat F\psi=\lambda\psi
```

那么本征值 $\lambda$ 一定是实数。

## 证明

从内积开始：

```math
\langle\psi|\hat F\psi\rangle
```

利用本征方程：

```math
\hat F\psi=\lambda\psi
```

得到：

```math
\langle\psi|\hat F\psi\rangle
=
\lambda\langle\psi|\psi\rangle
```

另一方面，利用厄米性：

```math
\langle\psi|\hat F\psi\rangle
=
\langle\hat F\psi|\psi\rangle
```

再代入本征方程：

```math
\langle\hat F\psi|\psi\rangle
=
\langle\lambda\psi|\psi\rangle
```

注意复数从 bra 侧提出时要取共轭：

```math
\langle\lambda\psi|\psi\rangle
=
\lambda^*\langle\psi|\psi\rangle
```

因此：

```math
\lambda\langle\psi|\psi\rangle
=
\lambda^*\langle\psi|\psi\rangle
```

整理：

```math
(\lambda-\lambda^*)\langle\psi|\psi\rangle=0
```

非零态满足：

```math
\langle\psi|\psi\rangle>0
```

所以：

```math
\lambda=\lambda^*
```

即 $\lambda$ 为实数。

## 物理意义

可观测力学量的测量结果必须是实数。厄米算符的实本征值正好可以作为可能测量结果。

## 难点注释

- 证明中最容易漏的是 $\lambda$ 从左边态提出要变成 $\lambda^*$。
- 范数 $\langle\psi|\psi\rangle$ 不能为 0，因为物理态不是零函数。
- 厄米性是可观测量算符的核心要求。

## 易错点

- 不要把 $\langle\lambda\psi|\psi\rangle$ 写成 $\lambda\langle\psi|\psi\rangle$。
- 本征值实数不代表本征函数一定是实函数。
- 复波函数可以对应实测量值。
