# 无限深方势阱

## 问题设置

取势阱：

```math
U(x)=
\begin{cases}
0,&0<x<L\\
\infty,&x\le 0\ \text{或}\ x\ge L
\end{cases}
```

墙外势能无限大，粒子不可能出现在墙外：

```math
\psi(x)=0,\quad x\le 0 \text{ 或 } x\ge L
```

边界处：

```math
\psi(0)=0,\quad \psi(L)=0
```

## 阱内方程

阱内 $U=0$，定态方程为：

```math
-\frac{\hbar^2}{2m}\frac{d^2\psi}{dx^2}
=
E\psi
```

整理：

```math
\frac{d^2\psi}{dx^2}+k^2\psi=0
```

其中：

```math
k=\frac{\sqrt{2mE}}{\hbar}
```

通解：

```math
\psi(x)=A\sin kx+B\cos kx
```

## 使用边界条件

由 $\psi(0)=0$：

```math
\psi(0)=B=0
```

所以：

```math
\psi(x)=A\sin kx
```

由 $\psi(L)=0$：

```math
A\sin kL=0
```

非零态要求 $A\ne 0$，所以：

```math
\sin kL=0
```

即：

```math
kL=n\pi,\quad n=1,2,3,\cdots
```

所以：

```math
k_n=\frac{n\pi}{L}
```

## 能级

由：

```math
E=\frac{\hbar^2k^2}{2m}
```

得：

```math
E_n=\frac{n^2\pi^2\hbar^2}{2mL^2}
```

## 归一化

本征函数为：

```math
\psi_n(x)=A\sin\frac{n\pi x}{L}
```

归一化：

```math
\int_0^L|\psi_n(x)|^2dx=1
```

即：

```math
|A|^2\int_0^L\sin^2\frac{n\pi x}{L}dx=1
```

积分为 $L/2$，所以：

```math
|A|^2\frac{L}{2}=1
```

取 $A$ 为正实数：

```math
A=\sqrt{\frac{2}{L}}
```

最终：

```math
\psi_n(x)=\sqrt{\frac{2}{L}}\sin\frac{n\pi x}{L}
```

## 难点注释

- $n=0$ 会得到零波函数，不是物理态。
- 最低能量 $E_1$ 不为 0，这叫零点能。
- 能级间隔随 $n$ 增大而变大。

## 易错点

- 不要把墙外波函数也写成指数衰减；无限深势阱墙外直接为 0。
- 无限墙处导数不要求连续。
- $n$ 从 1 开始。
