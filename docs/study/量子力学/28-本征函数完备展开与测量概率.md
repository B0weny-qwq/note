# 本征函数完备展开与测量概率

## 完备本征函数系

设厄米算符 $\hat A$ 的本征函数为：

```math
\hat A\phi_n=a_n\phi_n
```

如果这些本征函数构成正交归一完备系，则：

```math
\langle\phi_m|\phi_n\rangle=\delta_{mn}
```

任意态 $\psi$ 可以展开为：

```math
\psi=\sum_n c_n\phi_n
```

## 展开系数

两边左乘 $\langle\phi_m|$：

```math
\langle\phi_m|\psi\rangle
=
\sum_n c_n\langle\phi_m|\phi_n\rangle
```

利用正交归一：

```math
\langle\phi_m|\psi\rangle
=
c_m
```

所以：

```math
c_n=\langle\phi_n|\psi\rangle
```

在位置表象中：

```math
c_n=\int\phi_n^*(x)\psi(x)dx
```

## 测量概率

如果测量力学量 $A$，可能得到本征值 $a_n$。对应概率为：

```math
P(a_n)=|c_n|^2
```

如果 $\psi$ 已归一化，则：

```math
\sum_n |c_n|^2=1
```

## 平均值

测量平均值为：

```math
\langle A\rangle
=
\sum_n |c_n|^2a_n
```

也可写成算符夹心形式：

```math
\langle A\rangle
=
\langle\psi|\hat A|\psi\rangle
```

## 连续谱情况

如果本征值连续，求和换成积分：

```math
\psi=\int c(a)\phi_a\,da
```

概率密度为：

```math
P(a)da=|c(a)|^2da
```

## 难点注释

- 完备性不是正交性自动推出的，还需要本征函数足够“铺满”态空间。
- $c_n$ 是概率幅，$|c_n|^2$ 才是概率。
- 测量后态会变成对应本征态或本征子空间中的态。

## 易错点

- 不要漏掉复共轭：$c_n=\int\phi_n^*\psi dx$。
- 连续谱不能直接用普通求和。
- 若态未归一化，概率要先除以总范数。
