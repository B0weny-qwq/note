# 信息率失真函数

## 背景

无失真压缩有理论下界，允许失真时还可以继续压缩。第 4 章研究“允许多大失真时，最少需要多少信息率”。

## 失真度量

常见做法是定义失真函数 `d(x,y)`，再求平均失真：

```text
D = E[d(X,Y)]
```

离散信源常用汉明失真，连续信源常用平方误差失真。

## 率失真函数

定义：

```text
R(D) = min I(X;Y)
```

约束条件是平均失真不超过给定值 `D`。

理解：

- `D` 越大，允许失真越多，所需码率越低。
- `D = 0` 时回到无失真编码极限。

## 基本性质

必须掌握：

- `R(D)` 单调不增。
- `R(D)` 是下凸函数。
- 当 `D >= Dmax` 时，`R(D) = 0`。

## 离散信源求法

离散题通常给出：

- 信源概率分布。
- 失真矩阵。

求解重点：

- 写平均失真约束。
- 写互信息。
- 用参量表达式或已知结论求 `R(D)`。

## 连续信源结果

高斯信源在均方误差准则下有经典结论，是本章最重要的连续情形。

## 工程意义

率失真理论给有损压缩一个不可突破的下界。语音、图像、视频压缩都在追求以更低码率达到可接受失真。

## 做题提醒

- 先分清是“无失真编码”还是“限失真编码”。
- `R(D)` 是最小互信息，不是任意互信息。
- 离散信源多看失真矩阵，连续信源多看平方误差模型。
