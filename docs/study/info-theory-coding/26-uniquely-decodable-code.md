# 唯一可译码

## 定义

唯一可译码是指：任意有限长码流，只能对应唯一一种原始消息序列。

## 为什么重要

如果码流能对应多种原消息，那接收端就不知道原文到底是什么，编码就失去了意义。

## 详细例子

假设：

- `A -> 0`
- `B -> 01`
- `C -> 1`

收到码流 `01` 时：

- 可以解释成 `B`
- 也可以解释成 `A C`

这种码就不是唯一可译码。

## 生活类比

像把一句话写成没有空格的拼音：

```text
woaizhongguo
```

如果分词规则不唯一，就可能读出不同句子。

## 易错点

- 不是“每个码字不同”就一定唯一可译。

## 注意点

- 唯一可译是无失真信源编码的最基本要求之一。
