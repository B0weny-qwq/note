# 自信息量

## 定义

自信息量描述“一个事件发生后，给我们带来了多少信息”。

公式：

```text
I(x) = -log p(x)
```

如果用以 2 为底的对数，单位是 `bit`。

## 为什么要带负号

因为概率 `p(x)`` 在 0 到 1 之间，`log p(x)` 是负数。加负号后，信息量就是正的。

## 直觉理解

- 越常见的事，信息量越小。
- 越罕见的事，信息量越大。

## 详细例子

### 例 1：抛硬币

如果一枚均匀硬币正反概率各为 `1/2`：

```text
I(正面) = -log2(1/2) = 1 bit
```

### 例 2：彩票开奖

如果一个号码命中的概率极低，比如 `1/1000000`：

```text
I(命中) = -log2(1/1000000)
```

这个值很大，说明它极不常见，所以带来的“意外程度”很高。

## 生活类比

每天太阳升起，没人觉得惊讶，所以信息量很小。  
夏天下雪，极少发生，所以一旦发生，信息量很大。

## 易错点

- 自信息量不是“消息长度”。
- 自信息量描述的是单个事件，不是平均情况。

## 注意点

- 自信息量只依赖概率，不依赖语义重要程度。
- 对你很重要的消息，不一定在信息论里信息量大。
