# 信源的描述和分类

## 为什么要先给信源分类

因为不先弄清楚信源是什么样，就没法选正确的数学模型，也没法算熵、互信息和冗余度。

## 基本分类

### 离散信源

消息在时间和幅度上都离散。

例子：

- 文字
- 数字
- 键盘输入
- 文件中的字符流

数学模型通常是：

- 离散型随机变量
- 离散型随机序列

### 连续信源

消息在时间或幅度上连续。

例子：

- 语音
- 图像亮度
- 温度、电压、压力等模拟量

数学模型通常是：

- 连续型随机变量
- 随机过程

## 按记忆性分类

### 无记忆信源

当前输出与过去输出独立。

例子：

- 理想独立掷硬币
- 每次独立报码的随机源

### 有记忆信源

当前输出会受历史输出影响。

例子：

- 中文文本中“你”后面更可能出现“好”
- 语音波形前后样本高度相关

## 马尔可夫信源

马尔可夫信源是一类典型的有记忆信源。

它的特点是：

- 当前状态只与有限个过去状态有关
- 不必记住全部历史

比如一阶马尔可夫信源：

```text
当前输出只依赖前一个符号
```

## 详细例子

天气预测：

- 今天晴天，明天大概率还是晴天
- 今天下雨，明天继续下雨的概率更高

这就是“有记忆”，很适合用马尔可夫链描述。

## 生活类比

输入法联想就是典型有记忆信源模型：

- 你刚打了“信息”，下一词更可能是“论”
- 不同上下文会影响下一个符号的概率

## 易错点

- “连续信源”不是说输出永远连续不断，而是取值空间或时间变化连续。
- “无记忆”不是没有历史记录，而是历史对当前概率没有影响。

## 注意点

- 后续所有熵和互信息计算都隐含依赖这里的建模方式。
