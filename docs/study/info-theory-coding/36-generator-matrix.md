# 生成矩阵 G

## 定义

生成矩阵 `G` 用来把信息位向量映射成码字。

如果消息向量是 `u`，那么码字通常写成：

```text
c = uG
```

运算一般在模 2 下进行。

## 它解决什么问题

它把“编码”变成一个标准矩阵乘法问题。

## 详细例子

如果：

```text
u = [1 0 1]
```

给定生成矩阵 `G` 后，就能算出对应码字 `c`。

## 系统码中的常见形式

系统码常把原始信息位直接保留在码字一部分中。  
这类情况下生成矩阵常有单位阵结构。

## 生活类比

像做身份证号码生成：

- 前面是原始出生日期等信息
- 后面按规则自动算出校验位

生成矩阵就像这套“自动补全规则”。

## 易错点

- `G` 不唯一，不同基底选择可能得到不同 `G`。

## 注意点

- 题目一旦给 `G`，通常马上联想到“生成码字”和“求系统码结构”。
