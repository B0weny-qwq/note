# 信息、信号与信道的区别

## 先把几个最容易混的词分开

`信息`
: 你真正想表达的内容，比如“门开了”“温度是 28 度”。

`消息`
: 信息的具体表现形式，比如一句话、一个字、一个电压值。

`信号`
: 消息的物理载体，比如电压波形、无线电波、光强变化。

`信道`
: 信号传播的通路，比如网线、光纤、空气、串口线。

`信宿`
: 信息最后到达的对象，比如人、显示器、存储器、控制器。

## 通信系统模型

经典模型：

```text
信息源 -> 发送设备 -> 信道 -> 接收设备 -> 受信者
```

数字通信模型会更细：

```text
信源 -> 信源编码 -> 加密编码 -> 信道编码 -> 调制/线路编码 -> 信道
     -> 解调/线路译码 -> 信道译码 -> 解密 -> 信源译码 -> 信宿
```

## 详细例子

你给朋友发微信语音：

- 你脑子里的想法：信息
- 你说出的语音内容：消息
- 麦克风采集到的模拟电压：信号
- 网络链路和无线链路：信道
- 朋友的耳朵和大脑：信宿

## 生活类比

把“点外卖”映射到通信系统：

- 想吃什么：信息
- 你在 app 里输入的文字：消息
- 服务器之间传输的数据包：信号
- 互联网链路：信道
- 商家和骑手系统：接收端和信宿

## 易错点

- 信息不是信号，信号只是载体。
- 信道不是“内容”，而是“路”。

## 注意点

- 信息论研究时，通常不关心消息的语义，只关心可度量性和传输规律。
