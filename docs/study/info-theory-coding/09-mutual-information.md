# 互信息

## 定义

互信息表示两个变量共享了多少信息。

记作：

```text
I(X;Y)
```

## 常用公式

```text
I(X;Y) = H(X) - H(X|Y)
I(X;Y) = H(Y) - H(Y|X)
I(X;Y) = H(X) + H(Y) - H(X,Y)
```

## 直觉理解

如果知道 `Y` 之后，`X` 的不确定性下降很多，那么 `X` 和 `Y` 的互信息就大。

## 详细例子

### 例 1：完全无关

如果 `X` 和 `Y` 独立：

```text
I(X;Y)=0
```

因为知道 `Y` 对猜 `X` 没帮助。

### 例 2：完全决定

如果 `Y=X`，也就是 `Y` 完全由 `X` 决定：

```text
I(X;Y)=H(X)
```

因为知道 `Y` 就等于知道 `X`。

## 生活类比

看天气预报和决定是否带伞之间存在互信息。  
如果天气预报很准，它能大幅减少你对“是否带伞”的不确定性，互信息就大。

## 易错点

- 互信息不是“交集大小”的直观面积，而是概率意义上的共享信息量。

## 注意点

- 第 3 章的信道容量本质上就是让互信息尽可能大。
