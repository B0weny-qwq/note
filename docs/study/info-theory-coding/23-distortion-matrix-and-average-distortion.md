# 失真矩阵与平均失真

## 失真矩阵是什么

对于离散信源，可以把每一对 `(原符号, 重构符号)` 的失真写成一个矩阵。

比如：

```text
d(xi, yj)
```

每个元素表示原来是 `xi`、重构成 `yj` 时的代价。

## 平均失真是什么

平均失真就是长期平均下来，每次重构大概损失了多少。

可以理解为：

```text
D = E[d(X, Y)]
```

## 详细例子

假设源符号只有 `0` 和 `1`，重构符号也只有 `0` 和 `1`，汉明失真矩阵就是：

```text
0 1
1 0
```

含义：

- 原来是 0，重构成 0，失真 0
- 原来是 0，重构成 1，失真 1
- 原来是 1，重构成 0，失真 1
- 原来是 1，重构成 1，失真 0

如果系统中错一半、对一半，那么平均失真就是 0.5。

## 生活类比

老师批改选择题：

- 答对得 0 罚分
- 答错得 1 罚分

全班平均罚分，就相当于平均失真。

## 易错点

- 失真矩阵不是转移概率矩阵，它表示的是“代价”，不是“发生概率”。

## 注意点

- 平均失真必须结合概率分布一起算，不能只看矩阵值本身。
