# 率失真函数 R(D)

## 定义

率失真函数表示：在平均失真不超过 `D` 的条件下，最少需要多少码率。

公式：

```text
R(D) = min I(X;Y)
```

## 直觉理解

如果允许失真变大，压缩就能更狠，所需码率就会下降。  
如果要求完全不失真，码率就必须更高。

## 图像意义

`R(D)` 曲线通常是单调下降的：

- `D` 越小，要求越严格，`R(D)` 越高
- `D` 越大，允许失真越多，`R(D)` 越低

## 详细例子

你发一张照片给同学：

- 如果要求原图级别，一点都不能糊，文件就很大
- 如果允许“肉眼看起来差不多”，就可以压得很小

这就是率失真思想。

## 生活类比

搬家时装衣服：

- 不允许一点褶皱，就得平铺，占很多箱子
- 允许轻微压皱，就能压进更小空间

这里“压皱程度”像失真，“箱子体积”像码率。

## 易错点

- `R(D)` 是“最小互信息”，不是“任意满足条件的互信息”。
