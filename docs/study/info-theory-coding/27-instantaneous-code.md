# 即时码与非延长码

## 定义

即时码是指：当接收端读到一个完整码字后，可以立刻译出对应符号，而不需要继续看后面的码元。

即时码也叫非延长码。

## 为什么它比唯一可译更强

唯一可译只要求“最终能唯一还原”。  
即时码要求“读到这就能立刻决定”，条件更强。

## 详细例子

如果：

- `A -> 0`
- `B -> 10`
- `C -> 11`

那么收到 `0` 时立刻知道是 `A`，收到 `10` 时立刻知道是 `B`。  
因为没有任何码字是别的码字前缀，所以这是即时码。

## 生活类比

即时码像地铁站语音播报里固定完整短语，一听到完整词你就知道意思，不用再等下半句。

## 易错点

- 即时码一定唯一可译，但唯一可译码不一定即时。

## 注意点

- 工程里即时码很重要，因为译码延迟低、实现简单。
