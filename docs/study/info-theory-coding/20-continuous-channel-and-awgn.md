# 连续信道与高斯信道

## 连续信道是什么

如果输入、输出或噪声变量是连续取值的，通常就把它看成连续信道。

典型例子：

- 无线通信中的模拟幅度变化
- 模拟传感器输出
- 带噪声的连续波形传输

## 高斯信道为什么重要

课件里重点讨论的是加性高斯白噪声信道（AWGN）。  
原因是：

- 数学上好分析
- 工程上很常见
- 很多噪声叠加后近似高斯

## “加性高斯”是什么意思

`加性`
: 噪声直接加在信号上。

`高斯`
: 噪声服从高斯分布。

## 详细例子

发射信号是 `x`，接收端收到：

```text
y = x + n
```

其中 `n` 是高斯噪声。

## 生活类比

你对着电话讲话，原话是 `x`，线路里混入均匀分布的小杂声并不常见，很多实际随机扰动叠加后更像高斯噪声，这就是 `n`。

## 易错点

- 连续信道不是“无限快”，只是变量连续。

## 注意点

- 连续信道的容量分析通常最终会落到信噪比和带宽上。
