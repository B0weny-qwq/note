# 校验矩阵 H

## 定义

校验矩阵 `H` 用来检查一个向量是不是合法码字。

合法码字应满足：

```text
cH^T = 0
```

## 它解决什么问题

生成矩阵负责“造码字”，校验矩阵负责“验码字”。

## 详细例子

如果收到一个码字 `r`，你可以算：

```text
rH^T
```

- 结果为 0，说明可能无错
- 结果非 0，说明有差错

## 生活类比

像考试答题卡上的校验规则：

- 你填完答案后，系统会做一轮结构检查
- 如果不满足规则，就知道有问题

## 易错点

- `H` 不是拿来直接生成码字的。

## 注意点

- `H` 在伴随式、检错、纠错、标准阵列译码里都很关键。
