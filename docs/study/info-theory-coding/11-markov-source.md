# 马尔可夫信源

## 定义

马尔可夫信源是一类有记忆信源。  
它的当前输出概率只与有限个过去状态有关，而不需要依赖完整历史。

## 一阶马尔可夫信源

一阶的意思是：

```text
当前符号只和前一个符号有关
```

## 二阶马尔可夫信源

二阶的意思是：

```text
当前符号和前两个符号有关
```

## 状态转移图

课件里强调用状态转移概率图（香农线图）表示。

图上每条边表示：

- 从哪个状态出发
- 转到哪个状态
- 概率是多少

## 详细例子

天气系统：

- 今天晴，明天晴的概率 0.8
- 今天晴，明天下雨的概率 0.2
- 今天雨，明天晴的概率 0.3
- 今天雨，明天下雨的概率 0.7

这就是一阶马尔可夫模型。

## 生活类比

输入法联想、语音连续波形、股票涨跌趋势的简化模型，都常被近似成马尔可夫过程。

## 易错点

- 马尔可夫“有记忆”，但不是“记住全部历史”。

## 注意点

- 马尔可夫模型是理解有记忆信源熵率和冗余度的核心例子。
