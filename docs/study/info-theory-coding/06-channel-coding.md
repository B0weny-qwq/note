# 信道编码

## 背景

信道编码通过增加监督信息换取可靠性。核心不是压缩，而是检错和纠错。

## 差错控制基本方式

三种思路：

- 反馈重发 `ARQ`
- 前向纠错 `FEC`
- 混合纠错 `HEC`

选择时在实时性、复杂度和可靠性之间权衡。

## 线性分组码

这是本章基础。

重点对象：

- 生成矩阵 `G`
- 校验矩阵 `H`
- 码字、信息位、监督位

分析任务：

- 由 `G` 生成码字。
- 由 `H` 求校验关系和伴随式。

## 码距、检错和纠错能力

最小码距 `dmin` 决定性能：

- 检错能力：`dmin - 1`
- 纠错能力：`floor((dmin - 1) / 2)`

这是全章最常用的结论。

## 汉明码

汉明码是最典型的单错纠正码。

重点：

- 监督位个数和码长关系。
- 校验矩阵构造。
- 用伴随式定位错位。

## 循环码

循环码既是线性分组码，又具有循环移位封闭性。

常见工具：

- 生成多项式。
- 码多项式。
- 系统循环码构造。

## 卷积码与后续方法

课件后部还介绍：

- 卷积码。
- 网格图与状态图。
- Turbo/TCM 等扩展方法。

这些内容更偏工程实现和现代编码方案的入口。

## 做题方法

1. 先看是分组码还是卷积码。
2. 分组码先写 `G/H`。
3. 再求最小码距、检错能力、纠错能力。
4. 纠错题优先看伴随式或多项式除法。
