# 失真函数

## 定义

失真函数用来度量“原始信号”和“重构信号”之间差了多少。

一般写成：

```text
d(x, y)
```

其中：

- `x` 是原始符号或原始样本
- `y` 是解码后得到的重构符号或重构样本

## 为什么要定义失真

因为有损压缩不是“完全一样”，而是“差多少还算能接受”。  
失真函数就是这个“差多少”的数学标尺。

## 典型失真函数

### 汉明失真

定义非常简单：

- 一样，失真为 0
- 不一样，失真为 1

适合离散符号场景。

### 平方误差失真

常写成：

```text
(x - y)^2
```

适合连续信号，例如语音、图像亮度、电压采样。

## 详细例子

### 例 1：字符传输

原始字符是 `A`，重构字符也是 `A`，那么汉明失真为 0。  
如果重构成 `B`，那么汉明失真为 1。

### 例 2：温度测量

真实温度 25.2 度，重构温度 25.0 度：

```text
d(x,y) = (25.2 - 25.0)^2 = 0.04
```

## 生活类比

失真函数像评分尺：

- 如果是选择题，答案对错只有 0 和 1，这像汉明失真。
- 如果是投篮距离，差得越远扣分越多，这像平方误差失真。

## 易错点

- 失真不等于误码率，它更一般。
- 不同业务场景要选不同失真函数，不能一种到底。

## 注意点

- 失真函数一旦选定，后面的平均失真和率失真函数都会跟着它走。
