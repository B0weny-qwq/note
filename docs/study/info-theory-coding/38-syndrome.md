# 伴随式

## 定义

接收向量 `r` 与校验矩阵相乘得到的结果，叫伴随式。

常记作：

```text
S = rH^T
```

## 为什么它重要

伴随式相当于“错误指纹”：

- 如果 `S = 0`，通常表示无错或错码刚好落在某个特殊模式
- 如果 `S != 0`，可以用来定位错误模式

## 详细例子

如果一个单比特出错，不同出错位置对应的伴随式通常不同。  
于是你看见某个 `S`，就能反推出哪一位错了。

## 生活类比

像门禁系统报错码：

- `E01` 表示门没关好
- `E02` 表示卡权限不够

不同错误对应不同错误码。伴随式就是这种“错误代码”。

## 易错点

- 伴随式不是“纠错后的码字”，它只是用于判别和定位。

## 注意点

- 单错纠正题里，伴随式通常是最快入口。
