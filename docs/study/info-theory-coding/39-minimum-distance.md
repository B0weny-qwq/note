# 最小码距

## 定义

最小码距是一个码集中任意两个不同合法码字之间最少相差多少位，它是衡量一个码检错和纠错能力的核心指标。

## 码距是什么

两个码字之间不同位的个数，叫汉明距离，也叫码距。

## 最小码距是什么

所有不同合法码字两两之间的最小距离，记作：

```text
dmin
```

## 为什么重要

`dmin` 直接决定：

- 能检出多少位错误
- 能纠正多少位错误

## 常用结论

检错能力：

```text
dmin - 1
```

纠错能力：

```text
floor((dmin - 1) / 2)
```

## 详细例子

如果 `dmin = 3`：

- 最多能检出 2 位错
- 最多能纠正 1 位错

## 生活类比

像两个密码箱的外观差异：

- 差异越大，就越不容易认错
- 差异很小，就更容易混淆

## 易错点

- 检错能力和纠错能力不是一回事，别把两个公式混掉。

## 注意点

- 第 6 章很多题最后都要落到 `dmin` 上。
