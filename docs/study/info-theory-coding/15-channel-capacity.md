# 信道容量的定义

## 定义

信道容量表示：在给定信道条件下，单位时间或单位符号内能够可靠传输的最大平均信息量。

公式核心是：

```text
C = max I(X;Y)
```

## 为什么是“最大互信息”

因为输入分布可以选。  
同一个信道，在不同输入概率分布下，接收端得到的有效信息量不同。  
容量就是把所有可能输入策略里最好的那个挑出来。

## 详细例子

同一条路上运快递：

- 你可以全送大件，也可以大小件混送。
- 不同装载策略，对单位时间送达的有效货物量不同。

信息论里的“输入分布”就有点像“装载策略”。

## 生活类比

高速公路的“最大通行能力”不是看偶尔一次跑多快，而是看在合理调度下长期稳定能跑多少车。  
信道容量也是这样，它是稳定、可靠传输条件下的上限。

## 易错点

- 信道容量不是某次传输速度。
- 容量不是随便代个概率就得到的互信息，而是“最大互信息”。

## 注意点

- 后面所有 BSC、高斯信道、MIMO 信道，本质上都在求这一个量。
