# 线性分组码

## 定义

线性分组码把消息按固定长度分组编码。  
“线性”的意思是：任意两个合法码字按模 2 相加，结果仍然是合法码字。

## 为什么重要

因为这种结构规则整齐，便于：

- 数学分析
- 电路实现
- 检错和纠错

## 详细例子

假设一组合法码字有：

```text
000
011
101
110
```

这里：

```text
011 xor 101 = 110
```

结果还是码字，说明它具备线性结构。

## 生活类比

像一个标准化积木系统：

- 所有积木都遵循统一拼接规则
- 拼出来的新结构仍在系统规则内

## 易错点

- 分组码不一定线性，线性只是常见且特别重要的一类。
