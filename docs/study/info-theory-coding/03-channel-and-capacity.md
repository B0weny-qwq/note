# 信道与信道容量

## 背景

信源章节讨论“有多少信息”，信道章节讨论“最多能传多少信息”。容量是这一章的中心概念。

## 信道模型

常见信道分类：

- 离散信道与连续信道。
- 单符号信道与序列信道。
- 无记忆信道与有记忆信道。

离散信道通常由转移概率矩阵描述。

## 互信息与容量

容量定义：

```text
C = max I(X;Y)
```

也就是在所有输入分布中，找到使平均互信息最大的那个值。

## 二进制对称信道 BSC

这是最典型的离散信道模型。对误码率为 `p` 的 BSC：

```text
C = 1 - H(p)
```

结论：

- `p = 0` 时容量最大。
- `p = 0.5` 时容量为 0。

## 强对称与准对称信道

离散信道常见两类：

- 强对称信道：行列对称，容量求解最直接。
- 准对称信道：可分解成若干强对称子信道。

很多题的关键不是硬算，而是先认出它属于哪一类。

## 离散无记忆信道

对独立使用的 DMC，序列容量可由单次容量推广得到。无记忆性让大多数推导可以分解到单符号层面。

## 连续信道与高斯信道

连续信道容量最重要的结论是加性高斯白噪声信道容量：

```text
C = B log2(1 + S / N)
```

这里 `B` 是带宽，`S/N` 是信噪比。

## 影响容量的因素

- 信噪比越高，容量越大。
- 带宽越大，容量越大。
- 输入分布选择越接近最优，越能逼近容量。

## 做题步骤

1. 先写信道模型或转移矩阵。
2. 判断是 BSC、强对称、准对称还是一般 DMC。
3. 求互信息表达式。
4. 再对输入分布取最大值得到容量。
