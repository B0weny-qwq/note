# 信源、信息量与熵

## 背景

这一章的任务是把“消息”变成可计算的随机模型，然后用概率来定义信息量。

## 信源建模

常见分类：

- 离散信源与连续信源。
- 单符号信源与序列信源。
- 无记忆信源与有记忆信源。

香农信息论默认先从随机变量和随机过程建模出发。

## 自信息与熵

单个事件的信息量：

```text
I(x) = -log p(x)
```

平均信息量就是熵：

```text
H(X) = -Σ p(x) log p(x)
```

理解：

- 概率越小，单次信息量越大。
- 分布越均匀，熵越大。

## 联合熵、条件熵、互信息

三个关系式必须熟：

- `H(X,Y) = H(X) + H(Y|X)`
- `I(X;Y) = H(X) - H(X|Y)`
- `I(X;Y) = H(X) + H(Y) - H(X,Y)`

含义：

- 条件熵表示已知一个变量后剩余的不确定度。
- 互信息表示两个变量共享的信息。

## 序列信源与熵率

对长序列，更关心单位符号平均信息量，也就是熵率。

结论方向：

- 无记忆信源的熵率等于单符号熵。
- 有记忆信源由于相关性存在，熵率通常更小。

## 连续信源

连续信源使用概率密度和微分熵描述。微分熵和离散熵性质不完全一样，比较时要注意量纲和坐标变换问题。

## 冗余度

冗余度衡量实际信源和最大熵之间的差距。冗余越大，越有压缩空间。

## 做题核心

- 先写概率分布。
- 再算熵、条件熵、互信息。
- 判断是否独立、是否无记忆。
- 求编码极限时，把熵看成平均码长的下界。
