# 条件熵

## 定义

条件熵表示：在已知另一个变量之后，当前变量还剩下多少不确定性。

公式：

```text
H(X|Y)
```

## 直觉理解

知道更多背景信息后，通常更容易猜测结果，所以不确定性会下降。

## 详细例子

假设 `X` 是“今天是否带伞”，`Y` 是“天气是否下雨”。

- 如果不知道天气，`X` 比较难猜。
- 如果已经知道在下大雨，`X` 更容易猜成“带伞”。

于是：

```text
H(X|Y) < H(X)
```

## 生活类比

不知道上下文时，很难猜一句话的下一个字。  
如果知道前面已经出现“信息”，你更容易猜后面是“论”。  
这就是条件信息减少了不确定性。

## 重要结论

```text
条件熵不大于无条件熵
```

也就是：

```text
H(X|Y) <= H(X)
```

## 易错点

- 不是所有知道的信息都能大幅降低不确定性，关键看 `Y` 和 `X` 是否相关。

## 注意点

- 后面求互信息时，条件熵是最核心的组成部分。
