# 信源编码

## 背景

信源编码的目标是去掉冗余，让平均码长尽量接近熵。第 5 章一半讲判定条件，一半讲具体构造方法。

## 码的基本要求

常见层次：

- 非奇异码。
- 唯一可译码。
- 即时码（非延长码）。

关系上，即时码一定唯一可译，但唯一可译不一定即时。

## 平均码长和编码效率

关键量：

- 平均码长 `L`
- 编码效率 `η = H / L`

目标是让 `L` 尽量小，但仍满足唯一可译或即时码条件。

## 码树与 Kraft 条件

变长码经常借助码树分析。即时码的本质是任一码字都不是其他码字的前缀。

## 经典无失真编码

### 香农码

- 构造规则明确。
- 一般能接近熵，但不一定最优。

### 费诺码

- 基于概率分组。
- 对概率接近的分布比较方便。

### 哈夫曼码

- 课程里最重要。
- 对给定离散无记忆信源能得到平均码长最短的前缀码。

## 算术编码

算术编码不是按符号逐个分配码字，而是把整个序列映射成区间。

特点：

- 压缩性能高。
- 更接近理论极限。
- 实现复杂度高于普通 Huffman 编码。

## 有损编码引子

课件后部还引到量化和限失真压缩，和上一章的率失真理论对应。

## 做题顺序

1. 先判断码是否唯一可译、是否即时。
2. 再算平均码长。
3. 再算效率。
4. 构造最优变长码时优先想到 Huffman。
