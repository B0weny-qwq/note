# 8086 寄存器、内存与寻址

## 背景

8086 通过“段地址 + 偏移地址”访问内存。理解寄存器分工、默认段寄存器和寻址方式，是看懂后续指令的基础。

## 通用寄存器

8086 的常用 16 位通用寄存器：

```text
AX  BX  CX  DX  SI  DI  BP  SP
```

其中 `AX`、`BX`、`CX`、`DX` 可以拆成高 8 位和低 8 位：

```text
AX = AH + AL
BX = BH + BL
CX = CH + CL
DX = DH + DL
```

常见用途：

| 寄存器 | 常见用途 |
| --- | --- |
| `AX` | 累加器，常保存运算结果，也是乘除法和 I/O 的隐含寄存器 |
| `BX` | 基址寄存器，常参与内存寻址 |
| `CX` | 计数寄存器，`LOOP` 和移位计数常用 |
| `DX` | 数据寄存器，乘除法高位和大端口地址常用 |
| `SI` | 源变址寄存器，字符串源地址常用 |
| `DI` | 目的变址寄存器，字符串目的地址常用 |
| `BP` | 基址指针，访问栈中数据常用 |
| `SP` | 栈顶指针，配合 `SS` 指向栈顶 |

## 段寄存器

8086 的段寄存器：

```text
CS  代码段
DS  数据段
ES  附加段
SS  堆栈段
```

常见默认关系：

| 寻址形式 | 默认段寄存器 |
| --- | --- |
| `[BX]` | `DS` |
| `[SI]` | `DS` |
| `[DI]` | `DS` |
| `[BP]` | `SS` |
| `[BP+SI]` | `SS` |
| `[BP+DI]` | `SS` |

注意：

```text
只要寻址表达式中使用 BP，默认段一般就是 SS。
没有 BP 时，默认段一般是 DS。
```

## 物理地址

8086 物理地址计算公式：

```text
物理地址 = 段地址 × 10H + 偏移地址
```

例子：

```text
DS = 2000H
SI = 00ADH
```

访问：

```asm
MOV AL, [SI]
```

物理地址：

```text
2000H × 10H + 00ADH = 200ADH
```

## 直接寻址

直接给出偏移地址。

```asm
MOV AX, [100H]
```

含义：

```text
AX = DS:[0100H]
```

如果 `DS = 2000H`，物理地址为：

```text
20100H
```

## 寄存器间接寻址

偏移地址放在寄存器中。

```asm
MOV BX, [SI]
```

含义：

```text
BX = DS:[SI]
```

常用寄存器：

```text
BX  BP  SI  DI
```

## 寄存器相对寻址

寄存器加位移量形成偏移地址。

```asm
MOV CX, [BX+10H]
```

如果：

```text
BX = 0100H
```

偏移地址：

```text
0100H + 10H = 0110H
```

## 基址变址寻址

基址寄存器和变址寄存器相加。

```asm
MOV AX, [BX][SI]
```

等价于：

```asm
MOV AX, [BX+SI]
```

如果：

```text
BX = 0100H
SI = 00ADH
```

偏移地址：

```text
01ADH
```

## 基址变址相对寻址

基址寄存器、变址寄存器和位移量一起形成偏移地址。

```asm
MOV AX, [BX+SI+0062H]
```

偏移地址：

```text
BX + SI + 0062H
```

## BP 默认使用 SS

例子：

```asm
MOV CL, [BP+4]
```

如果：

```text
SS = 1500H
BP = 0010H
```

物理地址：

```text
1500H × 10H + 0014H = 15014H
```

## 段超越前缀

可以显式指定段寄存器。

```asm
MOV CH, ES:[BP+SI]
```

虽然表达式里用了 `BP`，但有 `ES:` 前缀，所以段地址使用 `ES`。

如果：

```text
ES = 2100H
BP = 0010H
SI = 00ADH
```

物理地址：

```text
2100H × 10H + 00BDH = 210BDH
```

## OFFSET

`OFFSET` 取得变量在段内的偏移地址。

```asm
MOV SI, OFFSET NUM
```

含义：

```text
SI = NUM 的偏移地址
```

注意区别：

```asm
MOV AX, NUM          ; 取 NUM 里面的内容
MOV AX, OFFSET NUM   ; 取 NUM 的偏移地址
```

## ORG

`ORG` 设置后续数据或代码的偏移地址。

```asm
ORG 3000H
NUM DB 09H, 03H, 08H
```

含义：

```text
NUM 的偏移地址从 3000H 开始
```

`ORG` 是伪指令，不会修改 `DS`。
